## 🧱 知识讲解

### 5.1  cin：让程序听你说话

`cin`（c + in = character input）是 C++ 的标准输入。它等你在键盘上敲东西，然后把你敲的内容读进变量。

```cpp
int age;
cout << "请输入你的年龄：";
cin >> age;
cout << "你今年 " << age << " 岁" << endl;
```

运行效果：

```
请输入你的年龄：12
你今年 12 岁
```

注意 `>>` 的方向：`cout <<` 是数据往外流（程序 → 屏幕），`cin >>` 是数据往内流（键盘 → 程序）。

### 5.2  一行输入多个值

`cin` 可以用空格或换行分隔多个输入：

```cpp
int a, b;
cin >> a >> b;        // 用户输入 "10 20"，a=10, b=20
cout << a + b << endl;  // 输出 30
```

用户既可以在一行里用空格分开输入（`10 20`），也可以分两行输入（先敲 10 回车，再敲 20 回车）。`cin` 会自动跳过空白字符。

### 5.3  输入不同类型的数据

```cpp
string name;    // string 类型：装一串文字
int age;
double score;

cout << "姓名：";  cin >> name;
cout << "年龄：";  cin >> age;
cout << "分数：";  cin >> score;

cout << name << "，今年 " << age << " 岁，考了 " << score << " 分" << endl;
```

运行效果：

```
姓名：小明
年龄：13
分数：96.5
小明，今年 13 岁，考了 96.5 分
```

> ⚠️ 用 `cin >>` 读 string 时，遇到空格就停止。如果用户叫"张 三"，`cin >> name` 只能读到"张"。解决方法是 `getline`（后面章节会讲）。

### 5.4  cout 的输出控制

**换行**：

```cpp
cout << "第一行" << endl;   // endl = 换行
cout << "第二行\n";         // \n 也是换行（在字符串内部）
cout << "第三行";           // 没有换行，下个输出会接在后面
```

`endl` 和 `\n` 的区别：`endl` 会额外刷新输出缓冲区（确保内容立刻显示），`\n` 只是一个换行符。竞赛中用 `\n` 稍快一点，日常用哪个都行。

**输出多个东西**：

```cpp
cout << "身高：" << 175 << "cm" << endl;
// 用 << 把文字、数字、变量串起来
```

**控制浮点数小数位数**（需要 `<iomanip>` 库）：

```cpp
#include <iomanip>
// ...
double pi = 3.14159265;
cout << fixed << setprecision(2) << pi << endl;  // 输出 3.14
cout << fixed << setprecision(4) << pi << endl;  // 输出 3.1416
```

### 5.5  printf 和 scanf：老派的输入输出

C++ 同时也支持 C 语言的输入输出方式——`printf` 和 `scanf`。它们比 `cout/cin` 更啰嗦，但在**大量数据输入输出**时快很多。竞赛选手几乎都用它们。

**printf 输出**：

```cpp
#include <cstdio>
// ...
int age = 13;
double score = 96.5;
printf("我叫小明，今年 %d 岁，考了 %.1f 分\n", age, score);
```

`%d` 是"这里放一个 int"，`%.1f` 是"这里放一个 double，保留 1 位小数"，`\n` 是换行。

| 占位符 | 含义 |
|--------|------|
| `%d` | 整数（int） |
| `%lld` | 大整数（long long） |
| `%f` | 浮点数（float/double） |
| `%.2f` | 浮点数，保留 2 位小数 |
| `%c` | 单个字符 |
| `%s` | 字符串 |

**scanf 输入**：

```cpp
int age;
double score;
scanf("%d %lf", &age, &score);  // 用户输入年龄和分数
```

注意：`scanf` 需要变量名前加 `&`（取地址符）。`%lf` 用于读 double（`printf` 输出 double 仍用 `%f`）。

> ⚠️ 初学阶段，用 `cin/cout` 就够了。进入第二册的算法训练后，再逐步切换到 `printf/scanf`。

### 5.6  输入缓冲区的概念

当你敲键盘的时候，字符不是直接送进程序的。它们先暂存在一个叫**输入缓冲区**的地方。等你按下**回车键**，缓冲区里的内容才一次性交给程序处理。

这就是为什么你可以慢慢打字，程序不会反应——它还在等你按回车。

## ⚠️ 容易犯的错

### 错 1：cin 方向写反

❌ `cin << age;` // 箭头方向错了
✅ `cin >> age;` // 数据流进 cin

### 错 2：printf 类型不匹配

❌ `printf("%d", 3.14);` // %d 是 int，但给的是 double
✅ `printf("%f", 3.14);`

❌ `printf("%d", 1234567890123LL);` // long long 用了 %d
✅ `printf("%lld", 1234567890123LL);` // long long 必须用 %lld

### 错 3：scanf 忘了写 &

❌ `scanf("%d", age);` // 忘了 &
✅ `scanf("%d", &age);` // & 不能丢

### 错 4：中英文符号再次出现

❌ `cout << "你好"；` // 中文分号
✅ `cout << "你好";` // 英文分号

### 提高题

**3. 编程题 — 计算器**

写一个程序，让用户输入两个整数，分别输出：和、差、积、商（整数除法）和余数。

示例运行：

```
请输入两个整数：17 5
和 = 22
```

**4. 编程题 — 自我介绍**

### 挑战题

**5. 编程题 — 时间转换**

让用户输入一个秒数（例如 3665），程序把它转换成"X 小时 Y 分 Z 秒"的格式。

示例运行：

```
请输入秒数：3665
3665 秒 = 1 小时 1 分 5 秒
```

（提示：1 小时 = 3600 秒，1 分钟 = 60 秒，善用 `/` 和 `%` 运算。）

## 🧠 本章小结

```
输入：cin >> 变量;       （数据往里流）
输入：scanf("%d", &变量); （C 风格，更快）

输出：cout << 内容;       （数据往外流）
输出：printf("%d", 变量);  （C 风格，更灵活）

endl / \n   = 换行
fixed + setprecision = 控制小数位数

输入缓冲区：你打字 → 暂存 → 按回车 → 一次性交给程序
```
