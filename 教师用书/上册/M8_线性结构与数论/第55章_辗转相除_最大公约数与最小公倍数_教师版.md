| 时间 | 环节 | 操作 |
|------|------|------|
| 0-12 分 | 导入 | 两个故事双线引入：①铺地砖——48dm×18dm 客厅，用正方形地砖铺满不切割。地砖边长只能是公约数，最大的是 GCD=6dm。②分蛋糕——小红 12 块、小兰 18 块，混在一起重新分组，每组配比相同。最大组数 = GCD(12,18) = 6 组。同一个数学概念，两个完全不同但都真实的生活场景 |
| 12-28 分 | 讲授 | 辗转相除法核心递推：`gcd(a,b) = gcd(b, a%b)`。手工推演 gcd(48,18)：48÷18=2 余 12→gcd(18,12)；18÷12=1 余 6→gcd(12,6)；12÷6=2 余 0→gcd(6,0)=6。板书画"辗转"过程——像拧毛巾，每拧一次数字变小，最后拧出 GCD。代码仅一行：`return b==0 ? a : gcd(b, a%b);`。迭代版：`while(b){int t=a%b;a=b;b=t;}` |
| 28-40 分 | 演示 | IDE 对比三种方法：①暴力枚举 `for(i=min(a,b);i>=1;i--)`——O(min(a,b)) ②辗转相除递归版——O(log min(a,b)) ③辗转相除迭代版。a=10⁹, b=10⁹-1 时暴力几乎卡死，辗转相除瞬间。让学生理解"算法优化不是锦上添花，而是能跑和不能跑的区别" |
| 40-45 分 | 小结 | "gcd = 一行代码的算法。两千多年前欧几里得的发现，至今没有被超越。这就是算法之美——**简单而永恒。**" |

## 🔥 重点难点

### 教学重点

1. **辗转相除法的递推直觉**：为什么 `gcd(a,b) = gcd(b, a%b)`？因为 a 和 b 的公约数集合 = b 和 a%b 的公约数集合。板书证明：设 d 是 a 和 b 的公约数，则 d|a 且 d|b → d|(a - k×b) → d|(a%b)。反之亦然。所以公约数集不变，最大公约数自然也不变。
2. **LCM 先除后乘**：`a / gcd * b` 而非 `a * b / gcd`——防中间结果溢出。这是竞赛中的"保命写法"。
3. **裴蜀定理的直觉**：任何能用 a 和 b 的整数线性组合表示的数，必须是 gcd(a,b) 的倍数。反过来，gcd(a,b) 的任意倍数都可以表示。这就是"水壶量水"问题能解或不能解的根本原因。

### 常见卡点 + 突破策略

| 卡点 | 表现 | 突破方法 |
|------|------|----------|
| LCM 直接 `a*b/gcd` 溢出 | a=10⁹, b=10⁹ 时 a×b=10¹⁸ 超出 long long | 强制写法：`a / gcd(a,b) * b`。先除后乘保证中间值不超过 max(a,b) |
| 辗转相除递归不终止 | 忘了 b==0 的边界条件 | 口诀："**余 0 则停，被除数就是答案。**"——当余数为 0 时，上一轮的除数就是 GCD |
| 多数的 GCD 写成 `gcd(gcd(a,b), gcd(c,d))` | 三个数用两两分组 | 正确做法是链式：`g = a[1]; for(i=2;i<=n;i++) g = gcd(g, a[i])`。GCD 满足结合律，链式和分组结果一样，但链式更简洁 |

## 💬 常见问答

**Q1："辗转相除为什么这么快？"**

A：每次 `a%b` 至少把大的数缩小一半。最坏情况是相邻的斐波那契数（如 F(10)=55, F(9)=34），此时每次只减少约 38%，但总步数仍是对数级别——O(log min(a,b))。比暴力枚举的 O(min(a,b)) 快了指数倍。

**Q2："扩展欧几里得算法是干什么的？"**

A：不仅能求 gcd，还能找到一组整数解 (x,y) 使 `ax + by = gcd(a,b)`。这是求模逆元、解同余方程、解决"水壶量水"问题的基础。完整算法在 M14 数论进阶展开。

**Q3："GCD 和 LCM 在实际竞赛中有哪些应用？"**

A：最简分数化简、周期性问题的公共周期、分数通分、判断方程有无整数解、同余方程……几乎每场竞赛都至少有一道题涉及 gcd。

**Q4："扩展欧几里得算法具体是怎么求 x 和 y 的？"**

A：在递归返回时回溯计算。当 b==0 时，x=1, y=0（因为 a×1+0×0=a=gcd）。回溯时利用递推关系：已知 `gcd(b, a%b) = bx' + (a%b)y'`，而 `a%b = a - (a/b)×b`，代入得新解。完整代码在 M14 展开。

**Q5："裴蜀定理的典型应用场景是什么？"**

A：判断 `ax + by = c` 是否有整数解——只需检查 `c % gcd(a,b) == 0`。例如"用 3 升和 5 升水壶量出 4 升水"——gcd(3,5)=1，1 整除 4，所以可行！具体步骤用 BFS 或扩展欧几里得给出。

## 📋 本章小测（15 分钟，闭卷）

### 一、填空题（每空 2 分，共 10 分）

1. `gcd(48, 18) = ____`。
2. `lcm(12, 18) = ____`。
3. 辗转相除法的核心递推是 `gcd(a,b) = gcd(b, ____)`。
4. 求 LCM 时应该写 `a / gcd(a,b) * b` 而不是 `a * b / gcd(a,b)` 是为了____。
5. `ax + by = c` 有整数解的条件是____。

### 二、写代码（10 分）

6. 写一个递归函数 `int gcd(int a, int b)` 求最大公约数。再写一个 `int lcm(int a, int b)` 求最小公倍数（调用 gcd）。

---

### 答案

1. 6
2. 36
3. a % b
4. 防溢出（先除后乘使中间值不超过 max(a,b)）
5. gcd(a,b) 整除 c（`c % gcd(a,b) == 0`）
6. ```cpp
int gcd(int a, int b) { return b == 0 ? a : gcd(b, a % b); }
int lcm(int a, int b) { return a / gcd(a, b) * b; }
```
