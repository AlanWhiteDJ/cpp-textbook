## 🗺️ 教学流程

### 第 1 课时：前缀和——构建 + 区间查询 + 最大子段和（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 导入 | 老师设定场景："年级 2000 人，年级组长问 100 次'第 l 名到第 r 名的总分'——每次都现场加一遍？"引出前缀和思想：提前算好累计总分表 `pre[i]`，查询时 `pre[r]-pre[l-1]` 一步搞定。类比：里程表——第 5 公里到第 20 公里 = 20-5 = 15km |
| 8-22 分 | 讲授 | 板书前缀和定义：`pre[0]=0; pre[i]=pre[i-1]+a[i]`（i 从 1 开始）。区间和公式：`sum(l,r) = pre[r] - pre[l-1]`。关键细节：① `pre[0]=0` 是必须的（否则 l=1 时 `pre[l-1]` 没定义）②数组下标从 1 开始更方便。演示：`a={3,1,4,1,5}` → `pre={0,3,4,8,9,14}` → `sum(2,4)=pre[4]-pre[1]=9-3=6` |
| 22-35 分 | 讲授 | **最大子段和（前缀和视角）**：经典问题——给定数组，找连续子段的最大和。前缀和思路：`sum(i,j) = pre[j] - pre[i-1]`，对于固定的 j，要使 sum 最大→需要 `pre[i-1]` 最小。所以遍历 j，维护 `minPre = min(minPre, pre[j-1])`，`ans = max(ans, pre[j] - minPre)`。与 DP 解法对比：DP 是"以 j 结尾的最大子段和"，前缀和是"j 之前的最小前缀和"——**同一个问题的不同视角** |
| 35-42 分 | 练习 | 学生上机：①读入 n 个数和 q 个查询，每个查询 l r，输出区间和（用前缀和）②最大子段和（用前缀和方法）。比较：q=10⁵ 时，不用前缀和 O(nq)=10¹⁰ 超时，用前缀和 O(n+q)=2×10⁵ 毫秒级 |
| 42-45 分 | 小结 | "前缀和 = 一劳永逸。O(n) 预处理后，每个查询 O(1)。这个公式要用到毕业：**`pre[r]-pre[l-1]`**。" |

### 第 2 课时：差分数组——区间修改 + 单点还原（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 导入 | "上节课是'区间求和快'——前缀和。如果问题是反过来呢——'对区间 [l,r] 每个数加 x，最后问每个数是多少'？每次操作都遍历 [l,r] 是 O(n×m)。有没有 O(1) 修改的方法？"——引出差分 |
| 8-22 分 | 🎮 不插电活动 | **"爬山海拔"**（详见活动区）。用海拔变化演示差分——上升/下降标记在变化点，最后的绝对海拔 = 从起点累积变化量 |
| 22-35 分 | 讲授 | **差分数组**：`diff[i] = a[i] - a[i-1]`。区间修改 `[l,r] + x`：`diff[l] += x; diff[r+1] -= x`。板书推导：为什么这样能 O(1) 修改？因为对原数组 [l,r] 加 x，只会改变 diff[l]（a[l] 增加了 x，差值变大 x）和 diff[r+1]（a[r+1] 没变但 a[r] 增加 x，差值变小 x）。最后 **前缀和还原**：`a[i] = a[i-1] + diff[i]`。强调：**前缀和和差分是互逆运算**——前缀和的差分等于原数组，差分的还原就是前缀和 |
| 35-42 分 | 练习 | 学生上机：初始全零数组，m 次操作每次 `[l,r]+x`，最后输出数组。先用暴力 O(n×m) 跑 m=100000 体会超时，再用差分 O(n+m) 瞬间出结果。**让超时成为教学的盟友** |
| 42-45 分 | 小结 | 口诀："**前缀和 = 区间求和 O(1)；差分 = 区间修改 O(1)。一对互逆的好搭档！**" |

### 第 3 课时：二维前缀和与差分 + 综合练习（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-15 分 | 讲授 | **二维前缀和**：`pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + a[i][j]`。子矩阵和 `sum(x1,y1,x2,y2) = pre[x2][y2] - pre[x1-1][y2] - pre[x2][y1-1] + pre[x1-1][y1-1]`。板书画 4×4 网格，标出四个关键区域，让学生理解"加两次减一次"的容斥原理。**二维差分**：区间修改 `(x1,y1,x2,y2) + v`——四角标记法：`diff[x1][y1]+=v; diff[x2+1][y1]-=v; diff[x1][y2+1]-=v; diff[x2+1][y2+1]+=v` |
| 15-35 分 | 独立练习 | 学生完成：①子矩阵和查询（二维前缀和）②海底高铁（POJ 经典）——n 个城市，m 次访问（按序经过某些城市段），求最少买票费用。提示：统计每条铁路经过次数，用差分 O(m) 统计，然后每条铁路选最便宜的购票策略。综合题：先用差分统计频次，再用前缀和的思想做决策 |
| 35-45 分 | 讲评 | 重点讲海底高铁的"差分统计频次"——这是差分最经典的应用之一。经过 `u→v`（u<v）的段：`diff[u]++; diff[v]--`——和区间修改一模一样！让学生体会到：**统计频次 = 区间加 1 = 差分** |

## 🔥 重点难点

### 教学重点

1. **前缀和公式 `pre[r]-pre[l-1]` 中的 `l-1`**：学生经常写成 `pre[r]-pre[l]`。要在黑板上用具体例子验证：`sum(2,4)=pre[4]-pre[1]` 而不是 `pre[4]-pre[2]`。
2. **差分区间修改的两个点**：`diff[l] += x` 和 `diff[r+1] -= x`。用"海拔变化"类比——上坡标记在起点，下坡标记在坡顶+1。
3. **二维的容斥公式**：学生容易搞混加减号。口诀："**大减左减上加左上**"——`pre[x2][y2] - 左 - 上 + 左上`。

### 常见卡点 + 突破策略

| 卡点 | 表现 | 突破方法 |
|------|------|----------|
| 前缀和 `pre[0]` 忘了初始化 | `pre[0]` 是垃圾值，所有区间和全错 | 强制规则：**`pre[0] = 0` 必须在第一行写**。板书时每次都在 pre 数组旁边标注 `pre[0]=0` |
| 一维差分 `r+1` 越界 | 当 r=n 时 `diff[n+1]` 越界 | 解决方法：①数组多开一位 `diff[n+2]` ②或者加判断 `if (r < n) diff[r+1] -= x`。在教学中明确告诉学生这个坑 |
| 二维前缀和的"多减了左上" | 子矩阵和公式丢掉了 `+ pre[x1-1][y1-1]` | 画图！在 4×4 格子上用彩色粉笔标出四个区域：pre[x2][y2]（全蓝）、减去 pre[x1-1][y2]（划掉上面）、减去 pre[x2][y1-1]（划掉左边）、加上 pre[x1-1][y1-1]（左上被划了两次，补回来） |
| 差分和前缀和混淆 | 区间求和用了差分，区间修改用了前缀和 | 对比表：前缀和=预处理后 O(1) 查区间和；差分=O(1) 区间修改后 O(n) 还原。强调用途不同 |
| 最大子段和——`minPre` 的初值 | `minPre = 0` 还是 `minPre = a[1]`？ | 推导：对于 j=1, `pre[1]-minPre` 需要等于 a[1]（选单个元素），所以 `minPre = 0`（因为 pre[0]=0）。如果 `minPre = pre[1]`，第一个元素就会被跳过 |

## 💬 常见问答

**Q1："前缀和和差分的关系怎么记？"**

A：**前缀和 = 积分，差分 = 微分**。这个类比很精确——前缀和把"单点值"变成"累计值"（像速度积分得到位移），差分把"累计值"变回"单点值"（像位移微分得到速度）。互逆运算：前缀和的差分 = 原数组；差分的前缀和 = 原数组。

**Q2："最大子段和为什么用前缀和也能做？之前不是说用 DP 吗？"**

A：同一个问题有多个视角是好事。DP 视角：`dp[i] = max(a[i], dp[i-1]+a[i])`。前缀和视角：`max(pre[j] - pre[i-1]) for i≤j` = 找前面最小的 pre。两个视角复杂度都是 O(n)，理解透一个就行。但前缀和视角能自然扩展到"区间修改+查询最大子段和"——这是线段树的经典问题。

**Q3："二维差分那四个角的符号，有什么记忆诀窍？"**

A：`(x1,y1)`——区间起点，加 v（"开始加"）。`(x2+1,y1)`——区间的右邻，减 v（"停止加"）。`(x1,y2+1)`——区间的下邻，减 v。`(x2+1,y2+1)`——右下角，加 v（"停止减"，因为被停了两次）。画一个矩形 + 上下左右的溢出区，就能自然推导。

**Q4："海底高铁那题，经过次数统计为什么用差分？"**

A：每次访问 `u→v`，对区间 `[u, v-1]`（或 `[u, v]` 取决于题意）的经过次数 +1。m 次访问就是 m 次区间修改——差分 O(m+n) vs 暴力 O(m×n)。统计完频次后，每条铁路独立决策——买纸质票还是办卡——就是一个简单的 `min` 比较。

**Q5："前缀和/差分是不是只能处理加法和减法？乘除法可以吗？"**

A：前缀和/差分依赖"可逆运算"——加法有减法逆运算。乘法的话，如果用前缀积（`pre[i]=pre[i-1]*a[i]`），区间积就是 `pre[r]/pre[l-1]`——但需要模逆元（涉及数论）。对于异或也可以：`xor(l,r) = pre[r] ^ pre[l-1]`。核心思想是通用的，只是逆运算不同。

## 📋 本章小测（18 分钟，闭卷）

### 一、填空题（每空 2 分，共 10 分）

1. 对于数组 `a[1..n]`，前缀和数组 `pre[i]` = __________，其中 `pre[0]` = __________。
2. 区间 `[l, r]` 的元素和 = __________。
3. 对区间 `[l, r]` 每个元素加 x，差分操作为 `diff[___] += x; diff[___] -= x;`
4. 二维前缀和子矩阵和的公式中，四个项是：`pre[x2][y2]` - __________ - __________ + __________。

### 二、改错题（5 分）

5. 以下代码想构建前缀和数组，有什么错误？
```cpp
int pre[100005];
for (int i = 1; i <= n; i++)
    pre[i] = pre[i-1] + a[i-1];
```

### 三、写代码（15 分）

6. （8 分）用差分数组处理：初始全零数组 a[1..n]，m 次操作每次 `[l,r]+1`，最后输出数组。

7. （7 分）用前缀和解决：给定数组，q 次查询区间和。

---

### 答案

1. `pre[i-1] + a[i]`，0
2. `pre[r] - pre[l-1]`
3. `diff[l]`，`diff[r+1]`
4. `pre[x1-1][y2]`，`pre[x2][y1-1]`，`pre[x1-1][y1-1]`
5. ①`pre[0]` 未初始化为 0 ②`a[i-1]` 下标不对——如果数组从 1 开始存，应为 `a[i]`；如果从 0 开始存，pre 也应对应从 0 开始。
6. ```cpp
int diff[100005] = {0};
while (m--) {
    int l, r; cin >> l >> r;
    diff[l]++; diff[r+1]--;
}
int cur = 0;
for (int i = 1; i <= n; i++) {
    cur += diff[i];
    cout << cur << " ";
}
```
7. ```cpp
int pre[100005] = {0};
for (int i = 1; i <= n; i++) pre[i] = pre[i-1] + a[i];
while (q--) {
    int l, r; cin >> l >> r;
    cout << pre[r] - pre[l-1] << endl;
}
```

## 🎮 不插电活动

### 活动 1：爬山海拔——差分可视化（第 2 课时用，约 15 分钟）

**道具**：
- 白板，画一条横线（水平面），上面标出位置 1~10
- 10 个磁贴代表海拔高度

**规则与流程**：

| 分钟 | 角色 | 操作 |
|------|------|------|
| 0-3 分 | 老师 | 设定场景：我们从海平面出发（海拔 0），走 10 步。每一步可能有"上升"或"下降"。差分 = 记录每一步的"海拔变化量" |
| 3-8 分 | 全体 | 例子：`diff = {0, +3, 0, -1, +2, 0, -4, 0, 0, +1}`。老师带全班一步步走：位置 0 海拔=0，位置 1 海拔=0+3=3，位置 2 海拔=3+0=3……每走一步算出海拔，把磁贴吸在白板上 |
| 8-12 分 | 老师 | 现在反过来：如果要从位置 3 到位置 6 所有位置海拔 +2（区间修改），哪些 diff 会变？——`diff[3] += 2`（位置 3 开始上升更多），`diff[7] -= 2`（位置 7 开始恢复到原来的变化量）。画图验证 |
| 12-15 分 | 老师 | 总结："diff 存的是**变化量**，不是绝对高度。绝对高度 = 从起点累积所有变化量。区间修改只改变两个 diff 值——起点的变化量和终点下一个的变化量。" |

**教学要点**：让学生用身体"走"出海拔——从教室一侧走到另一侧，每步大声喊出"上升 +X"或"下降 -X"。身体力学记忆比看板书更深刻。

### 活动 2：矩阵涂色——二维前缀和（第 4 课时用，约 8 分钟）

**道具**：白板上画 4×4 网格，用四种颜色粉笔

## 📦 补充题库

1. 如果数组下标从 0 开始，前缀和公式怎么改？  
（参考答案：`pre[i] = pre[i-1] + a[i-1]`（pre[0]=0），区间和 `sum(l,r) = pre[r+1] - pre[l]`——注意下标都偏移了 1。）

2. 前缀和 + 差分结合题："区间加法后查询单点" vs "单点加法后查询区间和"——分别用什么？  
（参考答案：前者用差分（区间改 O(1)，单点查 = 前缀和还原 O(n)）；后者用前缀和（单点改 O(1)，区间查 O(1)）。）

3. 二维前缀和：给定 n×m 的 0/1 矩阵，查询子矩阵内 1 的个数。（参考答案：用前缀和累加即可——值和计数一样处理。）

4. 进阶题：给定数组和 k，求有多少个子区间的和等于 k。提示：`pre[j] - pre[i-1] = k → pre[i-1] = pre[j] - k`。用哈希表记录每个前缀和出现的次数。（这题用 map/Hash 是前缀和 + 哈希的经典套路，超出本章范围但可以给学有余力的学生讲思路。）

5. 二维差分 + 二维前缀和综合：给 n×m 矩阵，q 次子矩阵加操作，最后输出整个矩阵。（参考答案：二维差分处理修改→二维前缀和还原。）
