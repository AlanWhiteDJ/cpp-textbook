## 🗺️ 教学流程

### 第 1 课时：cin 输入（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-5 分 | 导入 | "前几章的程序都是自言自语。今天让它们跟你聊天！" |
| 5-20 分 | 讲授 | cin 的基本用法。**箭头方向是关键**——cout << 是往外流，cin >> 是往里流。让学生站起来用手臂比划方向 |
| 20-35 分 | 实操 | 写第一个交互程序：问名字→输入→打招呼。每一步都在投影上先敲，学生跟着敲 |
| 35-45 分 | 扩展 | 多个 cin 串联输入。"输入两个整数，输出它们的和" |

### 第 2 课时：cout 控制（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-10 分 | 复习 | cin 用法回顾 |
| 10-30 分 | 讲授 | endl vs \n、串联输出、fixed + setprecision。重点演示：保留 2 位小数 vs 保留 6 位小数的区别 |
| 30-45 分 | 实操 | 算平均分程序——输入三科成绩，输出保留 2 位小数的平均分 |

### 第 3 课时：printf/scanf（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-5 分 | 导入 | "cin/cout 很方便，但有一种情况它不够好——要精确控制格式的时候" |
| 5-25 分 | 讲授 | printf 占位符表。让学生用 printf 重写前面 cout 的程序。重点：%d、%lld、%f、%.2f |
| 25-35 分 | 讲授 | scanf + & 规则。"& 不能忘！忘掉的后果——自己试试看"（让学生故意忘 &，看编译器怎么报错） |
| 35-45 分 | 对比 | 总结 cin/cout vs printf/scanf 的优劣。现阶段用 cin/cout，第二册再切 printf/scanf |

### 第 4-5 课时：练习 + 小测（90 分钟）

标准流程。

## 🔥 重点难点

### 重点

1. **cin >> 和 cout << 的方向**：不可逆。让肢体动作参与记忆
2. **printf 占位符对应关系**：%d→int、%lld→long long、%f→double。这是一道必须背下来的映射表
3. **scanf 的 &**：丢 & 编译不通过（或警告），但初学者往往看不懂报错

### 卡点突破

| 卡点 | 突破 |
|------|------|
| 方向混淆 | 口诀："cout <<：往外送（送客出门），cin >>：往里拉（拉人进门）" |
| printf 占位符 | 做一张口袋卡片让学生贴在电脑旁边 |
| scanf & | "& = address（地址），scanf 需要知道变量的地址才能往里面写。你可以不理解，但 & 不能忘" |
| 输入缓冲 | 用"收件箱"比喻：你发的消息先堆在收件箱里，按回车 = 一次性发送 |

## 💬 常见问答

**Q1："cin 和 scanf 到底哪个好？"**

A：就像筷子和叉子——都能吃饭。初学者用筷子（cin/cout）更简单。但吃牛排（处理大量数据）时，叉子（scanf/printf）更快。竞赛到后期几乎全用 scanf/printf。

**Q2："为什么 scanf 的 double 用 %lf，printf 的 double 用 %f？"**

A：这是一个 C 语言的历史遗留问题。不必深究原因，记住规则：scanf 读 double → %lf，printf 打 double → %f。

**Q3："用户输入的时候输错了怎么办？比如要求输数字，他输了一个字母。"**

A：好问题。这涉及到输入验证，后面会专门讲。现阶段假设用户"听话"，按要求输入。等基础打牢了再学防御。

## 🎲 课堂活动

### 活动：箭头体操（第 1 课时用）

**规则**：
- 全体起立
- 老师说"cout"，学生双臂向前推（数据出去）
- 老师说"cin"，学生双臂向内收（数据进来）
- 老师说"cout << age"，学生先推手、再喊"age 送出去"
- 越做越快，直到一片混乱大笑

**目的**：用身体记忆取代大脑死记。这个动作记忆能管整本书。

## ✅ 习题详解

### 基础题

**1. 填空题**
（1）cin  
（2）%d、%lld  
（3）&（取地址符）

**2. 代码改错**

三处错误：
1. `cin << x` → `cin >> x`（箭头方向反了）
2. `printf("%d", x)` 末尾缺分号
3. 缺少 `return 0;`

### 提高题

**3. 计算器（参考代码）**

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "请输入两个整数：";
    cin >> a >> b;
    cout << "和 = " << a + b << endl;
    cout << "差 = " << a - b << endl;
    cout << "积 = " << a * b << endl;
    cout << "商 = " << a / b << endl;
    cout << "余数 = " << a % b << endl;
    return 0;
}
```

### 挑战题

**5. 时间转换（参考代码）**

```cpp
#include <iostream>
using namespace std;

int main() {
    int total;
    cout << "请输入秒数：";
    cin >> total;
    int h = total / 3600;
    int m = (total % 3600) / 60;
    int s = total % 60;
    cout << total << " 秒 = " << h << " 小时 " << m << " 分 " << s << " 秒" << endl;
    return 0;
}
```
