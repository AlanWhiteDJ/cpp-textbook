## 🗺️ 教学流程

### 第 1 课时：嵌套循环入门（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-10 分 | 导入 | 在黑板画一个 5 行 7 列的星号矩形。问学生："用目前学的知识，怎么打印这个？"学生可能会说"写 5 行 cout"——追问："如果是 100 行 100 列呢？"——引出"一行里的 7 个星号本身也可以用一个循环来打印——循环套循环" |
| 10-28 分 | 讲授 | 使用板书的"表格推演法"。写 `for(int i=1; i<=3; i++) { for(int j=1; j<=4; j++) { cout << "*"; } cout << endl; }`。然后在黑板上画一个 3×4 的表格，每进入一次内层循环就在表格中填一个位置。关键口诀：**"外层走一步，内层跑一圈"**。外层 i=1 时，内层 j 从 1 到 4 跑完一圈→换行→外层 i=2……。让学生跟着在纸上画表格推演 |
| 28-38 分 | 互动 | 改变内层循环的范围让它依赖于外层变量：`for(int j=1; j<=i; j++)`——打印直角三角形。先预设输出让学生猜，再公布答案。让学生体会："内层循环的终止条件不一定是死的数字，可以是外层的变量——这就产生了变化的图形。" |
| 38-45 分 | 总结 | 口诀上墙："**外层管行，内层管列。外走一步，内跑一圈。**"这十二个字是嵌套循环的"心法"，整章都围着它转 |

### 第 2 课时：乘法表 + 模式训练（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 回顾 | 快速提问：外层 i 为 3 时，内层 `j<=i` 循环执行了几次？答案：3 次 |
| 8-22 分 | 讲授 | 九九乘法表实战：`for(i=1; i<=9; i++) { for(j=1; j<=i; j++) { cout << j << "*" << i << "=" << i*j << "\t"; } cout << endl; }`。注意这里 i 和 j 的位置：习惯上内层输出 `j*i`，先 j 后 i。重点：制表符 `\t` 让列对齐 |
| 22-35 分 | 编程练习 | 三个图形题递进：①打印倒直角三角形（`*****` 到 `*`）；②打印等腰三角形（考虑空格+星号）；③打印菱形（上半下半分开处理）。提示学生：先想清楚"每行几个空格、几个星号"，再写内层循环 |
| 35-42 分 | 讨论 | "图形打印和数学有什么联系？"——每行的空格数和星号数与行号 i 之间存在函数关系。如正三角形第 i 行有 i 个星号，这就对应了内层循环 `j<=i`。引导学生发现：写出循环的关键是**找到行号 i 和该行输出数量之间的数学关系** |
| 42-45 分 | 总结 | "图形打印是嵌套循环的'基本功'——练好它，你就能把任何规律性的二维图案变成代码。" |

### 第 3 课时：break 与 continue（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 导入 | 场景引入：①"找第一个能被 17 整除的数——找到了就不用继续找了"（break）；②"打印 1 到 100，但跳过所有 7 的倍数"（continue）。这两个需求用之前的知识很别扭 |
| 8-25 分 | 讲授 | **break**：立即终止整个循环，程序跳到循环外的下一条语句。演示：`for(i=1; i<=100; i++) { if(i%17==0) { cout<<i; break; } }`——找到第一个就停。**continue**：跳过本次循环的剩余部分，直接进入下一次迭代。演示：`for(i=1; i<=20; i++) { if(i%7==0) continue; cout<<i<<" "; }`——7 和 14 被跳过 |
| 25-38 分 | 对比实验 | 同一个嵌套循环里测试 break 和 continue。`for(i=1; i<=3; i++) { for(j=1; j<=5; j++) { if(j==3) break; cout<<j; } cout<<endl; }`——每行输出 12，因为内层在 j=3 时 break 退出。把 break 换成 continue：每行输出 1245（3 被跳过但循环继续）。强调：**break 和 continue 只作用于它们所在的最近一层循环**。写在内存 = 只影响内层；写在外层 = 影响外层 |
| 38-45 分 | 总结 | 三句话总结：break = 彻底不干了（退出整层循环）。continue = 这一趟不干了（跳进下一次）。return = 整个函数都不干了（以后学）。板书三件套区别表 |

### 第 4 课时：综合练习 + 调试技巧（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-15 分 | 集体纠错 | 典型嵌套错误：①内外层用了同一个变量名（都用 i）导致逻辑混乱；②换行 `cout<<endl;` 写在内层循环里——每打一个星号就换行；③内层循环条件写错，如 `for(j=1; j<=5; j--)` 死循环；④break 写在错误的层级——想跳出外层却只跳了内层。逐题讨论 |
| 15-30 分 | 独立编程 | 三道综合题：①打印数字金字塔（每行数字递增）；②找质数：输出 100-200 之间所有质数（嵌套循环+break）；③打印空心矩形（边框星号，内部空格——用 if 判断位置） |
| 30-45 分 | 调试教学 | 教嵌套循环的调试技巧：①在循环体内加 `cout << "i=" << i << " j=" << j << endl;` 实时看变量值；②先用小数值测试（如 3 行 3 列），确认逻辑没问题再放大；③纸笔 trace——在纸上画表格，逐行填写变量值。这是竞赛中最朴素的 debug 方式 |

## 🔥 重点难点

### 教学重点

1. **"外层走一步，内层跑一圈"的执行模型**：这十二个字必须刻进学生的肌肉记忆。所有嵌套循环的理解都建立在这个心智模型上。
2. **循环变量与图形规律的映射**：打印图形不是靠"感觉"，而是靠数学关系。第 i 行的空格数 = f(i)，星号数 = g(i)。找到 f 和 g 就找到了代码。
3. **break/continue 的作用域**：只影响最近的一层循环。这是学生最容易误判的地方——以为写了 break 就能逃出整个嵌套结构。

### 常见卡点 + 突破策略

| 卡点 | 表现 | 突破方法 |
|------|------|----------|
| 搞不清内外层变量关系 | 把内层和外层的循环变量都写成 i，或者分不清谁在变化 | 强制规范：外层永远用 i，内层永远用 j，第三层用 k——"ijk 铁律"。然后在纸上画表格，行标 i，列标 j |
| 换行位置不对 | endl 写在错误的地方导致图形散架 | 口诀："内层结束后再换行"——endl 写在内层 `}` 和外层 `}` 之间。用颜色标记：内层循环体用蓝色，endl 用红色 |
| continue 写成 break | 把"跳过"误写成了"停止" | 语义联想：continue = 继续（下一轮），break = 打破（整个循环）。吃早餐时筷子掉了——continue 是换一双继续吃，break 是不吃了走人 |

## 💬 常见问答

**Q1："嵌套循环最多能套几层？"**

A：语法上没有限制，理论上可以无限嵌套。但实践中有两个约束：①你的脑子——三层以上就很难追踪了；②时间——每多一层，总执行次数是乘法关系。三层 100×100×100 就是一千万次，再多就慢了。信息学竞赛中，三层很常见，四层就比较少了。

**Q2："break 能不能一下子跳出所有循环？"**

A：不能。C++ 的 break 只跳出一层。如果要跳出多层，可以用一个布尔标志变量：内层设置 `flag = true; break;`，外层检查 `if(flag) break;`。这是初学者最干净的写法（另一种是用 goto，但不推荐）。

**Q3："continue 和 break 哪个更常用？"**

A：break 远多于 continue。break 的场景非常多（提前找到结果、检测到错误、用户退出等）。continue 的场景相对少（过滤某些特殊情况）。但两个都要掌握——出题人会考 continue，而且 continue 在代码优化中有独特作用（减少嵌套层级）。

**Q4："为什么打印菱形要分上下两部分？"**

A：因为每行星号数的变化方向不一样：上半部分行号 i 增加，星号增加；下半部分行号 i 增加，星号减少。一个统一的公式也能写出来（用数学绝对值），但对初学者来说，分成两个清晰的循环更容易理解和调试。等你熟练了再追求"一行公式"的优雅。

## 📋 本章小测（15 分钟，共 7 题）

**1. 填空题**（2 分）
嵌套循环的核心口诀是：外层走________，内层跑________。

**2. 选择题**（2 分）
以下代码输出几行星号？
```cpp
for (int i=1; i<=3; i++) {
    for (int j=1; j<=i; j++) cout << "*";
    cout << endl;
}
```
A. 3行星号, 每行3个　　B. 3行星号, 分别为1/2/3个　　C. 6行星号　　D. 1行星号, 6个

**3. 填空题**（2 分）
break 语句的作用是________；continue 语句的作用是________。

**4. 判断输出题**（3 分）
写出以下代码输出：
```cpp
for (int i=1; i<=3; i++) {
    for (int j=1; j<=4; j++) {
        if (j == 3) continue;
        cout << j;
    }
    cout << endl;
}
```

**5. 编程题**（4 分）
用嵌套循环打印一个 4 行的等腰三角形（第 1 行 1 个星号居中，第 2 行 3 个……）。

**6. 分析题**（2 分）
以下代码中 break 跳出的是哪一层循环？
```cpp
for (int i=1; i<=5; i++) {
    for (int j=1; j<=5; j++) {
        if (i * j > 10) break;
        cout << i*j << " ";
    }
    cout << endl;
}
```

**7. 改错题**（2 分）
下面的代码想打印 3 行 5 列的星号矩阵，但输出不对，哪里错了？
```cpp
for (int i=1; i<=3; i++) {
    for (int j=1; j<=5; j++) {
        cout << "*";
        cout << endl;
    }
}
```

---

**参考答案：**

1. **一步**、**一圈**
2. **B**。i=1 时输出 1 个，i=2 时输出 2 个，i=3 时输出 3 个。总计 3 行，逐行递增。
3. break：**立即终止当前所在的那一层循环**；continue：**跳过本次循环的剩余部分，直接进入下一次迭代**。
4. 输出：
```
124
124
124
```
每行 j 从 1 到 4，但 j=3 时 continue 跳过 cout，所以每行输出 1、2、4。
5. 参考答案：
```cpp
for (int i=1; i<=4; i++) {
    for (int k=1; k<=4-i; k++) cout << " ";  // 空格
    for (int j=1; j<=2*i-1; j++) cout << "*";  // 星号
    cout << endl;
}
```
评分：空格循环正确 2 分，星号循环正确 1 分，换行位置正确 1 分。
6. break 跳出的是**内层循环**（j 循环）。break 只作用于它所在的最近一层循环。外层 i 循环不受影响，会继续 i=2, 3, 4, 5。
7. 错误：`cout << endl;` 写在了内层循环里面——每输出一个星号就换行。应该把 endl 移到内层循环的外面、外层循环的里面（即 `} cout << endl; }`）。

## 🎮 不插电活动

### 活动：时钟扮演（第 5 课时用）

**道具**：
- 三个大标签（A4 纸），分别写"秒针""分针""时针"
- 粉笔在地上画一个大圆圈（表盘），标注 12 个刻度位置
- 三个计数器（小凳子或椅子）分别写上 0-59（秒）、0-59（分）、1-12（时）

**角色分配**：
- 3 名学生分别扮演时针、分针、秒针
- 1 名学生当"报时员"（站在旁边大声报当前时间）

**游戏流程**：

**第一轮——演示三层嵌套**：
1. 三位扮演者站到各自起始位置：时针指 12、分针指 12、秒针指 12
2. 老师喊"开始"——秒针开始移动：
   - 秒针每走 1 格（1 秒），报时员喊一声"秒针动了！"
   - 秒针走完一圈（60 格 → 回到 12），分针走 1 格（1 分钟）
   - 分针走完一圈（60 格），时针走 1 格（1 小时）
3. 简化版：只跑一轮——秒针走 5 格 = 分针走一步一小格（模拟 5 秒）
4. 老师喊"停"后，问全班："秒针动了 5 次，分针动了 1 次。如果用嵌套循环表示，谁在最外层？"引导学生发现：
   - 时针 = 最外层（动得最慢）
   - 分针 = 中层
   - 秒针 = 最内层（动得最快）

**第二轮——用代码表达**：
- 老师在黑板逐行写出三层循环框架：
```cpp
for (int hour = 1; hour <= 12; hour++) {       // 时针
    for (int min = 0; min < 60; min++) {        // 分针
        for (int sec = 0; sec < 60; sec++) {    // 秒针
            cout << hour << ":" << min << ":" << sec << endl;
        }
    }
}
```
- 每次老师写完一层，就让对应扮演者演示一次"跑一圈"
- 关键提问："如果三层循环都跑完，cout 总共执行了多少次？"（12 × 60 × 60 = 43200 次）

**第三轮——break 演示**：
- "如果闹钟在 7:30 响，怎么让循环停下来？"
- 让秒针走到 7:30:00，然后喊"停！"，秒针/分针/时针全部停止
- 对应代码：`if (hour==7 && min==30 && sec==0) break;`——但只能跳出秒针层！
- 引发讨论：怎么同时跳出三层？→ 引入标志变量的概念

**教学要点**：
- 时钟是最经典的三层嵌套案例——每个孩子都认识时钟，一眼就能理解"秒动 60 次分才动 1 次"
- 这个活动也隐藏了"时间复杂度"的萌芽：三层嵌套 = 12×60×60 = 43200 次，让学生直观感受嵌套带来的爆炸式增长
- 总结板书：时钟三针的嵌套关系是"慢在外、快在内"——外层变化频率低，内层变化频率高。这是所有嵌套循环的共同模式

for (int i=0; i<3; i++) {

for (int i=1; i<=100; i++) {

评分：外层范围 100-200 正确 1 分，内层检测因子正确 1 分，发现非质数 break 正确 1 分，计数正确 1 分。

**5. 打印空心正方形**

## 📦 补充题库

1. 用嵌套循环输出如下图案（数字回字形）：
```
11111
12221
12321
12221
11111
```
（参考答案：每个位置的值 = 该位置到四条边的最短距离 + 1。`min(min(i, n-i+1), min(j, n-j+1))`。）

2. 如何用标志变量（flag）在嵌套循环中跳出所有层？（参考答案：`bool found=false; for(...) { for(...) { if(条件) { found=true; break; } } if(found) break; }`）

3. 用嵌套循环实现冒泡排序的"一趟"：把数组中最大的数冒到最后。（参考答案：内层 j 从 0 到 n-i-1，两两比较交换。这是排序算法的基础，为后续学习铺垫。）

4. break 和 continue 能用在 if 语句中吗？（参考答案：不能。break 和 continue 只能用在循环（for/while/do-while）或 switch 中。在单独 if 里写 break 会编译错误。）

5. 打印杨辉三角的前 8 行。（参考答案：外层行循环，每行的首尾元素为 1，中间元素 = 上一行同列 + 上一行前一列。`a[i][j] = a[i-1][j-1] + a[i-1][j]`。这题可以只讲思路不要求实做，属于挑战题。）
