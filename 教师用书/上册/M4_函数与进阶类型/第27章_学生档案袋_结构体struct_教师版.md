## 🗺️ 教学流程

### 第 1 课时：为什么要用结构体 + 定义语法 + 不插电活动（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 问题导入 | 老师拿出 3 张卡片，每张写有"姓名、年龄、分数"三个信息。"如果我要管理全班 40 个学生的信息，用数组怎么存？"——学生回答："三个数组！names[40]、ages[40]、scores[40]。"老师问："如果要对分数排序，要同时移动三个数组——写起来麻烦还容易对不齐。有没有办法把姓名、年龄、分数**打包**成一个东西？" → 引出结构体 |
| 8-15 分 | 🎮 不插电活动 | **"档案袋制作"**（详见活动区）。每个学生拿一张信息卡，写自己的名字、年龄、上次数模考分数。然后每 5 人一组，把卡片装进同一个文件袋（struct）。老师指令："按年龄从小到大排队！"——各组成员按年龄重新排列卡片，整袋提交 |
| 15-28 分 | 讲授 | 结构体定义语法四步：①关键字 `struct` ②结构体名（自定义，如 `Student`）③花括号 `{}` 内写成员（成员类型 + 成员名 + 分号）④末尾分号不能忘。板书示例：
```cpp
struct Student {
    string name;
    int age;
    double score;
};
```
拆解：`Student` 是**新的数据类型**（就像你发明了一种新的 `int`），`name`/`age`/`score` 是它的成员（就像一个档案袋里有三栏） |
| 28-38 分 | 类比 | "`struct Student` 是什么？"——它是一张**空白表格模板**，定义了有哪些栏目。模板本身不是数据。当你写 `Student s1;`，相当于拿模板复印了一张空表格，然后可以在上面填数据。表格上的每个栏目用 `.` 访问：`s1.name = "张三";` |
| 38-45 分 | 练习 | 每个学生在纸上写出：定义一个 `struct Book`，包含书名（`string`）、价格（`double`）、页数（`int`）三个成员。同桌互查语法（特别注意分号有没有漏） |

### 第 2 课时：结构体变量 + 成员访问（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-5 分 | 回顾 | "结构体的定义和变量的声明有什么区别？"——`struct S { ... };` 是定义模板，`S s;` 是创建实例 |
| 5-15 分 | 讲授 | **创建结构体变量的三种方式**：①先声明后赋值：`Student s1; s1.name = "张三"; s1.age = 14; s1.score = 95.5;` ②声明时初始化（C++11）：`Student s2 = {"李四", 15, 88.0};` ③用花括号初始化：`Student s3{"王五", 14, 92.0};`。**成员访问**：变量名 + `.` + 成员名。`s1.name` |
| 15-25 分 | 代码演示 | 老师现场写完整程序：定义 Student 结构体，创建 3 个学生变量，逐个打印信息。然后演示常见错误：①忘了 `.` 直接写 `name`（编译器不认识）②把结构体成员当普通变量用了 |
| 25-38 分 | 练习 | 学生上机：①定义 `struct Point { double x, y; }` 创建两个点，计算两点距离 ②定义 `struct Rectangle { double width, height; }`，写函数 `double area(Rectangle r)` 返回面积 |
| 38-45 分 | 讨论 | "结构体和数组有什么异同？"——同：都能存多个数据。异：数组里的所有数据**类型相同**，结构体里的数据**类型可以不同**；数组用下标 `[i]` 访问，结构体用 `.成员名` 访问 |

### 第 3 课时：结构体数组 + 结构体与函数（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-8 分 | 问题引入 | "一个 `Student` 变量只能存一个学生的信息。40 个学生怎么办？"——`Student class[40];` 这就是结构体数组。每个元素 `class[i]` 是一个完整的 Student 对象，`class[i].name` 访问第 i 个学生的名字 |
| 8-22 分 | 讲授 | **结构体数组操作**：①遍历打印所有学生信息 ②找出最高分的学生 ③计算平均分。老师逐段演示代码。关键点：`class[i].score`——先找到第 i 个学生（`class[i]`），再取他的分数（`.score`），顺序不能反 |
| 22-32 分 | 讲授 | **结构体作为函数参数**：`void printStudent(Student s)`——结构体可以像 int 一样传给函数（默认值传递，会复制整个结构体）。如果想在函数里修改结构体：`void updateScore(Student &s, double newScore)`——用引用传递（第 24 章知识） |
| 32-42 分 | 上机 | 学生完成：①创建 `Student arr[5]`，输入 5 个学生的信息 ②写函数 `Student findBest(Student arr[], int n)` 返回最高分学生 ③写函数 `double averageScore(Student arr[], int n)` 返回平均分 |
| 42-45 分 | 总结 | "结构体让我们可以把相关联的数据组织在一起，像一个整体一样传递和处理。下一节我们玩个更厉害的——给结构体数组排序！" |

### 第 4 课时：结构体排序 + 嵌套结构体（45 分钟）

| 时间 | 环节 | 操作 |
|------|------|------|
| 0-10 分 | 回顾+热身 | 快速回顾冒泡排序（来自 M3）。问："如果要对 Student 数组按 score 排序，冒泡排序的 `if (arr[j] > arr[j+1])` 要改成什么？"——`if (arr[j].score > arr[j+1].score)` |
| 10-25 分 | 讲授+演示 | **结构体排序**：对 Student 数组按分数从高到低冒泡排序。交换时：`swap(arr[j], arr[j+1])`——**整个结构体一起交换**，不需要分别交换 name、age、score。这是一个重要的认知：结构体变量可以整体赋值 |
| 25-35 分 | 讲授 | **嵌套结构体**（选讲）：结构体的成员可以是另一个结构体。例如：`struct Date { int year, month, day; }; struct Student { string name; Date birthday; double score; };`。访问嵌套成员：`s1.birthday.year`。类比：档案袋里还有一个小档案袋 |
| 35-45 分 | 擂台赛 | **"最快排序手"**：每组分到一个 Student 数组（纸卡），按老师指定的字段（第一轮按年龄、第二轮按分数）用冒泡排序排队。最快排好且正确的小组获胜 |

## 🔥 重点难点

### 教学重点

1. **结构体的定义语法**：`struct` 关键字 + 花括号 + 分号。分号是学生最容易忘的，必须反复强调。
2. **成员访问运算符 `.`**：区分"结构体变量名"和"成员名"的层级关系。
3. **结构体数组与排序**：按某个字段排序时，交换的是整个结构体，不是单个字段。

### 常见卡点 + 突破策略

| 卡点 | 表现 | 突破方法 |
|------|------|------|
| 忘记结构体定义末尾的分号 | `struct S { int x; }`（少了 `;`） | 口诀："**struct 后面花括号，花括号完加分号**"。形成条件反射：写完 `}` 马上按 `;` |
| 混淆结构体类型名和变量名 | `Student.name = "张三";`（Student 是类型不是变量） | 类比：`int = 5;` 对吗？不对，是 `int x = 5;`。同理，`Student` 是类型名，需要先声明变量：`Student s1; s1.name = "张三";` |
| 排序时只交换了排序字段 | 交换了 `score` 但没交换 `name` 和 `age`——导致数据错位 | 强调：交换时要交换**整个结构体变量**（`swap(arr[j], arr[j+1])` 或整个变量赋值），不是只交换那个字段。这样所有字段一起移动，数据不会乱 |

## 💬 常见问答

**Q1："结构体 struct 和第 9 章学的 class 有什么区别？"**

A：在 C++ 中，`struct` 和 `class` 的核心区别只有一点：默认访问权限不同。`struct` 的成员默认是 `public`（公开），`class` 的成员默认是 `private`（私有）。在其他方面（都能有成员变量、成员函数、构造函数等），它们几乎一样。实际上，很多 C++ 程序员习惯用 `struct` 表示"纯数据结构"（就是一堆数据放一起），用 `class` 表示"有行为的对象"。

**Q2："结构体能在定义的时候直接给成员赋默认值吗？"**

A：C++11 开始可以！例如：`struct Student { string name = "未知"; int age = 0; double score = 0.0; };`。这样创建 `Student s;` 时，成员会自动获得默认值。这非常方便，但不是所有竞赛环境都支持（取决于 C++ 标准版本），使用前确认评测机的标准。

**Q3："结构体数组能不能用 sort 排序？怎么告诉 sort 按哪个字段排？"**

A：可以！需要写一个比较函数（或 lambda 表达式）。例如按分数排序：`sort(arr, arr+n, [](Student a, Student b) { return a.score > b.score; });`。这是进阶用法，会在算法专题中详细讲解。初学阶段建议先用冒泡排序理解原理。

**Q4："能不能在结构体里放数组？"**

A：能。例如：`struct Student { string name; int scores[5]; };`——每个学生有 5 科成绩。访问：`s1.scores[2]`（先 `.scores` 拿到数组，再 `[2]` 取第 3 个元素）。

## 📋 本章小测（18 分钟，闭卷）

### 一、填空题（每空 2 分，共 8 分）

1. 定义结构体的关键字是 __________。
2. 访问结构体变量的成员使用 __________ 运算符。
3. 结构体定义末尾的 __________ 符号最容易遗漏。
4. `struct Point { int x, y; }; Point p;` 中，`Point` 是 __________，`p` 是 __________。

### 二、改错题（每题 6 分，共 12 分）

5. 以下代码有三处语法错误，请找出并改正：
```cpp
struct Student {
    string name
    int age;
    double score
}
```

6. 以下代码有什么逻辑错误？
```cpp
Student a = {"小明", 14, 88.5};
Student b = {"小红", 15, 92.0};
// 想交换 a 和 b 的分数
double t = a.score;
a.score = b.score;
b.score = t;
cout << a.name << " " << a.score;
```

### 三、写代码（每题 10 分，共 20 分）

7. 定义一个 `struct Car`，包含品牌（`string brand`）、价格（`double price`）、排量（`double engine`）三个成员。然后创建一个 Car 变量并初始化。
8. 已有 `Student arr[100]` 和 `int n`（实际学生数），写一段代码找出年龄最小的学生并打印其所有信息。

---

### 答案

1. `struct`
2. `.`（点运算符）
3. 分号（`;`）
4. 类型名（或自定义数据类型），变量名
5. 三处错误：① `string name` 后少分号 ② `double score` 后少分号 ③ `}` 后少分号
6. 只交换了分数，但名字和年龄没有跟着换——现在小明的名字配上了小红的分数，数据错位了。应交换整个结构体变量：`swap(a, b);` 或 `Student t = a; a = b; b = t;`
7. ```cpp
struct Car {
    string brand;
    double price;
    double engine;
};
Car myCar = {"比亚迪", 150000, 1.5};
```
8. ```cpp
int minIdx = 0;
for (int i = 1; i < n; i++) {
    if (arr[i].age < arr[minIdx].age) {
        minIdx = i;
    }
}
cout << arr[minIdx].name << " "
     << arr[minIdx].age << " "
     << arr[minIdx].score;
```

## 📦 补充题库

1. 定义一个 `struct Triangle`，包含三个顶点（用 `Point` 结构体表示），写函数计算三角形周长。（参考答案：使用两点距离公式计算三条边长，求和。）

2. 以下代码输出什么？
```cpp
struct S { int x; int y; };
S a = {1, 2};
S b = a;
b.x = 10;
cout << a.x << " " << b.x;
```
（参考答案：`1 10`。结构体变量赋值是**整体复制**（和 int 一样），之后 b 的修改不影响 a。）

3. 写一个函数 `int countAbove(Student arr[], int n, double threshold)`，返回分数超过 threshold 的学生人数。（参考答案：遍历数组，`if (arr[i].score > threshold) cnt++;`。）

4. 定义一个 `struct Time`（时、分、秒），写函数 `Time addTime(Time t1, Time t2)` 返回两个时间相加的结果（注意进位）。（参考答案：逐秒、分、时相加，分别处理进位。）

5. 结构体数组能不能用 `memset` 初始化为 0？（参考答案：如果结构体里只有基本类型（int、double 等），可以。但如果包含 `string` 类型，不能——`string` 不是平凡类型，`memset` 会破坏其内部结构。安全的做法是用 `={}` 初始化或 `fill`。）
