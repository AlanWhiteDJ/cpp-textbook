# 信息学竞赛 C++ 教材框架

> **面向**：小学高年级（4-6 年级）+ 初中低年级（7-8 年级）
> **对标**：2025 NOI 大纲（入门级 + 提高级）· CCF GESP C++ 1-8 级
> **范围**：NOI 入门级 → 提高级，不含 NOI 级
> **纯框架**：只定结构、阶段、模块、课时区间，不展开具体内容

---

## 一、整体架构

```
┌─────────────────────────────────────────────────┐
│                  第一阶段                        │

│                ≈ 250 课时                        │

└─────────────────────────────────────────────────┘
```

## 六、与 OI-Wiki 学习路线的关系

本框架的模块排布参考 OI-Wiki 学习路线但做了以下调整：

| OI-Wiki 路线 | 本框架调整 | 原因 |
|-------------|-----------|------|
| C++ 语言基础 → 直接跳 CSP-J | 拆为模块 1（语言 70 课时）+ 模块 2（进阶 50 课时） | 小学生需要更长的语法消化期 |
| 排序放在 CSP-J 早期 | 排序放在第一阶段中段（模块 3.1） | 需先掌握数组和函数才能理解排序 |
| 图论全部放在 CSP-S | 图的概念和遍历提前到第二阶段早期 | 与 GESP 7 级对齐，且概念可早接触 |
| DP 不做阶段区分 | 基础 DP（模块 6.1-6.4）与进阶 DP（6.5-6.8）分两段 | 避免认知过载 |
| 倍增法放在高阶 | 倍增法放在第一阶段（模块 3.6） | NOI 大纲标为【4】，可在入门级引入 |
