# 第37章 第3课时：高精度乘法——单精与高精的两种策略
**时长**：15-20分钟 | **对应教材**：第37章 37.4

## 🎯 本课时目标
- 掌握高精 × 单精乘法（进位用 long long）
- 掌握高精 × 高精乘法的"先全部累加再统一进位"策略
- 理解结果长度的计算：la + lb

## 📝 讲稿正文

乘法的难度比加减法高一个台阶——因为会有"错位相加"。

先看简单的：**高精 × 单精**——一个大数乘以一个普通 int。

思路：跟手工一样，大数的每一位乘以 b，加上进位，取个位，进位带走。

```cpp
int mulSingle(int a[], int la, int b, int c[]) {
    int lc = la;
    long long carry = 0;  // 用 long long！防止溢出
    for (int i = 1; i <= lc; i++) {
        carry += (long long)a[i] * b;
        c[i] = carry % 10;
        carry /= 10;
    }
    while (carry > 0) {
        c[++lc] = carry % 10;
        carry /= 10;
    }
    return lc;
}
```

⚠️ 注意 `carry` 用 `long long`！a[i] 最大 9，b 最大约 10⁹，a[i]×b 可能达到 9×10⁹，普通 int 溢出。所以必须用 `long long`。

再看**高精 × 高精**——两个大数相乘。这是"先全部累加再统一进位"的策略：

```cpp
int mul(int a[], int la, int b[], int lb, int c[]) {
    memset(c, 0, sizeof(int) * (la + lb + 5));
    int lc = la + lb;  // 乘积最大长度

    for (int i = 1; i <= la; i++) {
        for (int j = 1; j <= lb; j++) {
            c[i + j - 1] += a[i] * b[j];  // 先全部累加，不进位
        }
    }

    // 统一进位
    for (int i = 1; i <= lc; i++) {
        c[i + 1] += c[i] / 10;
        c[i] %= 10;
    }

    while (lc > 1 && c[lc] == 0) lc--;  // 去除前导零
    return lc;
}
```

为什么 a[i] × b[j] 贡献到 `c[i+j-1]`？因为 a 的第 i 位（10ⁱ⁻¹）乘以 b 的第 j 位（10ʲ⁻¹）结果是 10ⁱ⁺ʲ⁻²——对应 c 的第 i+j-1 位。

为什么要"先全部累加再统一进位"？（停顿）因为 c[i+j-1] 可能从多个 (i,j) 对收到贡献。如果每步都进位，可能打乱后续累加。先全加完，再一次性处理所有进位——代码清晰且不容易出错。

两个 n 位数相乘，结果最多 2n 位。所以 `c` 数组大小至少 `la + lb + 2`。

这道题是竞赛中的"手写大数乘法"——如果你能默写出这个`先累加再进位`的模式，任何高精度乘法都能应对。

## 🖥️ 板书/屏幕要点
- **高精×单精**：逐位乘，carry 用 long long
- **高精×高精**：`c[i+j-1] += a[i]*b[j]` → 统一进位
- **结果长度**：最大 la+lb，最小 la+lb-1
- **注意**：memset 清零 c，去前导零

## ✏️ 随堂练习
实现 2 的 n 次方（n≤100）。用高精×单精反复乘 2。输出 2¹⁰⁰ 的精确值。

## 📌 小结
高精×单精：逐位乘+进位，carry 用 long long。高精×高精：a[i]×b[j] 累加到 c[i+j-1]，最后统一进位。结果最多 la+lb 位。
