# 第35章 第3课时：海底高铁——差分实战经典
**时长**：15-20分钟 | **对应教材**：第35章 35.2（海底高铁）

## 🎯 本课时目标
- 用差分统计路径经过次数
- 理解"先统计次数、再计算花费"的两步策略
- 掌握差分在区间计数类问题中的应用

## 📝 讲稿正文

这节课用差分解决一道经典竞赛题——**海底高铁**。

题目大意：n 个地铁站，小明从站 1 出发依次经过 p₁, p₂, ..., p_m 站。每两个相邻站之间有两种出行方式——纸质票（每次固定价格 A）和 IC 卡（交工本费 C 后每次只需 B 元，B < A）。求最小总花费。

核心思路：先算出每条路段被经过多少次，然后对每条路段选更便宜的方式。

第一步：统计每条路段经过次数。如果小明从站 3 到站 5，他经过了 3→4 和 4→5 两段——但你不能用循环把每一段都 +1（那样 m 很大时会超时）。

用差分！把"路段 i→i+1 被经过一次"转化为差分操作 `cnt[i]++`、`cnt[i+1]--`（这里要注意：从 from 到 to，经过的路段是 from 到 to-1）：

```cpp
long long cnt[100005] = {0};

for (int k = 1; k < m; k++) {
    int from = p[k], to = p[k+1];
    if (from > to) swap(from, to);  // 统一从小到大
    cnt[from]++;    // 差分起点
    cnt[to]--;      // 差分终点（注意不是 to+1）
}
// 前缀和还原
for (int i = 1; i < n; i++)
    cnt[i] += cnt[i-1];
```

为什么 `cnt[to]--` 而不是 `cnt[to+1]--`？因为经过的路段是 `[from, to)`（左闭右开）——从 from 到 to 需要经过 from→from+1, ..., to-1→to，但不包括 to→to+1。所以差分区间是 `[from, to-1]`，终点减在 to 位置。

第二步：计算最小花费。

```cpp
long long ans = 0;
for (int i = 1; i < n; i++) {
    // cnt[i] 是路段 i→i+1 被经过的次数
    ans += min(cnt[i] * A[i], C[i] + cnt[i] * B[i]);
    // A=纸质票价格, B=IC卡单价, C=IC卡工本费
}
```

`cnt[i] * A[i]` 是全用纸质票的价钱，`C[i] + cnt[i] * B[i]` 是办 IC 卡的总花费。取两者较小值。

这道题的巧妙之处在于：**先不问钱，先用差分算出每条路的"人气"**，然后才对每条路做出最优选择。如果没有差分，统计"人气"就需要 O(nm) 的时间——铁定超时。

总结差分应用的套路：**观察到要多次对区间做相同操作（+1、+val……），就用差分把 O(n) 的单次操作变成 O(1)**。

## 🖥️ 板书/屏幕要点
- **差分统计经过次数**：`cnt[from]++; cnt[to]--;`
- **前缀和还原**：`cnt[i] += cnt[i-1]`
- **选择最优方式**：`min(cnt[i]*A[i], C[i]+cnt[i]*B[i])`
- **注意 long long**：答案可能很大

## ✏️ 随堂练习
简化版：3 个站，小明从 1→3→2→3，路段 (1,2) 和 (2,3) 各有票价 A=[5,3], B=[2,1], C=[10,8]。求最小花费。

## 📌 小结
海底高铁是差分的经典应用。先用差分 O(m) 统计每条路段经过次数，再 O(n) 计算每个路段的最优花费。差分把"区间 +1"从 O(n) 变为 O(1)，是这道题的核心。
