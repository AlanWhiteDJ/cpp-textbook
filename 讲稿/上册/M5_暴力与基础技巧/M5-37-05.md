## 📝 讲稿正文

这是 M5 模块的最后一课。我们先讲一个进阶技巧——**压位**，然后对高精度运算和整个 M5 模块做总复习。

高精度运算每次只存 0~9 的一位数，效率不太高——100 位的大数需要 100 个数组元素。如果能一次存 4 位呢？每个数组元素存 0~9999，100 位大数只需要 25 个元素，运算速度提升约 4 倍。

这就是压位：

```cpp
const int BASE = 10000;   // 每个元素存 4 位
const int WIDTH = 4;      // 输出时补齐 4 位

// 进位：c[i] % BASE；进位：c[i] / BASE
// 输出时用 printf("%04d", c[i]) 补齐前导零
```

原理完全一样，只是把底数从 10 换成了 10000。初学时不强求，但要知道有这个方法——很多竞赛高分选手都在用。

现在回顾高精度运算的四种运算：

| 运算 | 方向 | 核心变量 | 特殊处理 |
|------|------|----------|----------|
| 加法 | 低→高 | carry | 最后进位可能延长结果 |
| 减法 | 低→高 | borrow | 先比大小、去前导零 |
| 乘法 | 低→高 | long long carry / c[i+j-1] | 先累加再进位 |
| 除法 | 高→低 | 余数 r | 去前导零 |

记住三个黄金原则：
1. 倒序存储（个位在 a[1]）
2. 加减乘从低到高，除从高到低
3. 结果要去前导零

最后，让我们俯瞰整个 M5 模块「暴力与基础技巧」——你从一个只会写循环的新手，成长为掌握了 8 个算法技巧的入门竞赛选手：

- **枚举**：逐个尝试，三步模板 + 三大优化
- **模拟**：规则翻译，四步法 + 五种类型
- **排列/子集**：next_permutation + 二进制枚举
- **剪枝**：提前淘汰不可能分支
- **双指针**：对撞/快慢/滑动窗口，O(n²)→O(n)
- **前缀和/差分**：空间换时间，预处理 O(n) 查询 O(1)
- **离散化**：坐标压缩，三步走
- **高精度**：突破数据类型，用数组模拟竖式

这八个技巧，只要你掌握了，NOIP 入门组的前两题基本稳了。记住，算法不在多，在于**熟练**和**灵活**。下个模块我们进入排序与复杂度——正式讨论"什么样的算法是好算法"。

## 🖥️ 板书/屏幕要点
- **压位**：BASE=10000, WIDTH=4, 输出用 `%04d`
- **四种运算对比**：方向、核心变量、特殊处理
- **M5 八大技巧**：枚举→模拟→排列子集→剪枝→双指针→前缀和差分→离散化→高精度
- **黄金三原则**：倒序存储、方向规律、去前导零
