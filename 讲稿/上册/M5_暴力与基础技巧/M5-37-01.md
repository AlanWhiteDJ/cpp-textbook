## 📝 讲稿正文

同学们，你们知道 `long long` 最大能存多少吗？约 9×10¹⁸，也就是 19 位数字。但如果题目要算 100 的阶乘——有 158 位——或者两个 50 位数相乘——结果有 100 位——`long long` 就完全装不下了。

这时候怎么办？回到小学！**像小学生一样列竖式**。

你拿出一张纸，把两个大数字逐位对齐，从个位开始一位一位加、一位一位进位。代码要做的就是把这张纸上的过程用数组模拟出来。

第一步是**存储**。通常用字符串读入，然后转成整型数组——而且是**倒序存储**：下标 1 存个位，下标 2 存十位……这样进位时往高处加更方便。

```cpp
// 把字符串 s 转成整型数组 a（倒序，下标 1 存个位）
int strToArr(string s, int a[]) {
    int len = s.size();
    for (int i = 0; i < len; i++) {
        a[len - i] = s[i] - '0';  // s[0]（最高位）→ a[len]（最高位）
    }
    return len;
}
```

比如 "123" → a[1]=3, a[2]=2, a[3]=1。

第二步是**高精度加法**——和手工列竖式一模一样：从个位开始逐位相加，逢 10 进 1。

```cpp
int add(int a[], int la, int b[], int lb, int c[]) {
    int lc = max(la, lb);
    int carry = 0;
    for (int i = 1; i <= lc; i++) {
        int ai = (i <= la) ? a[i] : 0;
        int bi = (i <= lb) ? b[i] : 0;
        c[i] = ai + bi + carry;
        carry = c[i] / 10;    // 计算进位
        c[i] %= 10;           // 当前位保留个位
    }
    if (carry > 0) {           // 最高位还有进位
        c[++lc] = carry;
    }
    return lc;
}
```

我们来手工走一遍 `123 + 89`：
- i=1：a[1]=3, b[1]=9 → 3+9+0=12 → c[1]=2, carry=1
- i=2：a[2]=2, b[2]=8 → 2+8+1=11 → c[2]=1, carry=1
- i=3：a[3]=1, b[3]=0（89 没有百位）→ 1+0+1=2 → c[3]=2, carry=0
- carry=0 结束，结果 lc=3，倒序输出 c[3]c[2]c[1] = 212 ✓

输出时别忘了**倒序**——因为数组的 c[1] 是个位，c[lc] 是最高位：

```cpp
void print(int a[], int len) {
    for (int i = len; i >= 1; i--)
        cout << a[i];
}
```

高精度运算的本质：CPU 不认识"100 位整数"，但它认识 100 个 0~9 的数字。用数组模拟竖式，就没有数字大到算不了。

## 🖥️ 板书/屏幕要点
- **存储约定**：字符串输入 → 数组倒序存储，a[1]=个位, a[len]=最高位
- **加法核心**：`c[i] = ai + bi + carry; carry = c[i]/10; c[i] %= 10;`
- **最后进位**：`if (carry>0) c[++lc] = carry;`
- **倒序输出**：`for (i=len; i>=1; i--)`

## ✏️ 随堂练习
实现高精度加法。输入两个不超过 100 位的正整数，输出它们的和。测试 123456789 + 987654321。

## 📌 小结
高精度运算用数组模拟手工竖式。存储要倒序（个位在下标 1），加法逐位加、进位单独处理，输出倒序。这是所有高精度运算的基础。
