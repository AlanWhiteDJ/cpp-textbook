- 学会用计数数组维护滑动窗口内的统计信息

## 📝 讲稿正文

上节课学了滑动窗口的基本模式。这节课用它来解决两道竞赛题。

第一道：**A-B 数对**（NOIP 2007 普及组变体）。给定 n 个正整数和一个数 C，求满足 A - B = C 的数对 (A, B) 个数。

暴力是 O(n²)。怎么用双指针？排序后，对于每个 B，用两个指针 l 和 r 分别找到"第一个等于 B+C 的位置"和"最后一个等于 B+C 的位置"：

```cpp
long long countPairs(int a[], int n, int C) {
    sort(a, a + n);
    long long ans = 0;
    int l = 0, r = 0;
    for (int i = 0; i < n; i++) {  // 枚举 B = a[i]
        while (l < n && a[l] < a[i] + C) l++;
        while (r < n && a[r] <= a[i] + C) r++;
        ans += (r - l);
    }
    return ans;
}
```

这里 l 找到第一个 ≥ B+C 的位置，r 找到第一个 > B+C 的位置。区间 [l, r) 中的每个元素都等于 B+C，个数就是 r-l。

注意——l 和 r 不会重置！对于新的 B（更大的 a[i]），B+C 也更大，所以 l 和 r 只会往右走，不会左移。这也是双指针 O(n) 的原因。

第二道：**逛画展**（简化版）。n 幅画排成一排，每幅画有一个画家编号（1~m）。找最短的连续区间，包含所有 m 位画家的画至少一幅。

```cpp
int solve(int a[], int n, int m) {
    int cnt[2005] = {0};  // 窗口内每种画的数量
    int collected = 0;    // 已收集的画种数
    int ans = n + 1, l = 0;

    for (int r = 0; r < n; r++) {
        if (cnt[a[r]] == 0) collected++;
        cnt[a[r]]++;

        while (collected == m) {  // 集齐了，尝试收缩
            ans = min(ans, r - l + 1);
            cnt[a[l]]--;
            if (cnt[a[l]] == 0) collected--;
            l++;
        }
    }
    return ans;
}
```

这个模式的精髓：r 负责扩张（纳入新元素），当窗口满足条件后，l 持续收缩直到不再满足。每收缩一次更新一次最短长度。注意——收缩用的是 `while` 不是 `if`，因为可能要收缩多次。

两个指针都只向右移动，总复杂度 O(n)，比暴力 O(n²) 快得多。这个"扩张-收缩"模式能解决一大类子数组问题。

2. "逛画展中，为什么要用 while 收缩而不是 if？如果只收缩一次会错过什么？"
3. "如果逛画展的题目改成'最长区间使得区间内最多有两种画'，代码怎么改？"

## 🖥️ 板书/屏幕要点
- **A-B 数对**：排序+双指针，l 找 ≥B+C，r 找 >B+C，ans+=r-l
- **逛画展**：cnt[] 计数+collected 计数，扩张到集齐→while 收缩
- **通用模式**：r 扩张纳入 → while 满足条件时 l 收缩更新答案

## ✏️ 随堂练习
输入 n=6 幅画，画家编号为 `{1,2,2,1,3,1}`，m=3。用逛画展代码找出最短区间包含所有 3 位画家的画。答案应为 4（窗口 [2,2,1,3] 即位置 2~5）。

## 📌 小结
A-B 数对用排序配合双指针，l 和 r 都在有序数组中只右移。逛画展用计数数组维护窗口内的画种统计，满足条件时持续收缩。扩张-收缩是滑动窗口的万能模式。
