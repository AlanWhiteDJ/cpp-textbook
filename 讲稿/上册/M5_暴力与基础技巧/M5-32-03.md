## 📝 讲稿正文

上两节课我们学会了枚举排列。但还有一种常见需求——枚举一个集合的**所有子集**。比如有 4 个物品，每个可以"选"或"不选"，共有 2⁴ = 16 种方案。怎么用程序遍历这 16 种？

答案是——**二进制**！

想象 4 个开关排成一排，每个开关可以开（1）或关（0）。用 4 位二进制数表示所有开关状态——从 0000（全关）到 1111（全开），正好 16 种。每一位对应一个物品：第 i 位为 1 表示选第 i 个物品。

```cpp
int n = 4;
for (int mask = 0; mask < (1 << n); mask++) {
    cout << "子集 " << mask << ": ";
    for (int i = 0; i < n; i++) {
        if (mask & (1 << i)) {  // 第 i 位是 1 吗？
            cout << i << " ";
        }
    }
    cout << endl;
}
```

来理解这段代码的关键点。

第一，`1 << n` 是什么？1 左移 n 位。比如 n=4，`1 << 4` = 16（二进制 10000）。所以 `mask < (1 << n)` 遍历了 0 到 15。

第二，`mask & (1 << i)` 是判断第 i 位是否为 1。`1 << i` 产生一个只有第 i 位为 1 的数（比如 i=2 时是二进制 0100 = 4），然后 `mask & 那个数`——如果 mask 的第 i 位也是 1，结果非零；如果是 0，结果为零。

举个例子：mask=13（二进制 1101），i=2（二进制 0100）。13 & 4 = 1101 & 0100 = 0100 = 4 ≠ 0，说明第 2 位是 1。

这个技巧就叫**位运算**。对于子集枚举，你只需要掌握三个操作：

- 判断第 i 位：`mask & (1 << i)`
- 设置第 i 位为 1：`mask | (1 << i)`
- 设置第 i 位为 0：`mask & ~(1 << i)`

子集枚举的复杂度是 2ⁿ。n=20 时约 100 万次，还行；n=30 时约 10 亿次，超时。所以子集枚举一般用于 n ≤ 20 的场景。

注意：当 n > 30 时，`1 << n` 可能溢出 int（int 只有 32 位）。这时候要用 `1LL << n`（long long 的 1）。

## 🖥️ 板书/屏幕要点
- **子集枚举**：`for (mask = 0; mask < (1<<n); mask++)`
- **判断第 i 位**：`if (mask & (1 << i))`
- **设 1**：`mask | (1<<i)`；**设 0**：`mask & ~(1<<i)`
- **复杂度**：O(2ⁿ)，n≤20 可行；n>30 用 `1LL`

## ✏️ 随堂练习
输入 n 个整数，用二进制枚举找出所有和为 0 的子集。例如输入 `{-1, 1, 0}`，输出 `{0}` 和 `{-1, 1}`。

## 📌 小结
二进制枚举子集的核心是：用整数 mask 的二进制位表示每个元素的选/不选状态。`1<<n` 控制范围，`mask & (1<<i)` 判断选没选。这是竞赛中的必修技巧。
