## 📝 讲稿正文

一维有差分，二维也有——用来做子矩阵的批量修改。

二维差分的构建：`d[i][j] = a[i][j] - a[i-1][j] - a[i][j-1] + a[i-1][j-1]`（容斥原理决定符号）。

子矩阵 (x1,y1) → (x2,y2) 全部加 val，只需要四条指令：

```cpp
d[x1][y1]     += val;   // 右下区域全加
d[x2+1][y1]   -= val;   // 下面取消
d[x1][y2+1]   -= val;   // 右面取消
d[x2+1][y2+1] += val;   // 重叠角加回
```

四个角落，符号是 `+ - - +`。跟查询公式的 `+ - - +` 规律一致——很容易记。

操作完后，做二维前缀和还原：

```cpp
for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        a[i][j] = a[i-1][j] + a[i][j-1] - a[i-1][j-1] + d[i][j];
```

二维差分 O(1) 修改，O(nm) 还原。如果多次子矩阵修改，效率远超暴力。

现在换个话题——**最大子段和**的前缀和视角。之前你们学过 Kadane 算法求最大子段和，现在从前缀和的角度再看一次：

子段和 = `pre[r] - pre[l-1]`。要让这个值最大，对每个 r，需要找前面**最小的 `pre[l-1]`**。

```cpp
int maxSubarraySum(int a[], int n) {
    int pre = 0, minPre = 0, ans = a[1];
    for (int i = 1; i <= n; i++) {
        pre += a[i];
        ans = max(ans, pre - minPre);
        minPre = min(minPre, pre);
    }
    return ans;
}
```

O(n) 遍历，只用 O(1) 额外空间。这个写法和 Kadane 等价，但从前缀和角度理解更自然——"以当前位置结尾的最大子段和 = 当前前缀和 - 之前最小前缀和"。

前缀和模块总结：一维/二维前缀和、一维/二维差分。核心思维是"空间换时间"和"先算后用"。预处理好一张表，查询/修改就是 O(1) 的事。

## 🖥️ 板书/屏幕要点
- **二维差分四指令**：`d[x1][y1]+=v; d[x2+1][y1]-=v; d[x1][y2+1]-=v; d[x2+1][y2+1]+=v;`
- **符号规律**：`+ - - +`（与子矩阵查询一致）
- **最大子段和**：`ans = max(ans, pre - minPre)` → O(n)
- **模块总结**：空间换时间，先算后用

## ✏️ 随堂练习
在 3×3 零矩阵上用二维差分：子矩阵 (1,1)→(2,2) 加 1，(2,2)→(3,3) 加 2。输出最终矩阵。

## 📌 小结
二维差分四条指令完成子矩阵修改，符号规律 `+ - - +`。最大子段和 = 当前前缀和 - 之前最小前缀和。前缀和与差分的核心是"预处理一次，查询一步到位"。
