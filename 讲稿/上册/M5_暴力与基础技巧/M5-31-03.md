## 📝 讲稿正文

上节课我们写了石头剪刀布——双方手动输入。但如果想让计算机自己"扮演"双方呢？比如掷骰子——计算机自动生成 1~6 的随机点数。

在 C++ 里生成随机数，用 `rand()` 函数。但它每次运行产生同样的序列，所以要在程序开头"播种子"——用 `srand(time(0))`，让随机数跟当前时间挂钩。

来看掷骰子模拟：两人各掷 10 次，点数大的赢，平局各不得分。

```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int scoreA = 0, scoreB = 0;

    for (int i = 1; i <= 10; i++) {
        int a = rand() % 6 + 1;  // 1~6
        int b = rand() % 6 + 1;

        cout << "第 " << i << " 局 A:" << a << " B:" << b;
        if (a > b) { scoreA++; cout << " → A 胜" << endl; }
        else if (b > a) { scoreB++; cout << " → B 胜" << endl; }
        else { cout << " → 平局" << endl; }
    }
    cout << "A 胜 " << scoreA << " 局，B 胜 " << scoreB << " 局" << endl;
}
```

注意 `rand() % 6` 产生 0~5，加 1 变成 1~6。这是生成 `[1, n]` 随机整数的标准写法：`rand() % n + 1`。

现在我们加一条规则——**掷出 6 点可以再掷一次**！这是模拟题里最常见的"加规则"场景。额外掷出的点数加到本局总分上。

怎么实现？（让学生想）在掷出 6 后加一个 `while` 循环：

```cpp
int a = rand() % 6 + 1;
while (a == 6) {
    cout << "A 掷出 6，加掷一次！";
    int bonus = rand() % 6 + 1;
    cout << " 额外点数：" << bonus << endl;
    a += bonus;
    if (bonus != 6) break;  // 没再掷出 6，停止
}
```

看到了吗？模拟题的魅力就在这里——加规则就是加代码。题干多一行规则，你在对应的位置多写几行 `if` 或 `while`。

但注意——规则越来越多时，代码也会越来越长。这时候就要保持清醒：**一条规则对应一段代码，写完一条检查一条**。不要一口气全写完再调试，那样找 bug 会非常痛苦。

## 🖥️ 板书/屏幕要点
- **随机数**：`rand() % 6 + 1` → 生成 1~6
- **设置种子**：`srand(time(0))` → 每次运行不同结果
- **奖励回合**：`while (a==6)` 连续加掷
- **加规则 = 加代码**：一条规则对应一段代码

## ✏️ 随堂练习
给掷骰子加入"两人同时掷出 6 则本局作废重来"的规则。写代码实现。

## 📌 小结
`rand() % n + 1` 生成 `[1,n]` 随机数，用 `srand(time(0))` 设置种子。模拟题扩展规则就是在对应位置加代码。写复杂模拟时，一条规则一条规则地写、一条一条地测。
