- 理解快慢指针的思想：不同速度同向移动

## 📝 讲稿正文

上两节课我们学了对撞指针——一左一右往中间走。今天学另一种形态：**快慢指针**——两个指针同向移动，但速度不同。

最直观的应用是**找中位数**。慢指针每次走一步，快指针每次走两步。当快指针到末尾时，慢指针刚好在中间。

```cpp
int findMiddle(int a[], int n) {
    int slow = 0, fast = 0;
    while (fast + 1 < n && fast + 2 < n) {
        slow++;
        fast += 2;
    }
    return a[slow];
}
```

想象你在操场上跑步，跑得比你朋友快一倍。当你跑完一圈时，他刚好跑了半圈——快慢指针就是这个直觉。

快慢指针在链表中最常见（以后会学到判环），今天重点放在第三种形态——**滑动窗口**。

滑动窗口是什么？维护一个可变区间 `[l, r]`，根据条件动态调整区间大小。它像一条**毛毛虫**：头向前探（r 扩张），身体跟上（l 收缩），从不后退。

来看一道入门题——**最长无重复子串**。在一个字符串中找最长的不含重复字符的连续子串。

思路：维护窗口 `[l, r]`，用一个 `bool` 数组或 `set` 记录窗口内有哪些字符。右指针 r 不断右移，如果新字符已经在窗口中出现过，左指针 l 就右移直到移除那个重复字符。

```cpp
int longestUnique(string s) {
    bool inWindow[256] = {false};
    int l = 0, ans = 0;
    for (int r = 0; r < s.size(); r++) {
        while (inWindow[s[r]]) {  // 新字符重复了，收缩左端
            inWindow[s[l]] = false;
            l++;
        }
        inWindow[s[r]] = true;
        ans = max(ans, r - l + 1);
    }
    return ans;
}
```

关键点在于 `while` 而不是 `if`——因为可能需要收缩多次才能把重复字符移除。

滑动窗口和暴力的区别：暴力是"固定左端点，右端点走到头"，每个左端点都要把右端点重置。滑动窗口中，**两个指针都只向右走，从不回头**。每个元素最多被 l 经过一次、被 r 经过一次，总复杂度 O(n)。

三种双指针形态对比：对撞指针是"两端往中间"、快慢指针是"同向不同速"、滑动窗口是"同向维护区间"。它们共同的秘密是——**每个元素只被访问常数次，总复杂度 O(n)**。

## 🖥️ 板书/屏幕要点
- **快慢指针**：slow 走 1 步，fast 走 2 步
- **滑动窗口**：`[l, r]` 可变，r 扩张、l 收缩
- **最长无重复子串**：`while(inWindow[s[r]]) { l++ }`
- **复杂度**：O(n)，双指针都只向右、从不回头

## ✏️ 随堂练习
用滑动窗口找字符串 "abcabcbb" 的最长无重复子串。答案应是 3（"abc"）。

## 📌 小结
快慢指针找中点——快速的走两步、慢速的走一步。滑动窗口维护可变区间 [l, r]，r 扩张、l 收缩，两个指针都只右移，O(n) 解决子数组/子串问题。
