## 📝 讲稿正文

有一种模拟题特别考验耐心——**格式型模拟**。它不要求复杂的计算，而是要求你按照一堆格式规则输出字符串。少一个加号、多一个空格——全错。

来看多项式输出（NOIP 2009 普及组）。给定一元 n 次多项式的系数，按规范格式输出。规则包括：
- 系数为 0 的项不输出
- 最高次项系数为 1 时省略系数（只输出 "x"）
- 常数项只输出数字
- 负号直接输出，第一项正号不输出
- 指数 > 1 时输出 "^"

看似简单，但特殊情况一堆！怎么保证不遗漏？列一张"特殊情况检查表"：

| 情况 | 处理 |
|------|------|
| 系数 = 0 | 跳过 |
| 系数 = 1，i>0 | 不输出 1，只输出 x |
| 系数 = -1，i>0 | 输出 -x |
| 常数项 (i=0) | 只输出数字 |
| 第一项正系数 | 不输出 + |
| 非第一项 | 正输出+、负输出- |
| 指数 = 1 | 只输出 x，不输出 ^1 |
| 指数 > 1 | 输出 x^i |

对照这份清单逐条实现：

```cpp
void printPoly(int n, int a[]) {
    bool first = true;
    for (int i = n; i >= 0; i--) {
        if (a[i] == 0) continue;

        if (!first) cout << (a[i] > 0 ? "+" : "-");
        else {
            if (a[i] < 0) cout << "-";
            first = false;
        }

        int absCoef = abs(a[i]);
        if (i > 0 && absCoef == 1) cout << "x";
        else {
            cout << absCoef;
            if (i > 0) cout << "x";
        }
        if (i > 1) cout << "^" << i;
    }
}
```

你们看，代码不复杂——就是一条规则对应一段 if。但如果不提前列清单，写着写着就会忘记"系数为 -1 怎么处理""第一项要不要输出 +"。

做格式型模拟题的秘诀就是：**别靠脑子硬记，老老实实列清单，写完一条打一个钩。**

我见过很多同学在这个题上栽跟头——不是因为不会写，而是因为少了某个特殊情况。这道题在 NOIP 赛场上被视为"送分题"，但每年都有人拿不到满分。为什么？因为不够细心。

## 🖥️ 板书/屏幕要点
- **格式型模拟**：规则琐碎、考细心、列清单
- **特殊情况检查表**：系数0跳过 / 系数±1省略 / 首项符号 / 指数处理
- **实现策略**：列清单→逐条翻译→逐条打钩

## ✏️ 随堂练习
输入 n=3，系数为 `{0, -5, 3, -1}`（即 -x³+3x²-5x），输出格式化多项式字符串。

## 📌 小结
格式型模拟考的不是算法，是细心。用"特殊情况检查表"把所有边界列出来，一条一条对照实现。不要靠脑子硬记——大脑的内存是有限的，纸的容量是无限的。
