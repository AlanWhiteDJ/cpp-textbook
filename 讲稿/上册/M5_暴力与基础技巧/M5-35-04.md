## 📝 讲稿正文

一维前缀和很好用，但如果数据是二维矩阵呢？比如"多次查询某个子矩阵的元素和"。这时候需要——**二维前缀和**。

定义：`pre[i][j]` = 以 (1,1) 为左上角、(i,j) 为右下角的子矩阵元素之和。

想象你有一块矩形农田，`pre[i][j]` 是左上角 (1,1) 到 (i,j) 这块地的总产量。

构建公式用到**容斥原理**——一个区域被重复计算了要减掉：

```cpp
pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + a[i][j];
```

为什么？画个图就明白了：
- `pre[i-1][j]` = 上半部分（不含第 i 行）
- `pre[i][j-1]` = 左半部分（不含第 j 列）
- 但 `pre[i-1][j-1]` 这个左上角**被加了两次**！所以要减掉一次
- 最后加上当前格 `a[i][j]`

查询子矩阵 (x1,y1) → (x2,y2) 的和，同样是容斥原理：

```cpp
int query(int x1, int y1, int x2, int y2) {
    return pre[x2][y2]           // 大面积
         - pre[x1-1][y2]         // 减上面条
         - pre[x2][y1-1]         // 减左面条
         + pre[x1-1][y1-1];      // 加回重叠角
}
```

拿笔画一画：整个大矩形，剪掉上面一条、剪掉左边一条——但左上角那个小矩形被剪了两次，所以加回来一次。

完整构建代码：

```cpp
int pre[1005][1005];

for (int i = 1; i <= n; i++)
    for (int j = 1; j <= m; j++)
        pre[i][j] = pre[i-1][j] + pre[i][j-1]
                    - pre[i-1][j-1] + a[i][j];
```

注意矩阵下标也是从 1 开始——跟一维一样方便。

来看经典题——**领地选择**。在一个 n×m 矩阵中，找一个 c×c 的子正方形，使得元素和最大。

枚举所有可能的 c×c 正方形左上角，用二维前缀和 O(1) 查询每个正方形的和：

```cpp
for (int i = 1; i <= n - c + 1; i++)
    for (int j = 1; j <= m - c + 1; j++) {
        int sum = query(i, j, i + c - 1, j + c - 1);
        ans = max(ans, sum);
    }
```

复杂度 O(nm)，暴力 O(nmc²)。c=100 时差距就是 10000 倍。

**记忆技巧**：容斥原理的符号规律——奇数个减号？不是。记"加主体、减两条、加重叠角"，把这十二个字背下来就不会写错。

## 🖥️ 板书/屏幕要点
- **构建**：`pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + a[i][j]`
- **查询**：`pre[x2][y2] - pre[x1-1][y2] - pre[x2][y1-1] + pre[x1-1][y1-1]`
- **口诀**：加主体、减两条、加重叠角
- **领地选择**：枚举 c×c 正方形，O(nm) 完成

## ✏️ 随堂练习
3×3 矩阵 `{1,2,3; 4,5,6; 7,8,9}`，构建二维前缀和，查询子矩阵 (2,1) 到 (3,2)（即 4+5+7+8=24）。

## 📌 小结
二维前缀和基于容斥原理：构建 = 加左+加上-减重叠+当前；查询 = 大面积-上面条-左面条+重叠角。12 字口诀：加主体、减两条、加重叠角。
