# 第9章 第4课时：条件组合器——逻辑运算符
**时长**：15-20分钟 | **对应教材**：第9章 · 9.6节、9.7节

## 🎯 本课时目标
- 掌握三种逻辑运算符：`&&`（与）、`||`（或）、`!`（非）
- 理解短路求值（"见好就收"）机制及其安全优势
- 能用逻辑运算符把复杂条件浓缩成一行

## 📝 讲稿正文

同学们好！上节课我们学了嵌套 if 来处理复杂条件——比如判断闰年，if 里面套 if，套了三层，读得人头大。

今天我要教你们三个"条件组合器"——学会了它们，三层的嵌套能变成一行。

先看一个场景：学校要评"全额奖学金"，条件是"**分数 ≥ 90，而且体育也 ≥ 90**"。这个"而且"该怎么说？

再看另一个场景：商场搞活动，"**满 200 元或者会员**"就可以打折。这个"或者"又该怎么说？

C++ 里用三个**逻辑运算符**来表达这种组合条件：

| 运算符 | 含义 | 例子 | 结果为 true 的条件 |
|--------|------|------|---------------------|
| `&&` | 与（且） | `a > 5 && b < 10` | 两边都成立才行 |
| `\|\|` | 或 | `a > 5 \|\| b < 10` | 至少一边成立就行 |
| `!` | 非（取反） | `!(a > 5)` | a > 5 不成立 |

来，把刚才的两个例子写成 C++：

```cpp
// 奖学金：分数 ≥ 90 且 体育 ≥ 90
if (score >= 90 && pe >= 90) {
    cout << "全额奖学金！" << endl;
}

// 打折：满 200 或 会员
if (total >= 200 || isMember) {
    cout << "享受折扣！" << endl;
}
```

注意 `&&` 是键盘上数字 7 上面的符号，连着打两个。`||` 是回车键上面那个竖线，也是打两个。

现在我们来见证逻辑运算符的威力。上节课我们用手写了三层嵌套来判断闰年，惨不忍睹。现在用逻辑运算符，一行搞定：

```cpp
if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) {
    cout << "闰年" << endl;
} else {
    cout << "平年" << endl;
}
```

来，我们拆开读一下这个条件：**"year 能被 4 整除，并且不能被 100 整除"——这是一个组合；"或者 year 能被 400 整除"——这是另一个分支。整个条件的意思是：满足前者或者满足后者，就是闰年。**

（停顿，让学生消化）

是不是比三层嵌套清楚多了？一行顶六行，而且读起来像说人话。

好，接下来讲一个 C++ 特别聪明（也特别容易被坑）的特性——**短路求值**，我给它起了个更形象的名字叫"**见好就收**"。

规则很简单：
- `A && B`：如果 A 已经是 false，B 根本不会被计算。因为 && 要求两边都真，有一边假结果就是假，算 B 纯属浪费。
- `A || B`：如果 A 已经是 true，B 也不会被计算。因为 || 只要一边真结果就是真，算 B 也是浪费。

看这个例子：

```cpp
int x = 0;
if (x != 0 && 10 / x > 5) {
    cout << "成立" << endl;
} else {
    cout << "条件不成立" << endl;
}
```

注意！x 是 0！如果 C++ 傻乎乎地先去算 `10 / x`，那就是 `10 / 0`——除零错误，程序直接崩溃！

但实际上，这个程序跑得稳稳的，输出"条件不成立"。为什么？因为 && 先看左边 `x != 0`，发现是 false——"见好就收"，右边 `10 / x > 5` 根本就没执行！C++ 用它的小聪明在无意中保护了你的程序。

但是，这个特性也有坑人的时候。比如你在右边放了一个 `i++`：

```cpp
if (false && i++) { ... }
// i++ 不会执行！i 的值没变！
```

所以记住：**不要在 && 或 || 的右边放有副作用的操作**——比如 `++`、`--`、函数调用等。因为你不能保证它一定会被执行。

最后看第三个逻辑运算符 `!`——取反。它最简单，就是把真的变假，假的变真：

```cpp
if (!(score >= 60)) {   // 等价于 score < 60
    cout << "不及格" << endl;
}
```

`!` 有时候能让条件写得更自然，但不要用太多层——`!!flag` 这种写法会让人困惑。

好，动手环节：写一个程序判断输入的年份是不是闰年，用逻辑运算符一行搞定条件。再写一个程序，输入一个整数，判断它是不是在 1 到 100 之间（含两端），用 `&&` 实现。

## 🖥️ 板书/屏幕要点
- `&&` 与：两边都真才真 → "而且"
- `||` 或：一边真就真 → "或者"
- `!` 非：真假颠倒
- 短路求值：`&&` 见 false 就跑，`||` 见 true 就跑
- 闰年一行版：`(year%4==0 && year%100!=0) || year%400==0`

## ✏️ 随堂练习
写一个程序：输入三个整数 a, b, c，判断它们能否构成三角形。条件：任意两边之和大于第三边，即 `a+b>c && a+c>b && b+c>a`。用逻辑运算符一步判断。

## 📌 小结
- `&&`（与）、`||`（或）、`!`（非）能把复杂条件浓缩成一行
- 短路求值在无意中保护程序，但也可能跳过你想要的副作用
- 用逻辑运算符替代深层嵌套，代码更清晰
