# 第13章 第3课时：菱形——正三角加倒三角
**时长**：15-20分钟 | **对应教材**：第13章 · 13.4节

## 🎯 本课时目标
- 掌握等腰三角形的打印方法（空格 + 星号）
- 理解菱形 = 正三角 + 倒三角的组合思路
- 能独立写出菱形打印程序

## 📝 讲稿正文

同学们好！上节课我们画了矩形和三角形——都是用星号紧挨着左边开始画的。但你有没有注意过，真正的"三角形"图案应该是居中的、像金字塔一样的等腰三角形？

（在黑板上画一个等腰三角形，星号居中）

来，我们看一个 4 行的等腰三角形：

```
   *        第1行：3空格 + 1星号
  ***       第2行：2空格 + 3星号
 *****      第3行：1空格 + 5星号
*******     第4行：0空格 + 7星号
```

观察规律：第 i 行有几个空格？`n - i` 个（n=4）。第 i 行有几个星号？`2*i - 1` 个。

验证：i=1 时，空格 4-1=3，星号 2×1-1=1——对了！

所以等腰三角形的代码是：

```cpp
int n = 4;
for (int i = 1; i <= n; i++) {
    // 先打空格
    for (int j = 1; j <= n - i; j++) {
        cout << " ";
    }
    // 再打星号
    for (int j = 1; j <= 2 * i - 1; j++) {
        cout << "*";
    }
    cout << endl;
}
```

注意这里一层循环里面有两个内层循环——一个管空格，一个管星号。它们**顺序执行**：先打完所有空格，再打所有星号，然后换行。不是嵌套的关系——它们都平级地在外层循环里。

（在黑板上画出结构：外层 for 里面两个并列的 for）

好，有了等腰三角形，**菱形**就简单了——菱形 = 一个正等腰三角 + 一个倒等腰三角。

上半部分：4 行的正等腰三角（如上）。
下半部分：3 行的倒等腰三角——空格递增，星号递减。

```cpp
int n = 4;  // 菱形高度的一半

// 上半部分：正等腰三角
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n - i; j++) cout << " ";
    for (int j = 1; j <= 2 * i - 1; j++) cout << "*";
    cout << endl;
}

// 下半部分：倒等腰三角
for (int i = n - 1; i >= 1; i--) {
    for (int j = 1; j <= n - i; j++) cout << " ";
    for (int j = 1; j <= 2 * i - 1; j++) cout << "*";
    cout << endl;
}
```

输出：

```
   *
  ***
 *****
*******
 *****
  ***
   *
```

（指着输出结果，逐行讲解）

注意下半部分的 `i` 是从 `n-1` 往下减的：i=3, 2, 1。这样第三行时 i=3，空格=4-3=1，星号=5——跟上半部分对称。

这个"正三角 + 倒三角 = 菱形"的思路非常经典。它教会我们：**复杂图形可以拆成简单图形的组合。** 不要一上来就想着"怎么写一个菱形"，而是想"菱形可以拆成哪几部分"，然后分别实现。

（在黑板上写：分解 → 分别实现 → 组合）

这种"分而治之"的思维方式，不只是用来画图形——以后解决复杂算法问题，也是一样的套路。

现在动手练习：
1. 先写出 n=5 的等腰三角形
2. 再写出 n=5 的菱形（上半 5 行 + 下半 4 行）

（给学生练习时间，先做三角形，再做菱形）

小结：等腰三角 = 空格循环 + 星号循环。菱形 = 正等腰三角 + 倒等腰三角。遇到复杂图形，先拆再合。

## 🖥️ 板书/屏幕要点
- 等腰三角：空格 = n-i，星号 = 2i-1
- 一行内的两个内层循环是**并列**的，先空格后星号
- 菱形 = 正等腰三角（i: 1→n）+ 倒等腰三角（i: n-1→1）
- 拆解思维：复杂图形 → 简单图形组合

## ✏️ 随堂练习
1. 输入 n=5，打印等腰三角形
2. 打印 n=5 的完整菱形

## 📌 小结
- 等腰三角：两个并列内层循环（空格 + 星号）
- 菱形 = 分段实现：上半正三角 + 下半倒三角
- 分解思维：复杂问题拆成简单问题
- 注意下半部分的循环变量要递减
