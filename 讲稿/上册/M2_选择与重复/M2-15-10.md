### 第一层：选择结构（第 9-10 章）

**if 家族**：
- `if (条件) { ... }` —— 单分支：成立就做
- `if-else` —— 双分支：二选一，必走一条
- `else if` —— 多分支：从上往下，命中即停（顺序很重要！）
- 嵌套 if —— if 里面套 if

**条件怎么写**：
- 关系运算符：`==` `!=` `<` `>` `<=` `>=`
- 逻辑运算符：`&&`（且）、`||`（或）、`!`（非）
- **铁律**：`==` 是比较，`=` 是赋值

**switch-case**：
- 精确匹配多值 → `switch (整型/字符) { case 值: ... break; default: ... }`
- 穿透：case 只是入口，不加 break 一路到底
- 巧用穿透合并相同处理

**三目运算符**：`条件 ? 成立值 : 不成立值` —— 简单二选一用

### 第二层：循环结构（第 11-12 章）

**三种循环**：
- `for`：知道次数。初始化、条件、迭代三要素都在括号里
- `while`：知道条件、不知道次数。初始化和迭代替你自觉
- `do-while`：至少做一次。先执行再判断

**四种经典模式**：
| 模式 | 初始值 | 操作 | 典型场景 |
|------|--------|------|----------|
| 累加 | 0 | `sum += x` | 求和 |
| 累乘 | 1 | `fact *= x` | 阶乘 |
| 计数 | 0 | `cnt++` | 统计个数 |
| 找最值 | 第一个数 | `if (x > max)` | 最大/最小 |

**死循环与条件设计**：三检查点——初始化了吗？更新了吗？能退出吗？

### 第三层：循环控制（第 13-14 章）

**嵌套循环**：外层走一步、内层跑一轮。外层慢（行），内层快（列）。

**图案打印万能框架**：
```
外层 (行) → 可能的空格 → 内层 (列) → 决定打什么 → 换行
```

**break 与 continue**：
- break："脚崴了，立刻停" → 跳出当前循环（只跳一层！）
- continue："踩线了，这圈不算" → 跳过本轮，下轮继续
- 跳出多层：`bool found` 标记变量——内层插旗，外层检查

### 第四层：综合实战（第 15 章）

**三步选结构思维**：分岔吗？重复吗？嵌套吗？

**六大经典题型**：猜数字、菜单计算器、图案打印、水仙花数、质数判断、完数判断
