## 📝 讲稿正文

同学们好！欢迎来到第11章——for 循环。从这一章开始，我们的程序不再做"一次性"的事情，而是能**反复、重复、不停歇地**干活。

（在黑板上写一个大大的 "for"）

先问大家一个问题：体育课上，老师说"绕操场跑 3 圈"。你怎么跑的？

你不会跑完一圈停下来问老师："还要跑吗？"——你自己心里有数。你的大脑自动在做三件事：
1. 记住这是第几圈（从第 1 圈开始）
2. 每跑完一圈，检查"跑满 3 圈了吗？"
3. 没跑满，继续跑；跑满了，停下来

这三件事，正好对应 for 循环的**三个要素**：初始化、条件、迭代。

来看 for 循环的语法：

```cpp
for (初始化; 条件; 迭代) {
    循环体：重复执行的代码
}
```

三个部分，用分号隔开，一个都不能少：

| 要素 | 含义 | 跑步类比 |
|------|------|----------|
| 初始化 | 从哪里开始 | "第 1 圈" |
| 条件 | 什么时候继续 | "还没跑完 3 圈" |
| 迭代 | 每轮结束做什么 | "圈数 +1" |

好，写一个最简单的例子——输出 1 到 5：

```cpp
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// 输出：1 2 3 4 5
```

我们一步步在脑子里跑一遍：

1. `int i = 1`：创建一个变量 i，初始值为 1（初始化，只做一次）
2. `i <= 5`：1 ≤ 5？成立！进入循环体
3. `cout << i`：输出 1
4. `i++`：i 变成 2（迭代）
5. `i <= 5`：2 ≤ 5？成立！输出 2
6. `i++`：i 变成 3
7. ……重复……
8. `i++`：i 变成 6
9. `i <= 5`：6 ≤ 5？不成立！退出循环

（在黑板上画一个表格，每轮写上 i 的值、条件判断结果、输出内容）

注意一个关键点：**判断条件是在每一轮循环的开头执行的**。这意味着如果一开始条件就不成立，循环体一次都不执行。比如 `for (int i = 10; i <= 5; i++)`——初始 i=10，已经大于 5 了，一次都不跑。

好，现在看几个变体，体会循环变量的灵活用法：

```cpp
// 从 100 倒数到 1
for (int i = 100; i >= 1; i--) {
    cout << i << " ";
}

// 输出 2 到 100 的所有偶数
for (int i = 2; i <= 100; i += 2) {
    cout << i << " ";
}

// 输出前 10 个奇数：1, 3, 5, ..., 19
for (int i = 0; i < 10; i++) {
    cout << 2 * i + 1 << " ";
}
```

注意 `i += 2` 等价于 `i = i + 2`——每轮跳两步。`2 * i + 1` 是等差数列的通项公式：第 i 个奇数是 2i+1。

循环变量 `i` 是 for 循环的核心。它不仅是计数器，还可以参与各种计算——你想输出什么数列，就在循环体里用 `i` 构造什么公式。

关于变量命名：`i` 是全世界程序员最常用的循环变量名。如果有嵌套循环，第二个叫 `j`，第三个叫 `k`——这是从数学的向量记号（xi, xj, xk）沿用下来的习惯。虽然变量名本身无所谓，但用 `i` 能让你的代码更容易被其他程序员看懂。

最后讲一个超级大坑——**分号陷阱**。请看这段代码：

```cpp
for (int i = 1; i <= 5; i++);   // ← 注意末尾多了个分号！
{
    cout << i << " ";
}
```

运行结果是什么？只输出一个孤零零的 `6`——不是 `1 2 3 4 5`！

为什么？`for (...);` 那个分号让循环体变成了**空语句**。循环自己悄悄跑了 5 轮（i 从 1 升到 6），然后才执行下面花括号里的代码——但那时 i 已经是 6 了，而且花括号是孤立的，不属于循环。

记住：**for 的圆括号后面直接开花括号，中间不能有分号。**

## 🖥️ 板书/屏幕要点
- `for (初始化; 条件; 迭代) { 循环体 }`
- 执行顺序：初始化（1次）→ 判断 → 循环体 → 迭代 → 判断 → …
- 判断在每轮**开头**执行
- 循环变量 `i` 可以在循环体中做各种计算
- ⚠️ for 后面别乱加分号

## ✏️ 随堂练习
写一个 for 循环，输出 1 到 100 中所有 7 的倍数（7, 14, 21, …）。提示：可以 `i += 7`。

## 📌 小结
- for 循环 = 三个要素：初始化、条件、迭代
- 执行顺序：初始化 → 判断 → 循环体 → 迭代 → 判断……
- 循环变量 i 是核心，可以参与构造任意等差数列
- 分号大坑：`for (...);` 让循环体变空语句
