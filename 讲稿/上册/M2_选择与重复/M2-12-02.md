## 📝 讲稿正文

同学们好！上节课我们学了 while 的基本用法——密码验证。今天我们来学 while 循环的两个经典模式，这两个模式在信息学竞赛题里随处可见。

第一个模式叫"**哨兵输入**"。场景是：用户连续输入一串数字，以 -1 作为结束标志。你要统计这些数字的和。

（在黑板上画一个传送带，数字一个个过来，-1 上面画个红旗）

```cpp
int sum = 0, num;

cout << "请输入一串整数（-1 结束）：" << endl;
cin >> num;                    // 先读第一个数

while (num != -1) {            // 不是哨兵就继续
    sum += num;                // 累加
    cin >> num;                // 读下一个
}

cout << "总和为：" << sum << endl;
```

运行一下：输入 `5 10 3 -1`，程序输出"总和为：18"。

这个模式的关键结构是：

```
读第一个 → while (不是哨兵) { 处理 → 读下一个 }
```

注意，读取出现了两次——一次在循环外，一次在循环末尾。循环外的那次叫"**循环前准备**"——你得先拿到第一个数，才有东西判断。循环末尾的那次叫"**循环内更新**"——每处理完一个数，立刻读下一个，为下一轮判断做准备。

这个结构可以用一个更紧凑的写法——把 `cin >> num` 直接放在 while 的条件里：

```cpp
int sum = 0, num;
while (cin >> num && num != -1) {
    sum += num;
}
```

`cin >> num` 本身有一个"返回值"——如果读取成功，它的值是真；如果读取失败（比如输入了字母或遇到文件结尾），它的值是假。所以 `cin >> num && num != -1` 的意思是"如果能读到数，而且不是 -1，就继续"。

第二种写法更紧凑，但需要理解 `cin >> num` 能当条件用。初学者可以先用第一种（循环外读一遍），熟练了再用第二种。

第二个经典模式——**逐位处理**。场景：输入一个正整数，把它逐位拆开并反转输出。比如输入 1234，输出 4321。

思路：每次取个位，然后丢掉个位，重复直到数字变成 0。

```cpp
int n;
cin >> n;

while (n > 0) {
    int digit = n % 10;        // 取出个位
    cout << digit;             // 输出个位
    n /= 10;                   // 丢掉个位（整数除法）
}
// 输入 1234，输出 4321
```

来，模拟一遍：

```
初始：n = 1234

第1轮：digit = 1234 % 10 = 4  → 输出 4  → n = 1234 / 10 = 123
第2轮：digit = 123 % 10 = 3   → 输出 3  → n = 123 / 10 = 12
第3轮：digit = 12 % 10 = 2    → 输出 2  → n = 12 / 10 = 1
第4轮：digit = 1 % 10 = 1     → 输出 1  → n = 1 / 10 = 0
第5轮：n > 0？ → 0 > 0 不成立 → 退出循环
```

`n % 10` 取个位，`n /= 10` 丢个位——这两个操作是处理整数各位的万能钥匙。while 循环一直跑到 n 变成 0 为止。

这个模式可以变出很多花样：
- 统计位数：设一个计数器，每轮 `cnt++`
- 判断某位是否为特定值：在循环里加 if
- 数字反转后存储：`rev = rev * 10 + digit`

（板书逐位处理的模板：`while (n > 0) { n%10; n/=10; }`）

好，现在动手练习。写一个程序：输入一个正整数，输出它的位数、以及各位数字之和。例如输入 12345，输出"5 位数，各位和 15"。

（给学生练习时间）

小结：哨兵模式用特定值（如 -1）标记输入结束，核心是"循环前读一次，循环内读下一次"；逐位处理用 `n%10` 和 `n/=10` 逐个拆解数字，while 跑到 n 变 0。

## 🖥️ 板书/屏幕要点
- 哨兵模式：`cin >> num; while (num != -1) { ...; cin >> num; }`
- 紧凑版：`while (cin >> num && num != -1) { ... }`
- 逐位处理：`while (n > 0) { digit = n % 10; n /= 10; }`
- 模式本质：循环前准备 → 循环判断 → 循环内更新

## ✏️ 随堂练习
输入一个正整数，输出：
1. 它是几位数
2. 各位数字之和
例如输入 12345，输出"5 位数，各位数字之和 = 15"。

## 📌 小结
- 哨兵输入：用特定值结束输入，循环外读第一个，循环尾读下一个
- 逐位处理：`n%10` 取个位，`n/=10` 丢个位，while(n>0)
- 两个模式都体现了 while 的优势——不知道多少次，靠条件驱动
