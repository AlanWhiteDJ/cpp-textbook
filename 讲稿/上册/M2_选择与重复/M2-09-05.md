# 第9章 第5课时：流程图与本章总复习
**时长**：15-20分钟 | **对应教材**：第9章 · 9.8节 + 全章复习

## 🎯 本课时目标
- 学会画流程图，用图形描述程序的判断过程
- 梳理 if / if-else / else if 三种结构的适用场景
- 通过综合练习巩固第9章全部知识

## 📝 讲稿正文

同学们好！今天是第9章的最后一节课。前面四节课我们学了分支结构的各种武器——if、if-else、else if、关系运算符、逻辑运算符、短路求值。今天我们先学一个"思维工具"——流程图，然后把整章串起来复习一遍。

（拿起粉笔）

当程序逻辑变复杂的时候，你脑子里光靠想是不够的。大脑同时只能记 7 样左右的东西——但一段复杂的 if-else 链条可能涉及十几个判断。这时候，你需要画图。

流程图就是程序员的"草图"。它用几种简单的图形来描述程序的运行过程：

- **椭圆形**：开始和结束
- **长方形**：做一件事（执行一条语句）
- **菱形**：判断一个问题（两个出口：是/否）
- **箭头**：执行的顺序和方向

来，我们一起画成绩评级程序的流程图。先在黑板上画一个椭圆，写"开始"；画一个长方形，写"输入分数"；然后画一个菱形，写"score ≥ 90？"

（边画边讲解）

菱形有两个出口。上面标"是"，右边标"否"。"是"的箭头指向一个长方形，写"输出 A"。"否"的箭头指向下一个菱形："score ≥ 80？"……以此类推。最后所有分支汇合到一个椭圆，写"结束"。

记住：**菱形只用来问问题**——它是整个流程中唯一有"分叉"的图形。而且一个菱形只有两个出口，对应"是"和"否"。如果你需要更多的出口，就要用多个菱形串起来——这就是 else if 链条的图形表达。

（给学生在纸上两分钟临摹一遍）

好，画完了流程图，我们来做一个全章的知识树回顾。请拿出一张纸，跟我一起写。

第9章的核心就是两个字——**判断**。围绕判断，我们学了三层武器：

第一层：**单条件判断**。最简单的 `if (条件) { ... }` ——条件成立就做，不成立就跳过。这是单分支。

第二层：**二选一判断**。`if-else` ——两条路必走一条且只走一条。这里用到了六种关系运算符：`==` `!=` `<` `>` `<=` `>=`。

第三层：**多分支判断**。`else if` ——多个条件从上到下按顺序检查，第一个成立的就执行，然后跳出。**顺序至关重要**——必须从大到小或从小到大排列，宽松条件不能放前面。

在这三层之上，还有两个"增效工具"：

一个是**逻辑运算符**：`&&`（与）、`||`（或）、`!`（非）。它们能把多个条件组合起来，把嵌套的 if 浓缩成一行。

另一个是**短路求值**：`&&` 见 false 就跑，`||` 见 true 就跑——这个特性在无意中保护程序不出错，但也要小心它跳过你想要的副作用。

最后是**流程图**——先画图再写代码，思路清晰一半。

（在黑板上写出一个知识树）

好，现在请大家完成一个综合练习。这道题需要用到本章几乎所有知识点——

**题目**：输入一个年份和一个月份，输出这个月有多少天。需要考虑闰年的情况（2 月闰年 29 天，平年 28 天）。

提示步骤：
1. 先用 if-else 判断大小月（1/3/5/7/8/10/12 → 31 天，4/6/9/11 → 30 天）
2. 如果是 2 月，用逻辑运算符判断闰年决定天数
3. 如果月份不在 1~12 之间，输出"无效月份"

先画流程图，再翻译成代码。

（给学生 5-8 分钟练习，然后请一位同学到黑板上写答案）

好，我们来看这位同学的代码……（点评并纠正）

第9章到这里就结束了。我强调最后一遍：**if 里的 `==` 是两个等号，`=` 是一个等号——这是本章最大最大的坑。** 你们只要记住这一点，就已经避开了初学者最常犯的错误。

下堂课我们进入第10章——当选项太多的时候，else if 会写得像一堵墙。那时候，switch-case 和三目运算符就该登场了。

## 🖥️ 板书/屏幕要点
- 流程图四要素：椭圆（开始/结束）、长方形（执行）、菱形（判断）、箭头（走向）
- 知识树：if → if-else → else if → 逻辑运算符 → 短路求值
- 常见坑汇总：`=` vs `==`、else if 顺序、忘记花括号

## ✏️ 随堂练习
综合题：输入一个年份和一个月份，输出这个月有多少天。先画流程图再写代码。考虑闰年。

## 📌 小结
- 流程图用图形说话，菱形判断、箭头指向
- 先画图后写代码是解决复杂逻辑的好习惯
- 第9章全部武器：if / if-else / else if / 关系运算符 / 逻辑运算符 / 短路求值
- 铁律一则：if 里判断相等用 `==`，不是 `=`
