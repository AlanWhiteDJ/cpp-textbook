## 📝 讲稿正文

同学们好！今天我们来解决一个经典的数学编程题——水仙花数。

先看定义：一个三位数，如果它等于各位数字的立方和，就叫"水仙花数"。比如 153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153。验证一下——确实相等！所以 153 是水仙花数。

走三步选结构：

① **需要分岔吗？** 是！对每个三位数，要判断它是否满足"各位立方和等于自身"→ if 判断

② **需要重复吗？** 是！要检查所有三位数（100~999）→ 知道次数，用 for

③ **需要嵌套吗？** 否！循环里只有一个 if

结构很简单：**for 枚举 100~999 + if 判断。**

核心难点在于——怎么把一个三位数拆成百位、十位、个位？

（在黑板上写一个三位数 153，用粉笔分别圈出百位 1、十位 5、个位 3）

回顾 M1 学过的技巧：
- 个位 = `n % 10`（153 % 10 = 3）
- 十位 = `n / 10 % 10`（153 / 10 = 15, 15 % 10 = 5）
- 百位 = `n / 100`（153 / 100 = 1）

把这三个数字分别立方再相加，和 n 比较——相等就是水仙花数。

```cpp
for (int n = 100; n <= 999; n++) {
    int ge  = n % 10;          // 个位
    int shi = n / 10 % 10;     // 十位
    int bai = n / 100;         // 百位

    // 各位立方和
    int sum = ge * ge * ge + shi * shi * shi + bai * bai * bai;

    if (sum == n) {
        cout << n << " 是水仙花数" << endl;
    }
}
```

（运行代码，展示输出：153、370、371、407）

一共四个水仙花数。来，我们手动验证一下 371：3³=27，7³=343，1³=1，27+343+1=371——对的！

这个解题模式叫"**枚举 + 拆位 + 判定**"，在算法竞赛中极其常见：

1. **枚举**：用 for 循环遍历所有可能的值（100~999）
2. **拆位**：对每个值，用 `%` 和 `/` 提取出需要的各部分
3. **判定**：用 if 判断是否满足条件

（在黑板上写出这三个步骤）

这个模式可以推广到很多问题：
- 判断一个数是不是回文数
- 找所有满足特定条件的数
- 统计满足条件的数的个数

比如"四叶玫瑰数"——四位数的各位四次方之和等于自身。只需把范围改成 1000~9999，拆位改成四位（加一个千位），立方改成四次方。

现在动手练习：找所有的"四叶玫瑰数"（四位数，各位四次方和等于自身）。提示：千位 = n/1000，百位 = n/100%10。

（给学生练习时间）

小结：水仙花数是"枚举+拆位+判定"模式的经典例题。这个模式 = for 枚举范围 + %/ 拆位 + if 判定。记住它，以后遇到类似题直接套。
