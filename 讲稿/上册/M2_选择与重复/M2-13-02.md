# 第13章 第2课时：矩形与三角形——嵌套循环画图形
**时长**：15-20分钟 | **对应教材**：第13章 · 13.2节、13.3节

## 🎯 本课时目标
- 能用嵌套循环打印矩形（外层管行、内层管列）
- 理解三角形的秘密：内层循环上限 = 外层变量
- 能举一反三打印正三角形和倒三角形

## 📝 讲稿正文

同学们好！上节课我们学了嵌套循环的基本概念——外层管行，内层管列。今天我们来"画图"——用星号在屏幕上打印各种图形。虽然看起来像玩，但这其实在训练你对循环变量的控制力。

先来最简单的——**矩形**。我们要打印一个 5 行 8 列的星号矩形：

```cpp
for (int i = 1; i <= 5; i++) {       // 外层：控制行数
    for (int j = 1; j <= 8; j++) {   // 内层：控制每行打印个数
        cout << "*";
    }
    cout << endl;                     // 一行打完，换行
}
```

输出：

```
********
********
********
********
********
```

矩形最简单，因为内层上限 `8` 是固定的——每行都打 8 个星号，跟 `i` 没关系。

好，现在升级——**三角形**。第一行 1 颗星，第二行 2 颗，第三行 3 颗……第五行 5 颗。

（在黑板上画一个三角形）

关键发现：**第 i 行的星星数正好等于 i！** 所以内层循环的上限不能写固定的 5 或 8，而要写 `i`：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {   // ← j <= i 是关键！第 i 行打 i 颗星
        cout << "*";
    }
    cout << endl;
}
```

输出：

```
*
**
***
****
*****
```

看到没？内层上限 `j <= i` 不是固定的，而是**动态的**——i 变大，上限就变大；i 是多少，就打多少颗星。这就是三角形的秘密。

（在黑板上用不同颜色粉笔圈出 `j <= i`）

来，我们走一遍。i=1 时，内层 `j <= 1`，只打 1 颗星。i=2 时，内层 `j <= 2`，打 2 颗星……i=5 时，内层 `j <= 5`，打 5 颗星。

这个"内层上限用外层变量"的思路非常强大，可以变出各种形状。比如，第一行打数字 1，第二行打数字 1 和 2，第三行打 1 2 3……把星号换成输出 j：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {
        cout << j;              // 输出数字而不是星号
    }
    cout << endl;
}
// 输出：
// 1
// 12
// 123
// 1234
// 12345
```

再来看**倒三角形**——第一行 5 颗，第二行 4 颗……最后一行 1 颗：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 6 - i; j++) {   // 第 i 行打 6-i 颗
        cout << "*";
    }
    cout << endl;
}
// 输出：
// *****
// ****
// ***
// **
// *
```

来验证：i=1 时 `6-1=5` 颗；i=2 时 `6-2=4` 颗；i=5 时 `6-5=1` 颗——完美倒三角！

（在黑板上写出三种图案的公式对比）

总结一下内层上限的写法：

| 图案 | 内层上限 | 说明 |
|------|----------|------|
| 矩形 | 固定值（如 8） | 每行一样多 |
| 正三角 | `i` | 行号越大，星号越多 |
| 倒三角 | `n+1-i` 或 `6-i` | 行号越大，星号越少 |

核心就是一句话：**改变内层循环的上限，就能改变图形的形状。** 上限如果是固定值，就是矩形；如果跟随行号 i 变化，就能产生三角形。

现在动手练习：用嵌套循环打印下面的数字三角形——

```
1
22
333
4444
55555
```

（提示：不是输出 j，而是输出 i）

（给学生练习时间）

小结：矩形用固定上限，三角形用动态上限（`j <= i` 或 `j <= n+1-i`）。掌握了这个技巧，你能在屏幕上画出各种形状。

## 🖥️ 板书/屏幕要点
- 矩形：内层上限固定 `j <= 8`
- 正三角：内层上限 = i → `j <= i`
- 倒三角：内层上限 = n+1-i → `j <= 6-i`
- 数字三角形：星号改成输出 j 或 i


## 📌 小结
- 矩形 = 内层上限固定
- 三角形 = 内层上限跟随外层变量 i
- 正三角形 `j <= i`，倒三角形 `j <= n+1-i`
- 改变内层上限就能改变图形形状
