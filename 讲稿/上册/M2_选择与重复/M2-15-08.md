## 📝 讲稿正文

同学们好！第 2 课时我们写了基础版猜数字——一次猜对就结束。今天来做一个增强版，加上三个新功能：

1. **限制次数**：最多猜 7 次，超过就输
2. **历史记录**：猜完后显示所有猜测过程
3. **多局循环**：支持"再来一局"

先做功能 1——限次。基础版用的是 `while(true)`，现在要改成有上限的循环。用 for 最合适：

```cpp
int secret = rand() % 100 + 1;
int guess;

for (int cnt = 1; cnt <= 7; cnt++) {
    cout << "第" << cnt << "次猜测（共7次）：";
    cin >> guess;

    if (guess > secret) {
        cout << "太大了！" << endl;
    } else if (guess < secret) {
        cout << "太小了！" << endl;
    } else {
        cout << "恭喜猜对！你用了 " << cnt << " 次。" << endl;
        break;                    // 猜对提前退出
    }

    if (cnt == 7) {
        cout << "7 次机会用完！答案是 " << secret << endl;
    }
}
```

注意一个细节——猜对时用 break 提前退出。如果没猜对，for 循环跑到第 7 次自然结束。第 7 次猜完如果还没对，用 `if (cnt == 7)` 判断并显示答案。

现在加功能 2——历史记录。问题是我们还没学数组，怎么保存每次的猜测？一个取巧的办法：用 `string` 拼接：

```cpp
string history = "";   // 历史记录字符串

for (int cnt = 1; cnt <= 7; cnt++) {
    // ... 读入 guess ...

    // 把本次猜测拼进历史
    history += to_string(guess) + " ";

    if (guess == secret) {
        cout << "恭喜猜对！你的猜测过程：" << history << endl;
        break;
    }
    // ...
}
```

`to_string(guess)` 把整数转成字符串。每次猜完就拼进 `history`。注意需要 `#include <string>`。

最后加功能 3——多局循环。用 do-while 问"再来一局？"：

```cpp
char again;
do {
    // 一局游戏的完整代码
    // ...

    cout << "再来一局？(y/n)：";
    cin >> again;
} while (again == 'y' || again == 'Y');
```

（展示完整代码结构）

现在把所有部分整合起来：

```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
#include <string>
using namespace std;

int main() {
    srand(time(0));
    char again;

    do {
        int secret = rand() % 100 + 1;
        int guess;
        string history = "";

        cout << "\n=== 猜数字游戏 ===" << endl;
        cout << "我想好了一个 1~100 的数字，你有 7 次机会！" << endl;

        bool won = false;
        for (int cnt = 1; cnt <= 7; cnt++) {
            cout << "第" << cnt << "次：";
            cin >> guess;
            history += to_string(guess) + " ";

            if (guess > secret) {
                cout << "太大了！" << endl;
            } else if (guess < secret) {
                cout << "太小了！" << endl;
            } else {
                cout << "恭喜猜对！" << endl;
                cout << "你的猜测过程：" << history << endl;
                won = true;
                break;
            }
        }
        if (!won) {
            cout << "7 次机会用完！答案是 " << secret << endl;
            cout << "你的猜测过程：" << history << endl;
        }

        cout << "再来一局？(y/n)：";
        cin >> again;
    } while (again == 'y' || again == 'Y');

    cout << "再见！" << endl;
    return 0;
}
```

（带着学生逐段阅读和理解）

这个增强版猜数字融合了：for 限次循环、if-else 判断、break 提前退出、string 拼接历史、do-while 多局循环——几乎用上了 M2 所有核心知识。

这就是"迭代开发"的思维——先写一个能跑的简单版，然后逐步加功能。不要一上来就想写完美的最终版本，那会让你无从下手。

现在动手练习：在增强版基础上再加一个功能——"难度选择"。开始前让用户选难度：1=简单(1~50, 10次)、2=中等(1~100, 7次)、3=困难(1~200, 5次)。

（给学生练习时间）

小结：增强版猜数字展示了"迭代开发"的思维——基础版→加限次→加历史→加多局。这个思维比具体代码更重要。

## 🖥️ 板书/屏幕要点
- 迭代开发：基础版→限次→历史→多局→难度
- 限次：`for (cnt=1; cnt<=7; cnt++)` + break
- 历史：`string history += to_string(guess)`
- 多局：`do { ... } while (again=='y')`
- 融合知识点：for/while/if-else/break/string/do-while
