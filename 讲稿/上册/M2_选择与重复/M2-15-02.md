## 📝 讲稿正文

同学们好！今天我们来写 M2 的第一个"完整的应用程序"——猜数字游戏。这道题融合了循环和分支，是"三步选结构"思维的绝佳练习。

先来分析题目：程序随机生成一个 1~100 的整数，玩家每次猜一个数，程序回复"大了"或"小了"，直到猜对。

走三步选结构：

① **需要分岔吗？** 是！每次猜完要判断三种情况——猜大了、猜小了、猜对了。→ 用 if-else if-else

② **需要重复吗？** 是！需要反复猜直到猜对。知道次数吗？不知道——可能一次就中，可能猜 10 次。→ 用 while

③ **需要嵌套吗？** 是！循环里面要 if 判断大小。→ 循环套分支

结构定了：**while 循环套 if-else if-else 分支。**

现在写代码。先解决一个问题：怎么生成 1~100 的随机数？

```cpp
#include <cstdlib>   // rand() 和 srand()
#include <ctime>     // time()

srand(time(0));                     // 用当前时间做随机种子
int secret = rand() % 100 + 1;      // 生成 1~100 的随机数
```

`rand()` 生成一个 0~32767 的随机整数。`% 100` 取余得到 0~99，`+1` 变成 1~100。`srand(time(0))` 用当前时间做种子，确保每次运行生成不同的数字。

好，完整代码：

```cpp
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int secret = rand() % 100 + 1;  // 1~100 的秘密数字
    int guess, cnt = 0;

    cout << "我想好了一个 1~100 之间的数字，你猜猜看？" << endl;

    while (true) {
        cout << "你的猜测：";
        cin >> guess;
        cnt++;                       // 猜了一次

        if (guess > secret) {
            cout << "太大了！再试试。" << endl;
        } else if (guess < secret) {
            cout << "太小了！再试试。" << endl;
        } else {
            cout << "恭喜猜对！你一共猜了 " << cnt << " 次。" << endl;
            break;                   // 猜对了，退出循环
        }
    }
    return 0;
}
```

来，我们走一遍关键设计。

第一，`while (true)` —— 无限循环。因为我们不知道要猜几次，所以让循环永远跑，猜对了用 break 退出。这是 `while(true) + break` 的经典模式。

第二，`cnt++` —— 计数器。每次猜都加 1，最后告诉你猜了几次。这是第 11 章学的计数模式。

第三，if-else if-else —— 三种情况各走一条路。"太大""太小"都不 break（回到循环继续猜），只有猜对才 break。

（在黑板上画出这个程序的流程图）

现在做一个实验：用二分查找的策略来猜（每次猜中间值），看看最多几次能猜中 1~100 范围的数字？

答案是——最多 7 次。因为 2⁷ = 128 > 100。二分查找的威力：100 个可能，7 次提问一定找到。这就是计算机科学里"对数复杂度"的直观体现。

（带学生试几次二分查找）

现在动手练习：在你的电脑上跑一遍完整的猜数字游戏。然后做一个增强：记录历史猜测，猜完后把所有猜测过程打印出来。提示：需要用一个变量存储历史——但我们还没学数组。有个取巧的方法：用 `string` 把每次的猜测拼接起来。

（给学生练习时间）

小结：猜数字 = while(true) + if-else 判断 + break。这是循环套分支的经典例题。cnt 计数器记录次数，break 在猜对时退出。

## 🖥️ 板书/屏幕要点
- 猜数字 = while(true) + if-else 分支 + break
- 随机数：`srand(time(0)); rand() % 100 + 1;`
- 计数器 cnt 统计猜测次数
- 二分查找：100 个数最多 7 次找到

## 📌 小结
- 三步选结构：循环套分支
- `while(true)` + break 是不知道次数的标准模式
- `rand() % 100 + 1` 生成 1~100 随机数
- 计数器是循环的好搭档
