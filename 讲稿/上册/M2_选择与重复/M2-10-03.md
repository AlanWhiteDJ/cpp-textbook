# 第10章 第3课时：兜底与边界——default 与 switch 限制
**时长**：15-20分钟 | **对应教材**：第10章 · 10.4节

## 🎯 本课时目标
- 理解 default 的作用：处理所有 case 都不匹配的"兜底"情况
- 掌握 switch-case 的三条限制及其原因
- 能写出完整的、健壮的 switch-case 结构

## 📝 讲稿正文

同学们好！前面两节课我们学了 switch-case 的基本写法和穿透机制。今天讲两个重要的内容：default 的作用，以及 switch-case 不能用在哪里。

先看一个场景。你写了一个自动售货机程序，选项是 1~5。某天，一个小朋友乱按键盘，输入了 `99`。你的 switch 代码里没有 `case 99`——程序怎么办？是崩溃？是输出乱码？还是什么都不做？

（停顿）

C++ 的 switch 如果不写 `default`，遇到没有匹配的 case 时，程序**直接跳过整个 switch 块**——什么都不做。虽然不会崩溃，但这件事很不好——用户输入了错误的数字，程序什么都没说，用户会一头雾水。

所以，一个好的 switch 语句通常都包含 `default`：

```cpp
switch (choice) {
    case 1: cout << "可乐" << endl; break;
    case 2: cout << "雪碧" << endl; break;
    // ...
    default:
        cout << "没有这个编号，请重新选择" << endl;
}
```

`default` 用中文翻译就是"默认"。它的含义是：**如果上面的所有 case 都不匹配，就走这里。** 就像你去商店买东西，你有一张清单列出了要买的所有东西——default 就是"清单上没写的东西，一律不买"。

在 switch 里，default 通常放在最后，但实际上它可以放在任何位置——开头、中间、末尾都行。不过摆在最后最符合人的阅读习惯，所以我们都放最后。

（在黑板上写出完整的 switch 结构）

好，接下来讲 switch-case 的限制。很多同学学了 switch 之后，会觉得"哇这个好用"，然后什么都想用 switch。但 switch 是有明确边界的。

**限制一：switch 后面的表达式必须是整型或字符型。**

这意味着你可以写 `switch (n)`（int）、`switch (c)`（char）、`switch (year)`（long long），但不能写 `switch (price)`（double）——因为浮点数可能由于精度问题永远匹配不到 case，也不能写 `switch (name)`（string）——C++ 的 switch 不支持字符串比较。

为什么？因为 switch 在底层是用"跳转表"实现的——编译器会生成一个表格，每个 case 值对应一个跳转地址。这个表格要求索引值是整数——浮点数有精度问题不适合做索引，字符串更不可能。

所以如果你的条件涉及浮点数范围（比如"分数 > 89.5"），或者涉及字符串比较（比如"姓名是张三"），老老实实用 if-else。

**限制二：case 后面必须是常量。**

你不能写 `case x:`（x 是变量），也不能写 `case a+b:`（如果 a+b 在运行时才确定）。case 的值在编译时就得定下来。这也是跳转表的限制决定的。

这也意味着你不能写 `case 60..100:`——C++ 不支持范围 case。不过你可以用上节课学的技巧 `switch (score / 10)` 来变通。

**限制三：各 case 的值不能重复。**

如果你写了两个 `case 1:`，编译器直接报错。这很好理解——如果值 1 对应两个入口，程序该跳哪个？

好，总结一下：switch 的限制不是缺陷，是设计选择。它的优势在于代码清晰、执行高效（跳转表比逐条判断快）；代价是不灵活——不能判范围、不能用浮点、不能比字符串。

现在做一个综合练习：写一个"简易计算器"——输入两个整数和一个运算符（`+` `-` `*` `/`），用 switch 判断运算符计算并输出结果。注意：除法时检查除数是否为 0，运算符不合法时用 default 提示。

（给学生练习时间）

记住：default 是善后，switch 有边界。知道能用 switch 的地方写 switch，知道不能用 switch 的地方回到 if-else——这就是成熟的程序员。

## 🖥️ 板书/屏幕要点
- default：所有 case 都不匹配时的"兜底"
- 限制1：表达式必须整型/字符型（不能 double、不能 string）
- 限制2：case 必须是常量（不能变量、不能范围）
- 限制3：各 case 值不能重复
- 范围变通法：`switch (score / 10)`

## ✏️ 随堂练习
写一个"简易计算器"：输入两个整数和一个运算符（`+` `-` `*` `/`），用 switch 判断运算符计算并输出结果。注意除零检查和非法运算符的 default 处理。

## 📌 小结
- default 处理"都不匹配"的情况，是健壮程序的关键
- switch 三条限制：整型/字符型、case 常量、值不重复
- 遇到范围判断、浮点比较、字符串匹配 → 回到 if-else
- 完整的 switch = case 全覆盖 + break 防穿透 + default 兜底
