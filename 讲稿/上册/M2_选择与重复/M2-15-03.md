## 📝 讲稿正文

同学们好！上节课我们写了猜数字——while 套 if-else。今天写一个更复杂的程序：循环菜单计算器。

题目：程序显示菜单（1.加法 2.减法 3.乘法 4.除法 0.退出），用户选择后输入两个数计算结果，算完回到菜单，选 0 退出。

走三步选结构：

① **需要分岔吗？** 是！选 1 做加法、选 2 做减法……这是"多选一"→ 用 switch-case

② **需要重复吗？** 是！菜单要反复显示，直到用户选 0 → 用 while

③ **需要嵌套吗？** 是！循环里面套 switch → 循环套分支

结构定了：**while 循环套 switch-case 分支。**

（在黑板上画出结构图：一个大循环框里包着一个 switch）

好，我们来写代码。这道题还要用到 M1 的知识——四则运算和输入输出：

```cpp
#include <iostream>
using namespace std;

int main() {
    int choice;
    double a, b;

    while (true) {
        // 显示菜单
        cout << "\n=== 计算器菜单 ===" << endl;
        cout << "1. 加法  2. 减法  3. 乘法  4. 除法  0. 退出" << endl;
        cout << "请选择：";
        cin >> choice;

        // 判断是否退出
        if (choice == 0) {
            cout << "再见！" << endl;
            break;
        }

        // 检查输入合法性
        if (choice < 1 || choice > 4) {
            cout << "无效选择，请重试。" << endl;
            continue;   // 跳过本次循环，重新显示菜单
        }

        // 读入两个操作数
        cout << "输入两个数：";
        cin >> a >> b;

        // 根据选择执行运算
        switch (choice) {
            case 1:
                cout << a << " + " << b << " = " << a + b << endl;
                break;
            case 2:
                cout << a << " - " << b << " = " << a - b << endl;
                break;
            case 3:
                cout << a << " * " << b << " = " << a * b << endl;
                break;
            case 4:
                if (b == 0)
                    cout << "错误：除数不能为 0！" << endl;
                else
                    cout << a << " / " << b << " = " << a / b << endl;
                break;
        }
    }
    return 0;
}
```

来，我们逐块分析这段代码的设计。

**外层框架**：`while (true)` —— 让菜单反复显示，用户选 0 才 break 退出。这是循环菜单的标准框架。

**退出判断**：`if (choice == 0) break;` —— 放在最前面。用户选了 0 就不用再执行后面的代码了。

**输入校验**：`if (choice < 1 || choice > 4) continue;` —— 如果用户选了菜单上没有的数字（比如 5），提示错误后 `continue` 跳过本次循环，重新显示菜单。这里 continue 用得很合适——过滤掉无效输入，返回循环开头。

**switch 分支**：四种运算各一个 case。注意除法的除零检查——`if (b == 0)` 保护程序不崩溃。

这段代码看着长，但结构非常清晰——外层 while 管循环，内层 switch 管分岔，中间用 if 做合法性检查。它把 M1（运算、输入输出）和 M2（分支、循环、switch、continue）几乎全部知识都串起来了。

（在黑板上标注各个 M1 和 M2 知识点在代码中的位置）

这个程序还有一个细节：运算数的类型是 `double` 而不是 `int`。为什么？因为除法 5/2 的结果是 2.5——用 int 就变成 2 了。用 double 保留小数，更实用。

- continue 用于跳过无效输入
- 综合了 M1（运算）和 M2（分支/循环/break/continue）
