# 第13章 第1课时：循环里面套循环——嵌套循环的概念
**时长**：15-20分钟 | **对应教材**：第13章 · 13.1节

## 🎯 本课时目标
- 理解什么是嵌套循环，以及外层和内层的关系
- 掌握嵌套循环的执行顺序：外层走一步，内层跑一轮
- 能写出简单的嵌套 for 循环

## 📝 讲稿正文

同学们好！欢迎来到第13章——嵌套循环。前面三章我们学会了用 for、while、do-while 做重复的事。但你有没有想过——循环里面要做的事本身也包含重复，怎么办？

（指着墙上的时钟）

看墙上的时钟。时钟有三根针：时针、分针、秒针。秒针每走 60 圈，分针才走 1 格；分针每走 60 圈，时针才走 1 格。

秒针的每一圈都包含 60 次"滴答"——这已经是一层循环了。而秒针走 60 圈分针才走一格——这是循环套着循环。再往外，分针走 60 圈时针走一格——三层嵌套！

如果把时钟写成程序，大概是这样的：

```
对于 时针 从 1 到 12：
    对于 分针 从 0 到 59：
        对于 秒针 从 0 到 59：
            滴答！
```

这就是**嵌套循环**——一个循环的"身体"里面又写了一个循环。外面那个叫**外层循环**，里面那个叫**内层循环**。

（在黑板上画一个大方框套一个小方框）

再举一个更贴近生活的例子。体育课上，老师说："全班 4 排，每排同学依次报数 1 到 5。"你大脑里自动就是：

```
对于每一排（1 到 4）：
    对于排里的每个人（1 到 5）：
        报数！
```

写成 C++ 就是这样：

```cpp
for (int row = 1; row <= 4; row++) {    // 外层：控制排
    for (int col = 1; col <= 5; col++) { // 内层：控制每排的人
        cout << row << "排" << col << "号  ";
    }
    cout << endl;   // 一排报完，换行
}
```

输出效果：

```
1排1号  1排2号  1排3号  1排4号  1排5号
2排1号  2排2号  2排3号  2排4号  2排5号
3排1号  3排2号  3排3号  3排4号  3排5号
4排1号  4排2号  4排3号  4排4号  4排5号
```

现在让我们像慢镜头一样，一步步看这个程序是怎么执行的：

（在黑板上逐行走一遍）

- 外层 row=1：开始第1排。内层 col 从 1 到 5 跑一整轮 → 输出"1排1号"到"1排5号"
- 内层结束，执行 `cout << endl;` 换行
- 外层 row=2：开始第2排。内层 col 又从 1 到 5 跑一整轮
- ……以此类推
- 外层 row=4：最后一排跑完，外层也结束

核心规律：**外层循环每走一次，内层循环要完整地跑一轮。** 就像体育课上——每换到新的一排，那一排的 5 个人全部报完数，才轮到下一排。

（在黑板上写：**外层→慢，内层→快。外层走一步，内层跑一圈。**）

注意两个细节：

第一，变量命名。外层通常叫 `i`，内层叫 `j`。如果有第三层，叫 `k`。这是从数学的向量（xi, xj, xk）沿用下来的习惯，全球程序员都这样用。你在内层写 `i` 而外层也有 `i`？那就出事了——内层的 `i` 会"遮蔽"外层的 `i`，导致无法访问外层变量。所以**嵌套循环绝不能用相同的变量名。**

第二，`cout << endl;` 的位置。它应该放在**外层循环里、内层循环之后**。如果放到内层里面，每输出一个数字就换一行，输出就变成一竖条了。

现在动手练习：用嵌套 for 循环写一个程序，输出 3 行 6 列的星号矩阵。格式：

```
******
******
******
```

（给学生练习时间）

小结：嵌套循环 = 循环里面套循环。外层走一步，内层跑一轮。变量最好用 i、j、k 区分，换行放在外层。

## 🖥️ 板书/屏幕要点
- 嵌套循环：`for (i) { for (j) { ... } }`
- 外层控制"组"（如行），内层控制"组内的元素"（如列）
- 执行顺序：外层走一步，内层跑一轮
- 变量命名：i 外层、j 内层（绝不能重复）
- `endl` 放外层，在内层循环之后

## 📌 小结
- 嵌套循环 = 一个循环的循环体里又有一个循环
- 执行顺序：外层慢，内层快；外层走一步，内层跑一圈
- 命名习惯：i 外层 / j 内层，绝不重名
- 换行 `endl` 放在外层、内层之后
