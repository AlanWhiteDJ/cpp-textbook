# 第9章 第2课时：二选一——if-else 与关系运算符
**时长**：15-20分钟 | **对应教材**：第9章 · 9.2节、9.3节

## 🎯 本课时目标
- 掌握 `if-else` 二选一结构的语法和语义
- 熟练使用六种关系运算符（`==` `!=` `<` `>` `<=` `>=`）
- 能区分"单分支"和"双分支"的使用场景

## 📝 讲稿正文

同学们好！上节课我们学了 `if` —— 最简单的判断。但是，生活里很多时候不是"做或不做"，而是"**做这个或者做那个**"。

（站在讲台中间，用手指左边和右边）

比如：考试分数 ≥ 60，就输出"及格"；否则，输出"不及格"。这两条路必须走一条，而且只能走一条。你不能既及格又不及格，对吗？

这种情况，C++ 提供的结构叫 `if-else`。格式长这样：

```cpp
if (条件) {
    // 条件成立 → 走这条路
} else {
    // 条件不成立 → 走那条路
}
```

请注意，`else` 后面没有括号！因为 else 的意思是"剩下的所有情况"，不需要再写条件。

来，我们把刚才考试及格的例子写成代码：

```cpp
int score;
cin >> score;
if (score >= 60) {
    cout << "及格！" << endl;
} else {
    cout << "不及格，继续努力！" << endl;
}
```

（在黑板上画一个菱形，上面写"score ≥ 60？"，左边箭头写"及格"，右边箭头写"不及格"）

好，现在有一个非常重要的问题：if 后面括号里的条件怎么写？这就必须来学**关系运算符**了。关系运算符就是 C++ 里用来判断两个东西之间大小关系的符号。一共六个，我们一个个过一遍：

| 运算符 | 含义 | 例子 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | true |
| `!=` | 不等于 | `5 != 3` | true |
| `<` | 小于 | `3 < 5` | true |
| `>` | 大于 | `7 > 2` | true |
| `<=` | 小于等于 | `5 <= 5` | true |
| `>=` | 大于等于 | `5 >= 3` | true |

这里我必须停下来强调一个超级大坑——**判断相等用 `==`，不是 `=`！**（在黑板上用红笔重重圈出 `==`）

有同学可能不以为然："不就是一个等号和两个等号嘛，能有多大区别？"

来，咱们做一个实验。请敲下面这段代码，输入 `3`，看看输出什么：

```cpp
int x;
cin >> x;
if (x = 5) {
    cout << "x 等于 5！" << endl;
}
```

（等学生跑完）

什么？你明明输入了 `3`，程序竟然输出"x 等于 5"？你是不是觉得见鬼了？

其实不奇怪。`x = 5` 是**赋值**，它做了两件事：第一，把 5 塞进 x 里；第二，整个 `x = 5` 这个表达式本身的值也是 5。在 C++ 里，非零的值都当 true 处理，所以 `if (x = 5)` 的条件永远成立——不管 x 原来是什么，赋值后一定是 5，5 非零即真，条件必定成立。

正确写法是：

```cpp
if (x == 5) {   // 两个等号！这是比较！
    cout << "x 等于 5！" << endl;
}
```

（停顿，看着学生）

好，现在请你们记住一句口诀：**"if 里面，判断相等用双等，单等是赋值。"** 这句话你们可以贴在显示器上，每天看一眼，能帮你省下几十个小时的调试时间。

我们来练几个判断题，检验一下：
- `3 != 5` —— 真还是假？（真，3 不等于 5）
- `7 <= 7` —— 真还是假？（真，7 小于等于 7）
- `10 < 5` —— 真还是假？（假，10 不小于 5）

很好！现在请大家做一个综合练习：输入一个人的年龄，如果 ≥ 18 就输出"成年"，否则输出"未成年"。用 if-else 实现。

（给学生练习时间）

好了，今天的内容就到这里。记住三个要点：第一，if-else 是二选一，两条路必走一条且只走一条；第二，六个关系运算符要记熟，尤其是 `==` 两个等号；第三，`=` 是赋值不是比较——这条坑能坑你一整年，今天你知道了它就坑不到你。

2. "`x = 5` 和 `x == 5` 有什么区别？谁能用自己的话解释一下？"
3. "条件 `10 >= 10` 的结果是真还是假？为什么？"

## 🖥️ 板书/屏幕要点
- `if (条件) { A } else { B }` —— 二选一，必走一条且只走一条
- 六大关系运算符：`==` `!=` `<` `>` `<=` `>=`
- ⚠️ 死记硬背：`==` 是比较，`=` 是赋值
- `if (x = 5)` 永远为真——赋值表达式的值是被赋的那个值

## ✏️ 随堂练习
写一个程序：输入两个整数 a 和 b，输出其中较大的那个数。用 if-else 结构实现。

## 📌 小结
- if-else 是"二选一"的分支结构
- 六种关系运算符用于表达"条件"
- `==` 是比较相等，`=` 是赋值——这是最容易踩的坑
- else 后面不需要条件，它代表"所有剩余情况"
