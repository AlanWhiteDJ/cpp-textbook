# 第10章 第4课时：一行定乾坤——三目运算符 ? :
**时长**：15-20分钟 | **对应教材**：第10章 · 10.5节

## 🎯 本课时目标
- 掌握三目运算符 `? :` 的语法和语义
- 能用三目运算符简洁地表达"二选一"的赋值或输出
- 理解三目运算符的适用边界——简单二选一用三目，复杂逻辑回 if-else

## 📝 讲稿正文

同学们好！今天我们学一个长得有点奇怪但特别有用的运算符——三目运算符 `? :`。

先看一个场景：输入两个整数 a 和 b，把其中较大的那个赋值给变量 max。

用 if-else 写，要写 5 行：

```cpp
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}
```

就为了把较大的数放进 max 里，写了整整 5 行——两行花括号、一个 if、一个 else，太啰嗦了。

C++ 有一个更简洁的写法：

```cpp
int max = (a > b) ? a : b;   // 一行搞定！
```

这就是三目运算符 `? :`——C++ 里唯一一个需要三个操作数的运算符。它的格式是：

```
条件 ? 成立时的值 : 不成立时的值
```

翻译成人话：**"条件成立吗？成立就选冒号前面的，不成立就选冒号后面的。"**

（在黑板上画一个问号和冒号的图：问号前面是条件，问号和冒号之间是"成立值"，冒号后面是"不成立值"）

它之所以叫"三目"，是因为它需要三个东西：条件、值1、值2。对比一下，加法 `a + b` 是两目运算符（两个操作数），取负 `-a` 是单目运算符（一个操作数）。

来，看几个典型用法：

```cpp
// 求绝对值
int absValue = (x >= 0) ? x : -x;

// 判断奇偶并输出
cout << ((n % 2 == 0) ? "偶数" : "奇数") << endl;

// 判断及格
string status = (score >= 60) ? "及格" : "不及格";

// 找两个数中的最小值
int min = (a < b) ? a : b;
```

注意第三行的 `string` 类型——三目运算符的两个分支可以返回不同类型的值，但必须是"兼容的"（或者能自动转换的）。更安全的做法是让两个分支的类型一致。

这里有个小技巧：如果你要在 `cout` 里用三目运算符，因为 `<<` 的优先级比 `? :` 高，所以**整个三目表达式要加括号**：

```cpp
cout << (n % 2 == 0 ? "偶数" : "奇数");  // ✅ 括号包裹
cout << n % 2 == 0 ? "偶数" : "奇数";    // ❌ 优先级错误！
```

不加括号的话，编译器会把 `cout << n % 2 == 0` 当作整体先算，然后再算 `? :`，结果完全乱套。

好，现在讲一个非常重要的使用原则：**三目运算符不要嵌套！**

如果有人写出这样的代码：

```cpp
int r = a > b ? (c > d ? e : f) : (g > h ? i : j);
```

你能看懂吗？我反正是看不懂。这种代码只有写的人自己能看懂——而且过三天连他自己都不一定看得懂。

三目运算符的设计初衷是"简单二选一"。一旦条件变复杂了——嵌套的、多分支的——就老老实实回到 if-else。代码不是越短越好，是**越清晰越好**。

什么时候用三目，什么时候用 if-else？一句话：**如果你能用一句话说清楚"如果……就……否则……"，就用三目；如果需要"如果……就……否则如果……否则……"，就用 if-else。**

好，现在动手练习。用三目运算符写三段代码：
1. 输入一个整数，输出"正数"或"非正数"
2. 输入两个整数，输出较大的那个
3. 输入一个整数，输出它的绝对值

（给学生练习时间）

小结：三目运算符 `? :` 是简单二选一的神器，一行搞定 if-else 赋值。但要记住：不要嵌套，不要为了炫技牺牲可读性。简单场景用它，复杂场景回 if-else。

## 🖥️ 板书/屏幕要点
- `条件 ? 成立值 : 不成立值`
- 三目 = 三个操作数（条件、值1、值2）
- ⚠️ cout 里嵌套三目必须加括号：`cout << (条件 ? A : B);`
- ❌ 三目不要嵌套——复杂逻辑回 if-else

## 📌 小结
- `? :` 是 C++ 唯一的三目运算符
- 简单二选一用它，一行顶五行
- cout 里用必须加全括号
- 禁止嵌套——清晰 > 简短
