# 第11章 第2课时：循环变量与多变量 for
**时长**：15-20分钟 | **对应教材**：第11章 · 11.3节、11.4节

## 🎯 本课时目标
- 理解循环变量的作用域：在 for 括号内声明 vs 在 for 外声明
- 掌握多变量 for 循环的写法
- 能用 for 循环解决"从两端到中间"的对称问题

## 📝 讲稿正文

同学们好！上节课我们学了 for 循环的三要素，循环变量 `i` 贯穿始终。今天我们来深挖两个问题：第一，循环变量 i 出了循环还能用吗？第二，for 循环能同时管多个变量吗？

先看第一个问题。看这两段代码，有什么区别：

```cpp
// 写法一：i 在 for 的括号里声明
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
cout << i;   // ❌ 编译错误！i 不存在！
```

```cpp
// 写法二：i 在 for 之前声明
int i;
for (i = 1; i <= 5; i++) {
    cout << i << " ";
}
cout << "循环结束后 i = " << i << endl;   // ✅ i = 6，能访问！
```

（在黑板上画出两种写法的对比）

区别在于——写法一在 `for()` 的括号里用 `int i` 声明变量，这个 i 的**作用域**仅限于 for 循环内部。循环一结束，i 就"死"了，外面访问不到。写法二在 for 外面声明 i，它的作用域更大，循环结束后还能用。

这就像两种"借书"的方式。写法一是你在图书馆里借书——只能在馆内看，带不出去。写法二是你把书买回家——随便看多久。

（在黑板上写：**在 for 括号里声明的变量 → 出了循环就消失**）

绝大多数时候，你不需要在循环外面用循环变量。所以推荐写法一——把 i 关在循环里，干净利落，不会和外面的变量搞混。

知道这个规则后，看一个常见的坑：

```cpp
for (int i = 1; i <= 5; i++) {
    int sum = 0;  // 每轮循环都重新创建 sum！
    sum += i;
    cout << sum << " ";
}
// 输出：1 2 3 4 5（每次 sum 都被重置为 0）
```

这里 `sum` 每次循环都会重新创建并初始化为 0，所以累加根本没效果。应该把 `sum` 的声明放在循环外面。

好，接下来看第二个重点——**多变量 for**。

for 循环的初始化和迭代部分都可以同时处理多个变量。格式是用逗号分隔：

```cpp
for (int i = 0, j = n; i < j; i++, j--) {
    cout << "i = " << i << ", j = " << j << endl;
}
```

这段代码里，i 从 0 往上走，j 从 n 往下降——两个人在独木桥上对向而行，i < j 时他们还"在路上"，一旦相遇（i >= j）就停下来。

多变量 for 在处理"从两端到中间"的问题时特别方便。比如判断一个字符串是不是回文（正着读反着读一样），思路就是：i 从左边出发，j 从右边出发，每次比一对字符，一直比到 i 和 j 相遇。

再比如你想用一个循环同时输出奇数和偶数序列：

```cpp
for (int odd = 1, even = 2; odd <= 19; odd += 2, even += 2) {
    cout << "奇数：" << odd << "  偶数：" << even << endl;
}
```

注意：多个变量必须是**同一种类型**——`int i = 0, j = n` 可以，`int i = 0, double j = n` 不行。如果类型不同，可以先在外面声明。

多变量 for 有一个容易犯错的地方——写多了逗号点不到分号。记住：初始化里的多个声明用逗号隔，但三个大段（初始化、条件、迭代）之间是用分号隔的。比如 `for (int i = 0, j = n; i < j; i++, j--)`——i 和 j 之间是逗号，但初始化 `int i=0, j=n` 和条件 `i < j` 之间是分号。

（在黑板上圈出分号和逗号的区别）

好，现在动手练习：写一个 for 循环，同时输出数列 1 到 10 和 10 到 1，每行显示两个数：第 1 行 `1 10`，第 2 行 `2 9`……第 10 行 `10 1`。用多变量 for。

（给学生练习时间）

小结：循环变量的作用域决定了它在外面的可见性——推荐在 for 里声明，保持变量局部化。多变量 for 用逗号同时初始化多个变量，适合"两端向中间"或"同步更新"的场景。

## 🖥️ 板书/屏幕要点
- 作用域：在 for() 里声明的变量 → 仅循环内可用
- 写法选择：推荐 `for (int i=...)`，把变量关在循环里
- 多变量 for：`for (int i=0, j=n; i<j; i++, j--)`
  - 逗号隔变量，分号隔三要素
  - 所有变量必须同类型

## ✏️ 随堂练习
用多变量 for 循环，同时输出两个数列：1→10 和 10→1，每行一对。即第1行 `1 10`，第2行 `2 9`……第10行 `10 1`。

## 📌 小结
- 循环变量作用域取决于声明位置：括号内→循环内；括号外→全局
- 推荐在 for 中声明，保持变量局部化
- 多变量 for 用逗号分隔多个变量，适合对称问题
