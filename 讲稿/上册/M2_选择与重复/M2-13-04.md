# 第13章 第4课时：空心图形——边框判断法

## 📝 讲稿正文

同学们好！前面两节课我们用嵌套循环画了实心的矩形、三角形和菱形。今天学一个新花样——**空心图形**。只在外面一圈画星号，中间掏空。

（在黑板上画一个空心矩形，圈出边框）

空心是什么意思？就是"**第一行、最后一行、第一列、最后一列才打星号，其他地方打空格。**"

这个判断要用到第 9 章学过的 `||`（或运算符）和 `if`——在嵌套循环的每一轮，判断当前坐标 (i, j) 是不是在边框上。

先看空心矩形（5 行 8 列）：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= 8; j++) {
        if (i == 1 || i == 5 || j == 1 || j == 8) {
            cout << "*";        // 在边框上 → 打星
        } else {
            cout << " ";        // 在内部 → 打空格
        }
    }
    cout << endl;
}
```

（在黑板上标注每个条件）

`i == 1`：第一行 → 打星
`i == 5`：最后一行 → 打星
`j == 1`：第一列 → 打星
`j == 8`：最后一列 → 打星

四个条件用 `||`（或）连接——只要满足任意一个，当前位置就打星。四个都不满足 → 在内部 → 打空格。

输出：

```
********
*      *
*      *
*      *
********
```

注意："打空格"非常重要——如果不打空格，内部就是空的，视觉效果上边框会歪。**空格也是字符，不能省略。**

好，把这个思路推广到三角形。空心正三角形（5 行）：

```cpp
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {
        if (i == 5 || j == 1 || j == i) {
            cout << "*";        // 最后一行、第一列、对角线
        } else {
            cout << " ";
        }
    }
    cout << endl;
}
```

输出：

```
*
**
* *
*  *
*****
```

边框条件变了：`i == 5`（最后一行全打星）、`j == 1`（第一列全打星）、`j == i`（对角线打星）。

这三个条件分别对应三角形的三条边：底边、左边、斜边。恰好覆盖了一个三角形的三条轮廓线。

（在黑板上用不同颜色画出三条边对应哪个条件）

再升级——空心菱形。它是空心等腰三角形的上下组合，边框条件要同时考虑空格和空心：

```cpp
int n = 4;
// 上半部分
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n - i; j++) cout << " ";   // 左边空格
    for (int j = 1; j <= 2 * i - 1; j++) {
        if (j == 1 || j == 2 * i - 1) cout << "*";   // 两端打星
        else cout << " ";                             // 中间空格
    }
    cout << endl;
}
// 下半部分类似，略
```

空心的核心逻辑永远一样：**if 判断哪些位置该打星，else 打空格。** 区别只在于——不同的图形，边框条件不同。

总结一个通用模板：

```
for 外层（行）：
    for 内层（列）：
        if (边框条件) 打星
        else 打空格
    换行
```

现在动手练习：打印一个 n×n 的空心正方形。n 由用户输入，比如 n=5 时：

```
*****
*   *
*   *
*   *
*****
```

（给学生练习时间）

小结：空心图形的本质是在内层循环里加 if 判断——边框打星，内部打空格。边框条件 = 第一行/最后一行/第一列/最后一列/对角线，根据图形灵活组合。

## 🖥️ 板书/屏幕要点
- 空心 = 边框打星 + 内部打空格
- 边框条件用 `||` 连接
- 矩形边框：`i==1 || i==n || j==1 || j==m`
- 三角形边框：`i==n || j==1 || j==i`
- 通用模板：for-for-if（星/空格）


## 📌 小结
- 空心图形 = 内层循环里加 if 判断
- 边框条件决定哪些位置打星
- else 必须打空格，不能省略
- 不同图形的边框条件不同，但框架相同
