## 📝 讲稿正文

同学们好！第13章我们学嵌套循环时，画了矩形、三角形、菱形、空心图形。今天我们来一个"图案大练兵"——把所有图形串起来复习，确保你能面对任何图案要求都快速写出代码。

先回顾一下嵌套循环的核心框架：

```
for (外层：控制行) {
    // 可能打印前置空格（等腰图案需要）
    for (内层：控制列) {
        // 决定打星号还是空格
    }
    cout << endl;
}
```

所有图案都遵循这个框架。区别只在于三个参数：
1. **内层循环次数**（每行打多少个）
2. **要不要前置空格**（居中效果）
3. **打星号还是空格**（空心效果）

我们来做一个"快速反应"训练——我说图案特征，你来说内层条件和特殊处理。

**图案 1：n 行矩形，每行 m 个星号**
（学生回答：内层 `j <= m`，无空格，全打星）

**图案 2：n 行正三角形**
（学生回答：内层 `j <= i`，无空格，全打星）

**图案 3：n 行倒三角形**
（学生回答：内层 `j <= n+1-i`，无空格，全打星）

**图案 4：n 行等腰三角形**
（学生回答：前置空格 `n-i` 个，星号 `2i-1` 个）

**图案 5：n 行空心正三角形**
（学生回答：内层 `j <= i`，边框（最后行/第一列/对角线）打星，其余空格）

（快速过一遍，每题不超过 1 分钟）

好，现在来一道综合性图案题——**打印菱形数字图案**。输入 n=4，输出：

```
   1
  121
 12321
1234321
 12321
  121
   1
```

这道题融合了菱形框架和数字输出。来，一起分析：

上半部分：（i 从 1 到 n）
- 前置空格：n-i 个
- 左半数字：1 到 i（递增）
- 右半数字：i-1 到 1（递减）

下半部分：（i 从 n-1 到 1）
- 同上半部分逻辑

```cpp
int n = 4;
// 上半部分
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n - i; j++) cout << " ";   // 空格
    for (int j = 1; j <= i; j++) cout << j;          // 左半递增
    for (int j = i - 1; j >= 1; j--) cout << j;      // 右半递减
    cout << endl;
}
// 下半部分
for (int i = n - 1; i >= 1; i--) {
    for (int j = 1; j <= n - i; j++) cout << " ";
    for (int j = 1; j <= i; j++) cout << j;
    for (int j = i - 1; j >= 1; j--) cout << j;
    cout << endl;
}
```

（在黑板上逐行标注每个循环的作用）

注意这里出现了"一行三个内层循环"的结构——空格、左半、右半——三个循环顺序执行，互不嵌套。

现在总结图案题的"万能思路"：

1. 先观察规律，写出第 i 行的表达式（几个空格？几个什么字符？）
2. 根据规律确定外层循环范围
3. 确定内层循环的边界（用 i 表达）
4. 如有空心需求，在内层加 if 判断

掌握了这个思路，任何图案题都不怕。

- 三个可调参数：次数、空格、星/空
- 菱形数字图案：空格 + 左半 + 右半
- 一行多内层 = 并列关系

- 找规律 → 翻译成循环条件
