## 📝 讲稿正文

取模运算符 `%` 大家从学 if-else 那章就开始用了——判断奇偶：`n % 2 == 0`。但 `%` 在数学上有一套完整的理论，叫**模运算**。它是数论的"基础设施"，几乎每个竞赛题都离不开它。

（在黑板上写）

模运算之所以重要，是因为它能**控制数值范围**。在竞赛中，答案往往大得离谱——比如 C(1000, 500) 是一个几百位的数，没法直接存。题目通常会要求"答案对 10⁹+7 取模"。这时候你就需要模运算的性质。

**性质一：加法保留模**

(a + b) % m = ((a % m) + (b % m)) % m

什么意思？你可以先把 a 和 b 分别取模，加起来再取一次模，结果和直接加起来取模一样。这就允许我们在计算的过程中"控制大小"。

**性质二：乘法保留模**

(a × b) % m = ((a % m) × (b % m)) % m

同样，可以先取模再乘再取模。

**性质三：减法保留模**

(a - b) % m = ((a % m) - (b % m) + m) % m

注意这个 `+ m`！因为 C++ 中 `-7 % 3` 的结果是 `-1`，而数学上我们期望得到 `2`。加上 m 再取模，就能把负数结果转正。

来看个实际例子：

```cpp
// 计算 (a * b) % mod，防止溢出
int modMul(int a, int b, int mod) {
    return (1LL * a * b) % mod;  // 1LL 是 long long 类型的 1，强制类型提升
}

// 计算 (a + b + c) % mod
int modAdd(int a, int b, int c, int mod) {
    return ((a % mod + b % mod) % mod + c % mod) % mod;
}
```

`1LL * a * b` 这个小技巧：乘上 1LL（long long 类型的 1）后，整个表达式自动提升为 long long，避免了 int 溢出。在竞赛代码里你会经常看到 `1LL *` 这个模式。

（停顿）

好，有了模运算的基础，我们引入一个重要的概念——**同余**。

如果 a 和 b 除以 m 的**余数相同**，就说 **a 与 b 模 m 同余**，记作：

> **a ≡ b (mod m)**

比如 17 ÷ 12 余 5，5 ÷ 12 也余 5，所以 17 ≡ 5 (mod 12)。

同余有两个关键性质：
- 如果 a ≡ b (mod m) 且 c ≡ d (mod m)，则 a + c ≡ b + d (mod m)
- 如果 a ≡ b (mod m) 且 c ≡ d (mod m)，则 a × c ≡ b × d (mod m)

翻译成人话：**同余关系在加法和乘法下是"保持"的**。做完加法和乘法再取模，不影响最终结果的模值。这是为什么"边算边取模"是合法的——你每一步取模，最终答案取模的结果不变。

（转身写板书的最后一块）

最后分享几个数论小技巧，实用又好玩。

**闰年判定**：一年是闰年，当且仅当 (能被 4 整除但不能被 100 整除) 或者 (能被 400 整除)。

```cpp
bool isLeap(int year) {
    return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
}
```

**3 的倍数判定**：一个数能被 3 整除，当且仅当它的各位数字之和能被 3 整除。比如 123 → 1+2+3=6，6%3==0 → 123 能被 3 整除。

```cpp
bool divisibleBy3(int n) {
    int sum = 0;
    while (n > 0) { sum += n % 10; n /= 10; }
    return sum % 3 == 0;
}
```

这个技巧有什么用？当数字大到存不进 int 时（比如 1000 位的数字），你可以把它存成字符串，逐位加起来判断——不需要大整数运算。

## 🖥️ 板书/屏幕要点

```
模运算三大性质：
  (a+b)%m = ((a%m)+(b%m))%m
  (a×b)%m = ((a%m)×(b%m))%m
  (a-b)%m = ((a%m)-(b%m)+m)%m  ← 注意 +m

同余：a ≡ b (mod m) → a 和 b 除以 m 余数相同
  同余保持加法和乘法

实用技巧：
  防溢出：1LL * a * b % mod
  闰年：(y%4==0 && y%100!=0) || y%400==0
  3的倍数：各位数字之和 %3 == 0
```

## 📌 小结

模运算是数论的"基础设施"——加法、乘法、减法三大性质让"边算边取模"成为可能。同余是模运算的语言——a ≡ b (mod m) 简洁地表达了"余数相等"。记住：负数取模要加 m 转正，大数乘法用 1LL 防溢出。到这儿，初等数论的基础就全部学完了。下一章我们学习两个数之间的关系——最大公约数和最小公倍数，以及两千年前的算法：辗转相除法。
