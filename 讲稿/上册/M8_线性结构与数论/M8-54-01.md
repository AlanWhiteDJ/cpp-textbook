## 📝 讲稿正文

同学们，从这节课开始，我们暂时告别栈、队列、链表这些"结构"，走进一个完全不同的世界——**数字本身**。你从小学就开始和数字打交道，加减乘除滚瓜烂熟。但今天我们要用程序的眼光重新审视数字——看看"整除"这件小事藏着多少门道。

（在黑板上写）

先说最基础的概念：**整除**。整数 a 除以非零整数 b，如果余数是 0，我们就说 **b 整除 a**，记作 **b | a**。注意这个竖线，和除号 `/` 不一样——`b | a` 表示的是一种关系：b 是 a 的约数。

这时候，b 叫 a 的**约数**（也叫因数、因子），a 叫 b 的**倍数**。

比如：3 | 12（3 整除 12），3 是 12 的约数，12 是 3 的倍数。  
5 | 12？不成立——12 ÷ 5 余 2，不是 0。

用程序判断整除，简单到只有一行：

```cpp
bool divides(int b, int a) {
    return a % b == 0;   // 余数为 0 就是整除
}
```

（停顿，换个语气）

约数有几个基本性质，大家想一想就能明白，不需要证明：

第一，**1 是任何正整数的约数**。任何数除以 1 都余 0。  
第二，**每个数本身也是自己的约数**。12 ÷ 12 = 1 余 0。  
第三，**约数不会比这个数本身大**（除了这个数本身和 1 之间的大小关系）。如果 b | a 且 b ≠ a，那么 |b| < |a|。  
第四，**传递性**：如果 b | a 且 c | b，那么 c | a。比如 3 | 6，6 | 24，那么 3 | 24。

（停下来，让学生记笔记）

好，现在我们看一个有趣的练习。如果我让你找出 12 的所有约数，你怎么找？

最笨的办法：从 1 到 12 一个个试——`if (12 % i == 0) 输出 i`。这没问题，但如果这个数是 10 亿呢？你得试 10 亿次——程序得跑好久。

回忆一下我们之前判断质数用过的技巧：约数是**成对出现**的。如果 d 是 n 的约数，那么 n/d 也是。比如 12 的约数：1 和 12、2 和 6、3 和 4。

每一对中，较小的那个一定不超过 √n——因为如果两个都大于 √n，乘起来就超过 n 了。所以枚举约数也只需要枚举到 √n：

```cpp
void printDivisors(int n) {
    for (int i = 1; i * i <= n; i++) {
        if (n % i == 0) {
            cout << i << " ";            // 较小的约数
            if (i != n / i) {
                cout << n / i << " ";    // 较大的约数（去重）
            }
        }
    }
}
```

注意那行 `if (i != n / i)`——这是为了避免完全平方数把中间那个数输出两遍。比如 36，6 × 6 = 36，如果不去重就会输出两个 6。

今天的内容比较基础，但它们是后续质数判定、质因数分解、GCD 的基石。**把基础打牢，后面的路才好走。**

## 🖥️ 板书/屏幕要点

```
整除：a % b == 0 → b | a（b 整除 a）
  约数（因数）：b 是 a 的约数
  倍数：a 是 b 的倍数

约数基本性质：
  ① 1 是任何正整数的约数
  ② n 本身是 n 的约数
  ③ 传递性：b|a 且 c|b ⇒ c|a

约数成对出现：
  d 和 n/d 是一对，较小的 ≤ √n
  枚举到 √n 即可，注意完全平方数去重
```

## ✏️ 随堂练习

1. 输入一个正整数 n，输出它的所有约数（不要求排序），用 √n 优化。
2. 输入 n，判断它是不是完全平方数（约数个数为奇数）。
3. 统计 1~100 中，拥有最多约数的数是哪个（提示：从 1 到 100 逐个枚举约数计数，最后找出最大的）。

## 📌 小结

今天我们回到数字的基础——整除。b | a 表示 b 整除 a，此时 b 是 a 的约数。约数成对出现，枚举只需要到 √n。这个"成对出现"的技巧在质数判定和质因数分解中还会大量使用。下节课我们进入数论的核心——质数的判定。
