## 📝 讲稿正文

前面两节课我们用的都是 STL 里的 `stack`——现成的、开箱即用。但有同学会问：如果我不想用 STL，我自己怎么实现一个栈？

答案是：**数组加一个 "栈顶指针"**。非常简单，三行代码就能搞定核心操作。

（在黑板上画图）

想象一个数组 `a[100]`，再搞一个变量 `top` 表示"栈顶位置"——注意这里 top 不是函数名，是我们自己定义的整型变量。初始化时 `top = -1`，表示栈为空。

压栈 push(x)：`top` 先加一，再把 x 放进 `a[top]`。  
弹栈 pop：`top` 减一就行了——元素还在数组里，但我们已经"看不到"它了。  
查看栈顶：`return a[top]`。  
判空：`return top == -1`。

```cpp
const int MAXN = 1000;
int a[MAXN];     // 用数组模拟栈空间
int top = -1;    // 栈顶指针，-1 表示空

void push(int x) {
    top++;
    a[top] = x;      // 等价于 a[++top] = x;
}

void pop() {
    top--;           // 栈顶下移，原数据"逻辑删除"
}

int getTop() {
    return a[top];   // 返回栈顶元素
}

bool empty() {
    return top == -1;
}
```

看，就这么简单。`top` 这个变量就是灵魂——它永远指向栈顶元素的位置。push 的时候它往上走，pop 的时候它往下走。

这种"数组模拟"的思路在竞赛中极其常用——不只是栈，后面我们学链表、学队列，都可以用数组模拟。速度快、不容易出错。

好，现在用我们刚学的数组栈，来做个实用的小程序：**十进制转二进制**。

大家学过怎么手算十进制转二进制——不断地除以 2，把余数记下来，最后从下往上读。比如 13：

13 ÷ 2 = 6……余 1  
6 ÷ 2 = 3……余 0  
3 ÷ 2 = 1……余 1  
1 ÷ 2 = 0……余 1  

从下往上读余数：1101。验证一下：1×8 + 1×4 + 0×2 + 1×1 = 13，正确！

注意这里有一个关键动作：**从下往上读**。也就是说，先算出来的余数后输出，后算出来的余数先输出——这正是 LIFO！用栈太合适了。

每次的余数 push 进去，最后依次 pop 出来：

```cpp
#include <iostream>
using namespace std;

const int MAXN = 100;
int a[MAXN], top = -1;

void push(int x) { a[++top] = x; }
void pop() { top--; }
int getTop() { return a[top]; }
bool empty() { return top == -1; }

int main() {
    int n;
    cin >> n;

    if (n == 0) {
        cout << 0 << endl;
        return 0;
    }

    while (n > 0) {
        push(n % 2);   // 余数入栈
        n /= 2;        // n 变小
    }

    while (!empty()) {
        cout << getTop();  // 逆序输出
        pop();
    }
    cout << endl;
    return 0;
}
```

输入 13，push 的顺序是：1 → 0 → 1 → 1（最后的 1 在栈顶）。  
pop 输出的顺序是：1 → 1 → 0 → 1，恰好就是 1101！

（停顿，让学生尝试）

如果把 `n % 2` 改成 `n % 8`，这就是十进制转八进制。改成 `n % 16` 就是转十六进制——当然十六进制要处理 10~15 变成 A~F 的问题，加个 if 判断就行。

最后给大家留个思考题：除了进制转换，栈的"回溯"特性还能做什么？你玩过走迷宫吗——走到死胡同要退回去，退回的动作就是 pop，之前的每个岔路口就是 push。这个思路以后学 DFS（深度优先搜索）时会再次遇到。

## 🖥️ 板书/屏幕要点

```
数组模拟栈：
  a[MAXN] + int top = -1
  push: a[++top] = x;
  pop:  top--;
  top():a[top];
  empty:top == -1

十进制 → 二进制（栈版）：
  while(n>0): push(n%2), n/=2;
  while(!empty): 输出 top(), pop();

本质：利用 LIFO 实现逆序输出
```

## 📌 小结

本课时我们学到了两个重要内容：一是用数组加栈顶指针模拟栈——这是竞赛中的基本功，简单高效；二是利用栈的 LIFO 特性实现进制转换——把"从下往上读"变成了天然的 pop 顺序。这两者背后都是同一个思想：**后进先出 = 逆序处理**。下一章我们转向栈的兄弟——队列，看看 FIFO 的世界有什么不同的风景。
