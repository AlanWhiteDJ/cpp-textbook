# 第55章 第2课时：最小公倍数与多个数的 GCD/LCM
**时长**：15-20分钟 | **对应教材**：第55章

## 🎯 本课时目标
- 理解最小公倍数（LCM）的概念
- 掌握 LCM = a / GCD × b 公式（先除后乘）
- 会求多个数的 GCD 和 LCM
- 掌握约分、通分等实际应用

## 📝 讲稿正文

上节课我们学会了用辗转相除法高效求 GCD。这节课我们来看 GCD 的"搭档"——**最小公倍数（LCM）**，以及怎么处理三个或更多个数的情况。

（在黑板上写）

LCM 的定义很简单：能同时被 a 和 b 整除的**最小正整数**。记作 **lcm(a, b)**。

比如 LCM(12, 18)：12 的倍数有 12, 24, **36**, 48……18 的倍数有 18, **36**, 54……最小的公共倍数是 36。所以 LCM(12, 18) = 36。

那怎么算 LCM 呢？有一个关键公式：

> **LCM(a, b) = a / GCD(a, b) × b**

数学上也等价于 `a × b / GCD(a, b)`，但注意——**在代码里必须"先除后乘"！**

为什么呢？如果 a 和 b 都很大（比如都是 10⁹），a × b = 10¹⁸，这个结果已经溢出 int 了，就算你用 long long，如果 a 和 b 是 10¹⁸ 级别连 long long 也溢出。但是 a / GCD 肯定不会溢出（因为除数 ≥ 1），然后再乘 b 也安全得多。

```cpp
int lcm(int a, int b) {
    return a / gcd(a, b) * b;   // 先除后乘！
}
```

来验证：LCM(12, 18) = 12 / 6 × 18 = 2 × 18 = 36。✓

（停顿，让学生记笔记）

好，单个 GCD 和 LCM 学会了。现在看**多个数**的情况。求三个数 a、b、c 的最大公约数：

```cpp
int result = gcd(gcd(a, b), c);
```

LCM 同理：

```cpp
int result = lcm(lcm(a, b), c);
```

这个性质成立是因为 GCD 和 LCM 都有**结合律**——先算哪两个，结果都一样。比如 gcd(gcd(12, 18), 30) = gcd(6, 30) = 6；gcd(12, gcd(18, 30)) = gcd(12, 6) = 6。完全一致。

多个数的 LCM 有个有趣的应用：求 1 到 N 所有数的最小公倍数。比如 LCM(1,2,3,4,5) = 60。这个数有个特点——它能被 1 到 5 中任意一个数整除。这在处理"周期问题"时非常有用：如果小明每 3 天去一次图书馆，小红每 4 天去一次，他们多少天后再次相遇？LCM(3,4)=12 天后。

（转身写板书的另一块）

好，现在我们把 GCD 和 LCM 串起来做个综合应用：**约分**。

给你一个分数 24/36，怎么把它化成最简分数？分子分母同时除以它们的 GCD 就行：

```cpp
int a = 24, b = 36;
int g = gcd(a, b);   // g = 12
cout << a/g << "/" << b/g << endl;   // 输出 2/3
```

再进一步：两个分数相加怎么约分？比如 1/6 + 1/4。先通分（分母取 LCM），再相加，再约分（分子分母除以 GCD）：

```cpp
// a/b + c/d = (a*d + c*b) / (b*d)，然后约分
int numerator = a * d + c * b;
int denominator = b * d;
int g = gcd(numerator, denominator);
cout << numerator/g << "/" << denominator/g << endl;
```

你看，学完了 GCD 和 LCM，分数运算一下子变得很清爽。

最后再强调一遍：**LCM 一定要先除后乘**。`a * b / gcd` 在 a=50000, b=50000 时已经溢出 int（2.5×10⁹），而 `a / gcd * b` 完全安全。这个细节竞赛中反复考，大家刻在脑子里。

## 🖥️ 板书/屏幕要点

```
LCM(a,b) = a / GCD(a,b) × b  ← 先除后乘！

多个数：
  GCD(a,b,c) = gcd(gcd(a,b), c)
  LCM(a,b,c) = lcm(lcm(a,b), c)

实际应用：
  约分：分子分母 ÷ GCD
  通分：分母取 LCM
  周期问题：相遇时间 = LCM(周期1, 周期2)

⚠️ LCM 先乘后除可能溢出！
```

## 📌 小结

LCM 和 GCD 是一对密不可分的搭档——通过公式 LCM = a/GCD×b 关联。关键细节：先除后乘防溢出。多个数的 GCD/LCM 利用结合律依次计算。约分、通分、周期问题都是 GCD/LCM 的实际应用场景。下节课我们进入更深的领域——扩展欧几里得算法和裴蜀定理，看看 ax+by=c 这样的一次方程怎么解。
