## 📝 讲稿正文

上几节课我们一直在和栈打交道——后进先出，叠盘子。今天我们要认识栈的兄弟——**队列（queue）**。它的规则正好相反。

（做排队手势）

中午下课铃一响，你冲向食堂排队。谁排在前面，谁先打到饭；排在后面的，只能先等着。不会有人从队伍中间插进去——至少文明人不会。

这就是队列的规则：**先进先出**。英文叫 FIFO——First In, First Out。最早进去的，最早出来。

在 C++ 里，队列同样有现成的，在 `<queue>` 头文件里：

```cpp
#include <iostream>
#include <queue>
using namespace std;

int main() {
    queue<int> q;         // 声明一个存整数的队列

    q.push(10);           // 入队：从队尾进去
    q.push(20);           // 20 排在 10 后面
    q.push(30);           // 30 排在最后面

    // 此时队列：队首 10 → 20 → 30 队尾

    cout << q.front() << endl;  // 看队首 → 10
    cout << q.back() << endl;   // 看队尾 → 30
    cout << q.size() << endl;   // 元素个数 → 3

    q.pop();               // 出队：队首的 10 走了
    cout << q.front() << endl;  // 新队首 → 20

    cout << q.empty() << endl;  // 空吗？→ 0（不空）

    return 0;
}
```

我们来对比一下队列和栈的操作名——很容易搞混，大家注意区分：

队列用 **front()** 看队首，用 **back()** 看队尾。栈用 **top()** 看栈顶——因为栈只有一个口，没有"首尾"之分。

队列的 six operations：push（入队尾）、pop（出队首）、front、back、empty、size。

和栈一样，queue 的 **pop() 也没有返回值**。你要先 `front()` 取值再 `pop()`——这个坑在栈和队列里一模一样。

（停顿）

好，现在我们来模拟一个真实的排队场景：

```cpp
queue<string> line;

line.push("小明");
line.push("小红");
line.push("小刚");

while (!line.empty()) {
    cout << line.front() << " 打完饭了！" << endl;
    line.pop();
}
```

这段代码的输出是：
```
小明 打完饭了！
小红 打完饭了！
小刚 打完饭了！
```

先排队的先打完——严格的先来后到。如果换成栈，顺序就反了：最后排的小刚最先打完，这不合理对不对？

（停顿，让学生思考）

所以，栈和队列虽然操作类似，但**顺序逻辑完全不同**。选错容器，程序逻辑会完全错掉。记住一个简单的判断：你需要的顺序是"后来的先用"（撤销、回溯）→ 栈；"先来的先用"（排队、轮流）→ 队列。

现在请大家上机试试：创建一个 queue，依次 push 1 到 5，然后依次弹出并输出。对比上节课栈的输出（5 4 3 2 1），这次应该是 1 2 3 4 5——完全相反。

## 🖥️ 板书/屏幕要点

```
队列 Queue：先进先出 FIFO

操作速记：
  push(x) → 入队尾
  pop()   → 出队首（无返回值！）
  front() → 查看队首
  back()  → 查看队尾
  size()  → 元素个数
  empty() → 是否为空

包含头文件：#include <queue>
声明：queue<int> q;

Stack vs Queue：
  Stack: top() 弹出栈顶       LIFO
  Queue: front()/back() 弹出队首 FIFO
```

## ✏️ 随堂练习

1. 写程序：依次向 queue 中 push 1~5，然后 while 循环逐个 front + pop 输出，观察顺序。
2. 修改以上程序：每次 pop 前输出当前队首和队尾，观察队列状态变化。
3. 模拟银行排队：5 个客户依次进入队列，每叫一个号（pop），输出"请 X 号客户到窗口办理"，X 用 front() 取。

## 📌 小结

今天我们认识了队列——FIFO 的"排队"模型。六个核心操作要记牢，特别注意用 front()/back() 而非 top()，以及 pop() 同样没有返回值。队列和栈是一对镜像兄弟——一个正序、一个逆序。下节课我们看看队列的高级玩法：循环队列和 BFS 骨架。
