## 📝 讲稿正文

上节课我们学会了判断一个数是不是质数。这节课更进一步：**把一个合数拆成质数的乘积**——这就是质因数分解。

（在黑板上写）

比如 60 = 2 × 2 × 3 × 5，也可以写成 60 = 2² × 3 × 5。100 = 2² × 5²。任何大于 1 的整数，都可以**唯一地**分解成质数的乘积——这叫做**算术基本定理**，是数论的基石。

怎么做质因数分解？思路和判断质数类似——从小到大试除。遇到能整除的因数，就一直除到不能整除为止：

```cpp
void factorize(int n) {
    for (int i = 2; i * i <= n; i++) {
        while (n % i == 0) {   // 能除尽就一直除
            cout << i << " ";   // i 是一个质因子
            n /= i;
        }
    }
    if (n > 1) cout << n;  // 最后剩下的也是一个质因子
}
```

我们走一遍 n = 60：

i = 2：60 % 2 == 0 → 输出 2，n = 30；30 % 2 == 0 → 输出 2，n = 15；15 % 2 ≠ 0，退出 while。  
i = 3：15 % 3 == 0 → 输出 3，n = 5；5 % 3 ≠ 0。  
i = 4：4 × 4 = 16 > n=5，循环结束。  
最后 n = 5 > 1 → 输出 5。

输出：2 2 3 5。完美！

（停顿）

有个同学可能想问：i = 4 的时候，4 是合数——为什么没有输出 4？因为 4 的质因子 2 在前面已经被除干净了，等到 i = 4 的时候，n 已经不被 4 整除。所以循环里输出的 i 天然都是质数——非常巧妙。

最后那行 `if (n > 1) cout << n;` 非常关键。循环结束后，n 可能剩下一个大于 √n 的质因子（比如输入是 97，循环什么也没输出，直接输出 97）。很多人都漏掉这行，导致少输出一个质因子——这是个高频 bug。

现在，有了质因数分解，我们可以轻松算出一个数的**约数个数**。

如果 n = p₁^a₁ × p₂^a₂ × … × pₖ^aₖ（每个 p 是不相同的质数），那么 n 的约数个数 = (a₁+1) × (a₂+1) × … × (aₖ+1)。

为什么？因为每个质因子可以选 0 次、1 次……到 a 次，总共 (a+1) 种选择。乘法原理——各选各的，乘起来就是总数。

比如 60 = 2² × 3¹ × 5¹，约数个数 = (2+1)×(1+1)×(1+1) = 3×2×2 = 12 个。验证：1,2,3,4,5,6,10,12,15,20,30,60——确实是 12 个。

```cpp
int countDivisors(int n) {
    int cnt = 1;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            int exp = 0;          // 指数
            while (n % i == 0) {
                n /= i;
                exp++;
            }
            cnt *= (exp + 1);     // 乘上 (指数+1)
        }
    }
    if (n > 1) cnt *= 2;          // 最后一个质因子的指数是 1
    return cnt;
}
```

好，这是算个数。如果要**枚举所有约数并排序输出**呢？我们前面学过的"成对出现"技巧——枚举到 √n，把 i 和 n/i 都收集起来，最后排序：

```cpp
vector<int> getDivisors(int n) {
    vector<int> divs;
    for (int i = 1; i * i <= n; i++) {
        if (n % i == 0) {
            divs.push_back(i);
            if (i != n / i) divs.push_back(n / i);
        }
    }
    sort(divs.begin(), divs.end());
    return divs;
}
```

这里用了 vector——前面学的动态数组，刚好派上用场。先 push_back 收集，再 sort 排序，最后返回。简洁明了。

```
算术基本定理：任何 >1 整数可唯一分解为质数乘积

质因数分解：
  for i=2 to √n:
    while(n%i==0): 输出i, n/=i
  if(n>1): 输出n    ← 别漏！

约数个数 = ∏(exp+1)
  60=2²×3×5 → (2+1)(1+1)(1+1)=12

枚举全部约数：成对出现 + vector + sort
```

## 📌 小结

质因数分解是数论的核心操作——从小到大试除，能除尽就一直除，最后别漏掉大于 √n 的因子。利用分解结果可以算出约数个数和枚举所有约数。这些技巧看似简单，但它们是后续学习 GCD、LCM 和模运算的基础。下节课我们进入模运算的世界——看看 `%` 这个老朋友还有哪些你不知道的本领。
