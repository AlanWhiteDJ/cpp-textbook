# 第3章 第3课时：命名规则与赋值——"="不是"等于"

**时长**：15-20分钟 | **对应教材**：第3章 · 3.3-3.4节

## 🎯 本课时目标
- 学生能说出 C++ 变量命名的硬性规定
- 学生能区分合法和非法的变量名
- 学生能正确理解赋值号 `=` 的含义——"把右边的值放进左边"

## 📝 讲稿正文

同学们，前面两节课我们学会了声明变量、初始化变量。今天我们要聊两个话题：第一，变量怎么取名字；第二，那个等号 `=` 到底是什么意思。

先说取名。给变量取名，C++ 有硬性规定，也有约定俗成的习惯。规定是铁律——违反了编译器直接报错；习惯是修养——不违反也能跑，但别人看了会皱眉。

**硬性规定三条：**

第一条，变量名只能由**字母、数字、下划线**这三种字符组成。注意，是英文的字母，英文的数字，英文的下划线。不能有空格，不能有减号，不能有中文。

第二条，变量名的**第一个字符不能是数字**。必须以字母或下划线开头。

第三条，**不能用 C++ 的关键字**当变量名。关键字就是 C++ 自己已经占用了的单词——比如 `int`、`return`、`if`、`while`。你不能声明一个叫 `int` 的变量，因为 `int` 已经被 C++ 用来表示整数类型了。

我们来做一个快速判断。我念几个变量名，你们告诉我合法还是非法，为什么——

（互动环节）

`score` ——合法，全字母。
`my_score` ——合法，字母加下划线。
`player2` ——合法，字母开头，含数字。
`2player` ——**非法**，数字不能开头！
`my-score` ——**非法**，减号不行，会跟运算符混淆。
`int` ——**非法**，这是关键字。
`我的分数` ——不推荐，虽然某些编译器能过，但这不是好习惯。用英文。

好了，硬性规定就这些。下面是**习惯——不遵守也不报错，但强烈建议。**

第一条，**变量名要有意义。** `score` 比 `s` 好一万倍，`studentCount` 比 `x` 好一万倍。一个好变量名，不用看上下文就知道它是干什么的。

第二条，**用英文，哪怕英文不好。** 用 `length` 不要用 `changdu`，用 `width` 不要用 `kuan`。为什么呢？因为全世界的程序员都用英文给变量取名——这是行业通用语言。你写拼音，换一个不认识中文的人就完全看不懂了。

第三条，**多个单词用下划线连接或驼峰命名，任选一种，保持一致。** 下划线风格：`max_score`、`student_count`。驼峰风格：`maxScore`、`studentCount`。两种都可以，但别在一个程序里混着用——一会儿 `max_score` 一会儿 `maxScore`，会把自己搞晕。

---

好，取名讲完了，我们来聊一个更重要、更容易搞错的问题——**那个等号 `=` 到底是什么意思？**

在数学课上，`=` 表示"等于"，左边等于右边。比如 `x = 5`，意思是 x 等于 5。

但在 C++ 里，`=` 的含义完全不同——它是**赋值号**，意思是"**把右边的东西放进左边的变量**"。

这个区别极其重要。我举一个在数学课上完全不可能、但在 C++ 里天天出现的例子：

```cpp
int x;
x = 10;        // 把 10 放进 x
x = 20;        // 把 20 放进 x（原来的 10 被覆盖了）
x = x + 5;     // 把 x 当前的值（20）加上 5，结果（25）存回 x
```

大家看最后一行——`x = x + 5`。如果按数学课的理解，这行代码是矛盾的：x 怎么可能等于 x+5？但在 C++ 里，这句话的意思是：**先把右边算出来——x 现在是 20，20+5=25，然后把 25 存回 x。** 执行完这行，x 从 20 变成了 25。

C++ 的赋值逻辑永远是：**先算右边的表达式，再把结果赋给左边的变量。** 右边算完之前，左边不变。

这个"="，是你从数学思维切换到编程思维的第一座桥。以后你会经常写 `x = x + 1`、`sum = sum + n` 这样的代码——让一个变量更新自己。这在数学里是不可能的，在编程里是家常便饭。

好，今天的内容有点抽象，但极其重要。下节课我们来学 `const`——一种"不能改的变量"。

## 🖥️ 板书/屏幕要点
```
命名硬性规定（违反报错）：
  ① 只能字母、数字、下划线
  ② 数字不能开头
  ③ 不能用关键字（int、return、if...）

命名习惯（不报错但推荐）：
  ① 有意义：score > s
  ② 用英文：length > changdu
  ③ 多词连接：max_score 或 maxScore（二选一）

赋值号 = 的真正含义：
  C++: = 是"赋值"，不是"等于"
      把右边的值放进左边的变量

  int x;
  x = 10;      // x ← 10
  x = x + 5;   // x ← (x+5)  先算右边，再存左边
```

## 📌 小结（30秒总结）
变量名三条规定：字母数字下划线、数字不开头、不用关键字。好变量名要有意义、用英文、风格一致。赋值号 `=` 不是数学的"等于"，而是"把右边算出来放进左边"——这是你从数学思维转入编程思维的第一道关。
