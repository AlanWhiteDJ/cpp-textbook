# 第7章 第1课时：打开数学工具箱——绝对值与平方根

**时长**：15-20分钟 | **对应教材**：第7章 · 7.1-7.3节

## 🎯 本课时目标
- 学生知道 `#include <cmath>` 是数学工具箱的入口
- 学生能用 `abs` 求整数绝对值，`fabs` 求浮点绝对值
- 学生能用 `sqrt` 求平方根

## 📝 讲稿正文

同学们，前六章我们学会了加减乘除、取模、自增自减——这些都是 C++ 的基本运算。但从今天开始，我们要打开一个更高阶的工具箱。

你们数学课上已经学过了：绝对值、平方根、幂运算、三角函数……如果每一个都要你自己从头写代码实现，那编程就没法玩了。好消息是——**C++ 早就帮我们把这些数学函数写好了，你只需要知道怎么调用它们。**

这个数学工具箱的名字叫 **`<cmath>`**——C Math，C 语言数学库。它就像一把瑞士军刀：绝对值、平方根、幂运算、取整、对数、三角函数……几十种功能折叠在一个工具箱里。

用之前只需要写一行：

```cpp
#include <cmath>   // 打开数学工具箱
```

这跟你之前写 `#include <iostream>` 的道理一样——数学函数不放在 `<iostream>` 里，它们在 `<cmath>` 这个专门的箱子里。不写这行，所有数学函数都会报"未定义"错误。

---

好，工具箱打开了，我们来一件一件拿出里面的工具。

**第一件：绝对值——`abs` 和 `fabs`。**

绝对值就是把负号去掉——-5 的绝对值是 5，3.14 的绝对值还是 3.14。

整数的绝对值用 `abs`：

```cpp
#include <cmath>
#include <iostream>
using namespace std;

int main() {
    int a = -100, b = 42;
    cout << abs(a) << endl;        // 100
    cout << abs(b) << endl;        // 42
    cout << abs(a - b) << endl;    // |-100-42| = 142
    return 0;
}
```

小数的绝对值用 `fabs`（f 表示 float/浮点）：

```cpp
double x = -3.14;
cout << fabs(x) << endl;   // 3.14
```

注意：如果你对小数用 `abs`，C++ 会先把小数截断成整数，再求绝对值——结果就错了。所以记住：**整数用 `abs`，小数用 `fabs`。** 竞赛里整数用得多，`abs` 是你最常用的。

---

**第二件：平方根——`sqrt`。**

sqrt = square root，平方根。比如 √25 = 5，√2 ≈ 1.414。

```cpp
cout << sqrt(25) << endl;   // 5
cout << sqrt(2) << endl;    // 1.41421
cout << sqrt(0) << endl;    // 0
```

`sqrt` 的参数和返回值都是 `double` 类型。你传一个整数进去，C++ 会自动帮你转换：

```cpp
int len = 9;
double side = sqrt(len);   // side = 3.0
```

有一点必须注意：**不能对负数求平方根！** 在实数范围内，√(-1) 没有意义。`sqrt(-1)` 在 C++ 里会返回一个叫 `nan`（Not A Number）的特殊值——程序不报错但结果是错的。所以写 `sqrt` 之前，确保参数是非负的。

绝对值加平方根还有一个经典组合——勾股定理：

```cpp
double a = 3, b = 4;
double c = sqrt(a * a + b * b);   // c = 5
cout << "斜边长：" << c << endl;
```

直角边 3 和 4，斜边就是 5——这个经典公式，用 `sqrt` 一行就搞定。

好了，今天打开了数学工具箱，学会了两样工具：`abs`/`fabs` 和 `sqrt`。把 `<cmath>` 写在文件头，下节课我们继续翻箱底——`pow` 幂运算和 `max`/`min`。

## 🖥️ 板书/屏幕要点
```
#include <cmath>   ← 数学工具箱，必须写！

abs(x)   — 整数绝对值
  abs(-100) = 100
  abs(a - b) = a 和 b 的距离

fabs(x)  — 浮点绝对值
  fabs(-3.14) = 3.14
  ⚠ 整数→abs，小数→fabs

sqrt(x)  — 平方根 √x
  sqrt(25) = 5
  sqrt(2)  ≈ 1.41421
  ⚠ 参数不能是负数！sqrt(-1) → nan

经典组合：
  斜边长 = sqrt(a*a + b*b)   （勾股定理）
```

## ✏️ 随堂练习
写程序计算 √(3² + 4²)——用 `sqrt` 和 `pow` 或直接乘法实现。再算 √100 + | -9 | 的结果。最后测试 `sqrt(-1)`，看看输出什么，记录在笔记上。如果输出 nan，查一下 nan 的全称是什么意思。

## 📌 小结（30秒总结）
数学工具箱 `<cmath>` 是你编程的瑞士军刀。绝对值用 `abs`（整数）和 `fabs`（小数），平方根用 `sqrt`（参数不能为负）。勾股定理 `sqrt(a*a + b*b)` 一行搞定。记住打开工具箱要写 `#include <cmath>`，否则编译器不认识这些函数。
