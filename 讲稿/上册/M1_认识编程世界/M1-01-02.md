# 第1章 第2课时：从硬件到软件——计算机的"身体"和"灵魂"

**时长**：15-20分钟 | **对应教材**：第1章 · 1.1-1.2节

## 🎯 本课时目标
- 学生能清晰区分硬件和软件，并各举三个例子
- 学生能用自己的话解释"程序是什么"
- 学生能理解"内存 = 工作台，外存 = 仓库"这个比喻

## 📝 讲稿正文

上节课我们讲了计算机的五个组成部分——输入设备、输出设备、运算器、控制器、存储器。大家还记得吗？（快速互动回顾）

好。那今天我要问大家一个问题——**计算机只有硬件就能工作吗？**

（让学生思考）

我们打个比方。你买了一部全新的手机，拆开包装，摸得到屏幕、电池、摄像头，这些都是**硬件**——看得见摸得着的物理部件。

但是你按开机键，屏幕亮了，出现了桌面、图标。桌面和图标你摸得到吗？摸不到。它们不是螺丝拧上去的，而是**软件**——程序和数据，存在存储器里的"灵魂"。

大家想一想：**硬件和软件的关系，就像身体和灵魂。** 身体是硬件，看得见；思想是软件，看不见但指挥身体做事。没有软件，计算机就是一堆废铁——有电、有零件，但什么都干不了。

好，讲到"软件"，我就要引出今天的一个核心概念——**程序**。

"程序"这个词听起来很高级对不对？有人觉得程序是很神秘的东西，是黑客戴着兜帽在暗室里敲出来的。其实，**程序就是给计算机的指令清单，跟写菜谱、写组装说明书没有本质区别。**

我举个生活中的例子。你去奶茶店点单："一杯珍珠奶茶，去冰，三分糖。"这就是你给店员的一份"指令清单"。店员拿到这份清单，按照你的要求一步一步做出来。

程序也一样。你给计算机一份指令清单，计算机按清单一条一条执行。但计算机比奶茶店员死板一万倍——你对店员说"一杯珍珠奶茶"，他能理解你要的是奶茶不是咖啡。但计算机不行，你必须每一条都说得清清楚楚，不能有任何模糊。

比如，你要写一个"算加法"的程序，得这样说：

第一步：告诉计算机，准备接收第一个数字。第二步：告诉计算机，准备接收第二个数字。第三步：把两个数字加起来。第四步：把结果给我看。

这就是一个程序。只是四条指令而已。

现在我再帮大家巩固一下上节课讲的**内存和外存**的区别。这个特别重要，因为很多初学者在这里搞混。

我问你们——你们电脑的 C 盘、D 盘是内存吗？

（让学生回答）

**不是！** C 盘 D 盘是硬盘，属于**外存**。内存是另一回事。我给大家一个终身管用的比喻：**内存是你的工作台，外存是你的仓库。**

你写作业的时候，从书包（仓库）里把课本拿出来，摊在桌面（工作台）上。写完了，把作业本放回书包。桌面上的东西，妈妈打扫卫生就收走了——内存就这样，关机就消失。但书包里的东西一直在——外存就这样，永久保存。

打开文件，就是从仓库搬到工作台。保存文件，就是从工作台搬回仓库。就这么简单。

好了，今天我们搞清楚了硬件和软件的区别、程序是什么、内存和外存的区别。下节课我们要进入一个更关键的问题——人怎么跟计算机"说话"？为什么不能直接用中文下命令？

## 💬 互动提问
1. "你们家的微波炉里面有程序吗？它接收什么输入，做什么处理，输出什么？"
2. "如果计算机没有软件，只有硬件，它能做什么？"
3. "打开一个 Word 文档，编辑它，然后保存——这个过程中，数据什么时候在内存，什么时候在外存？"

## 🖥️ 板书/屏幕要点
```
硬件 = 摸得着的（屏幕、键盘、CPU、内存条、硬盘）
软件 = 摸不着的（操作系统、浏览器、游戏）

程序 = 给计算机的指令清单
      就像菜谱、组装说明书

内存 vs 外存：
  内存 = 工作台（快、关机消失）
  外存 = 仓库（慢、永久保存）

  打开文件 = 仓库 → 工作台
  保存文件 = 工作台 → 仓库
```

## ✏️ 随堂练习
在纸上列出你身边至少 3 件硬件和 3 个软件。对于每一件硬件，指出它属于计算机五部分中的哪一部分。对于每一个软件，用一句话说明它是干什么的。

## 📌 小结（30秒总结）
硬件是身体，软件是灵魂，两者缺一不可。程序就是给计算机的指令清单，跟菜谱一样，必须精确到每一步。内存是工作台——快但断电就消失；外存是仓库——慢但永久保存。这是计算机最基础也最重要的概念，一辈子都要记住。
