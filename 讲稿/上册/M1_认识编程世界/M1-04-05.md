## 📝 讲稿正文

同学们，今天是第四章的最后一节课。我们学过了六种类型——int、long long、double、char、string、bool。今天做三件事：第一，用量尺量一量每种类型占多少空间；第二，做一个"选择指南"帮你在不同场景选对类型；第三，全章复习。

先说量空间。C++ 提供了一个内置的"量尺"——`sizeof`。你告诉它一个类型名，它告诉你这个类型占几个字节。

```cpp
cout << "int: " << sizeof(int) << " 字节" << endl;             // 一般是 4
cout << "long long: " << sizeof(long long) << " 字节" << endl; // 一般是 8
cout << "double: " << sizeof(double) << " 字节" << endl;       // 一般是 8
cout << "char: " << sizeof(char) << " 字节" << endl;           // 1
cout << "bool: " << sizeof(bool) << " 字节" << endl;           // 1
```

在你的电脑上实际运行一下——大部分电脑结果都一样：int=4，long long=8，double=8，char=1，bool=1。string 用 `sizeof` 测不太准确，因为它内部结构复杂，我们现在知道它能装很多字符就够了。

`sizeof` 不光能量类型，还能直接量变量：

```cpp
int x = 100;
cout << sizeof(x) << endl;  // 输出 4
```

---

好，现在来做今天的重头戏——**类型选择指南**。

以后你每声明一个变量，脑子里都应该有一瞬间的判断：我用什么类型？

我来帮你们建一个"条件反射"：

**场景一：计数、人数、年龄、小范围的数字。** → 用 `int`。比如 `int count = 42;`、`int age = 13;`。这是你用的最多的类型。

**场景二：可能很大的数——乘积、总数、大数据量。** → 用 `long long`。比如 `long long total = 0;`。竞赛中很常见，因为数据量动不动就上百万上千万。

**场景三：带小数的——面积、平均分、百分比。** → 用 `double`。比如 `double avg = 91.5;`、`double pi = 3.14159;`。

**场景四：单个字符——成绩等级、符号、字母标记。** → 用 `char`，单引号。比如 `char grade = 'A';`。

**场景五：名字、一句话、任意文字。** → 用 `string`，双引号，别忘了 `#include <string>`。

**场景六：是非判断——真或假、对或错。** → 用 `bool`。比如 `bool ok = true;`。

这六条判断规则，你们最好抄在笔记本的第一页，以后每次拿不准就翻回来看。

---

最后，我们来串第四章全章内容。

第四章的标题叫"程序的六种容器"。为什么是容器？因为每种类型都是内存里的一个盒子，不同大小的盒子装不同范围的数据。

数字组的三个兄弟：int 是主力（4 字节，±21 亿），long long 是大胃王（8 字节，大到离谱），double 是精确小数（8 字节，15 位精度）。数字溢出不会报错，但结果全错——这是最隐蔽的 bug。

文字组的两个：char 是单人间（1 字节，单引号，一个字符），string 是大家庭（双引号，任意长度，需要额外头文件）。单引号是 char，双引号是 string——搞混了编译器会发火。

判断组的独苗：bool 只有两个值——true 和 false——但它是所有逻辑判断的基石。

## 🖥️ 板书/屏幕要点
```
sizeof — 量类型的"尺子"
  sizeof(int)       → 4（字节）
  sizeof(long long) → 8
  sizeof(double)    → 8
  sizeof(char)      → 1
  sizeof(bool)      → 1

类型选择指南：
  计数/人数/年龄      → int
  大数/乘积/大数据    → long long
  小数/平均分/面积    → double
  单个字符/等级/符号  → char（单引号）
  名字/一句话/文字    → string（双引号 + #include <string>）
  是/否 对/错        → bool

六型总表：
  数字：int | long long | double
  文字：char（单引号）| string（双引号）
  判断：bool（true/false）
```

## 📌 小结（30秒总结）
六种类型三种用途——int/long long/double 管数字，char/string 管文字，bool 管判断。sizeof 是量尺，帮你确认每种类型的大小。选类型的黄金法则：不确定会不会溢出就用 long long，需要小数用 double，记住 char 单引号 string 双引号。下一章，让程序跟你聊天——cin 和 cout 的交互世界。
