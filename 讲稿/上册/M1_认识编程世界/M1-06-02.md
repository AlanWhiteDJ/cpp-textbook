# 第6章 第2课时：取模运算 % ——被低估的"编程利器"

**时长**：15-20分钟 | **对应教材**：第6章 · 6.3节

## 🎯 本课时目标
- 学生能理解取模运算的含义和用法
- 学生知道 `%` 只能用于整数
- 学生能用 `%` 实现判断奇偶、提取个位、循环等问题

## 📝 讲稿正文

同学们，上节课我们学了加减乘除。今天我们来学第五种算术运算符——**取模 `%`**。

取模是什么？就是"求余数"。你小学就学过：17 ÷ 5 = 3 ……余 2。在 C++ 里，`17 % 5` 的结果就是 2。就这么简单。

```
17 ÷ 5 = 3 余 2
17 % 5 = 2        ← 取模，只要余数
17 / 5 = 3        ← 整数除法，只要商
```

你看，`/` 取商，`%` 取余数。分工明确。

取模运算有一个硬性规定：**只能用于整数，不能用于浮点数。** `5.5 % 2` 编译器会直接报错。这是 C++ 的规则，记住就行。

好，取模看起来很简单——不就是求余数嘛，有什么了不起？但你绝对想不到，这个"简单的余数运算"在编程里有多少神奇的用法。我给你们展示三个经典场景。

**场景一：判断奇偶。**

一个整数除以 2，如果余数是 0，它就是偶数；余数是 1，它就是奇数。

```cpp
int n;
cin >> n;
cout << n % 2 << endl;   // 输出 0 就是偶数，输出 1 就是奇数
```

以后你学了条件判断语句，就可以写成"如果 `n % 2 == 0` → 输出'偶数'，否则 → 输出'奇数'"。这段逻辑你现在就可以在脑子里模拟。

**场景二：提取个位数字。**

一个整数对 10 取模，得到的就是它的个位数。

```cpp
int n = 123;
int ge = n % 10;   // ge = 3（个位）
int shi = n / 10 % 10;  // shi = 2（十位）
int bai = n / 100 % 10;  // bai = 1（百位）
```

这招在竞赛里太常用了——遇到"把一个数的每一位取出来"的问题，`% 10` 和 `/ 10` 就是你的左右手。

**场景三：循环回到起点。**

假设今天是星期三（可以编号为 3），那 100 天后是星期几？100 ÷ 7 = 14 周余 2 天。星期三加 2 天 = 星期五。用取模一步搞定：

```cpp
int today = 3;                  // 3 代表星期三
int after100 = (today + 100) % 7;  // (3+100)%7 = 103%7 = 5 → 星期五
```

任何让你"绕圈子"的问题——星期几、时钟、轮转——取模都能帮你优雅地解决。

小总结一下取模的三大经典场景：奇偶判断用 `% 2`，取个位用 `% 10`，循环问题用 `% N`。

还有一个要提醒的——**取模不能除以 0！** `x % 0` 和 `x / 0` 一样，会让程序崩溃。在写取模运算之前，注意除数不能是 0。

好了，今天的取模虽然是小学知识，但它在编程里的威力远超你想象。这些用法装进你的工具箱，以后会经常用到。下节课我们学一个更"狡猾"的运算符——自增自减 `++` 和 `--`。

## 🖥️ 板书/屏幕要点
```
取模 % = 求余数

  17 % 5 = 2    （17 ÷ 5 = 3 余 2）
  17 / 5 = 3    （整数除法，取商）

  ⚠ % 只能用于整数！5.5 % 2 → 编译错误

三大经典场景：
  ① 判断奇偶    n % 2 == 0 → 偶数
                n % 2 == 1 → 奇数
  ② 取个位      n % 10 → 个位数
                n / 10 % 10 → 十位数
  ③ 循环绕圈    (today + 100) % 7 → 100天后星期几

取数位公式：
  个位 = n % 10
  十位 = n / 10 % 10
  百位 = n / 100 % 10
```

## ✏️ 随堂练习
输入一个三位整数（如 368），分别输出它的百位、十位、个位。用今天学的 `/` 和 `%` 运算符实现。测试数据：368 → 百位3, 十位6, 个位8；507 → 百位5, 十位0, 个位7。

## 📌 小结（30秒总结）
取模 `%` 就是求余数，只能用于整数。三大神技：`% 2` 判奇偶，`% 10` 取个位，`% N` 做循环。搭配除法 `/`，你可以轻松提取一个数的任意位数。这个看似简单的运算符，是编程里的瑞士军刀——功能不多，但一招封喉。
