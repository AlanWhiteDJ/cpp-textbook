# 第3章 第1课时：变量是什么——内存里的"贴标签格子"

**时长**：15-20分钟 | **对应教材**：第3章 · 3.1-3.2节

## 🎯 本课时目标
- 学生能用自己的话解释"变量是什么、为什么叫变量"
- 学生能在代码中正确声明一个 int 变量
- 学生能区分"声明"和"赋值"两个概念

## 📝 讲稿正文

同学们好！前面两章，我们让计算机说出了 Hello World。但有一个问题——程序每次说完话就忘了。比如你让它算 3+5，算出来是 8。但如果你下次想用这个 8 继续算，程序已经不记得了。

这就像一个记性很差的人——你告诉他一件事，他转个身就忘了，你得重新说一遍。

那能不能让程序"记住"东西呢？可以！这就是今天的主角——**变量**。

我先讲一个故事。小明记性不好，每次要用一个数字都得重新输一遍。第一天他算 3+5，得出 8。第二天老师问他："昨天的 8 还在吗？"小明说："没了，我只记得今天的。"老师叹了口气说："你就不能把 8 存起来吗？给它贴个标签，什么时候要用就拿过来？"

**这就是变量的由来。变量 = 内存里的一个贴了标签的格子。你把数字放进去，给它取个名字。以后要用这个数字，叫它名字就行。**

好，我们来具体看一下。计算机的内存，你可以想象成一排一排的储物柜。每个柜子可以放一个数字。现在你要存一个数字——怎么做呢？

第一步，**声明变量**——你找一个空的储物柜，在上面贴一张标签，标签上写这个柜子的名字，同时规定这个柜子里放什么东西。

在 C++ 里，声明一个整数变量的写法是：

```cpp
int score;
```

`int` 是说"这个柜子只装整数"，`score` 是柜子的名字。分号表示说完了。

第二步，**给变量赋值**——你把东西放进柜子里：

```cpp
score = 95;
```

把 95 放进名字叫 score 的柜子里。

第三步，**使用变量**——你按标签找到柜子，取出里面的东西：

```cpp
cout << score;  // 输出 95
```

三步走完，你就完成了一次"让程序记住数字"的操作。

我们也可以声明的同时就赋值——这叫做**初始化**：

```cpp
int score = 100;   // 声明并初始化为 100
int year = 2026;   // 声明并初始化为 2026
```

一行搞定，简洁又安全。

你也可以一行声明多个同类型的变量：

```cpp
int a, b, c;   // 声明三个整数变量
```

但这个写法不太推荐——最好一行只声明一个，代码更清晰。

好，我们来总结一下今天的关键词：**变量声明**就是找柜子贴标签，**变量赋值**就是把东西放进柜子，**变量使用**就是按标签取东西。记住这三个动作，变量你就掌握了三分之一。

下节课我们聊聊变量的"名字"——能取什么名字，不能取什么名字，以及怎么取个好名字。

2. "声明变量时不赋值，柜子里是什么？你猜猜看。"
3. "变量为什么叫'变'量？它跟数学里的'常量'有什么区别？"

## 🖥️ 板书/屏幕要点
```
变量 = 内存里贴了标签的格子

三个动作：

① 声明：int score;     找柜子，贴标签，规定类型
② 赋值：score = 95;    把东西放进柜子
③ 使用：cout << score; 按标签取出东西

初始化（声明 + 赋值一步完成）：
  int score = 100;
  int year = 2026;

目前知道的类型：
  int = 整数（整型）
```

## ✏️ 随堂练习
在 IDE 里写一个小程序：声明一个 int 变量 `age`，赋值为你的年龄，然后用 `cout` 输出"我今年 [age] 岁"。编译运行，确认年龄正确输出。

```cpp
int age = 13;
cout << "我今年 " << age << " 岁" << endl;
```

## 📌 小结（30秒总结）
变量就是内存里的一个贴了标签的格子——声明是找柜子贴标签，赋值是把东西放进去，使用是按标签取东西。int 是目前用来装整数的柜子类型。声明时顺手初始化（`int x = 0;`），是编程里最重要的好习惯之一。
