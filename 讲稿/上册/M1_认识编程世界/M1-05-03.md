# 第5章 第3课时：cout 输出控制——换行、小数、美化输出

## 📝 讲稿正文

同学们，前两节课我们学会了用 cin 读数据、用 cout 输出。但到目前为止，我们的输出还不够"精致"——小数一长串，格式也不够整齐。今天我们就来学怎么让输出更漂亮。

先说**换行**。C++ 有两种换行方式：

第一种，`endl`——end line 的缩写，"结束这一行"。你们一直在用：

```cpp
cout << "第一行" << endl;
cout << "第二行" << endl;
```

第二种，`\n`——写在字符串里面，也是一个换行符：

```cpp
cout << "第一行\n第二行\n第三行";
```

两种写法效果一样。现阶段用 `endl` 更直观——它就是"这一行输出完了，光标换下一行"。以后学到大量输出、对性能有要求的时候，`\n` 会更快。但现阶段你爱用哪个用哪个。

说到输出多个东西，我们之前一直在用 `<<` 把它们串起来：

```cpp
cout << "身高：" << 175 << "cm，体重：" << 65 << "kg" << endl;
```

`<<` 就像一个胶水，把文字、数字、变量粘成一句话输出。你可以无限串联。

---

接下来是今天最重要的内容——**控制小数位数。**

假设你要输出圆周率 3.14159265，但只想保留两位小数——3.14。怎么做？

C++ 提供了一个工具叫 `setprecision`，但它藏在 `<iomanip>` 这个头文件里。写法是这样的：

```cpp
#include <iomanip>  // 格式化输出的工具箱

// ...

double pi = 3.14159265;
cout << fixed << setprecision(2) << pi << endl;  // 输出 3.14
cout << fixed << setprecision(4) << pi << endl;  // 输出 3.1416
```

注意两个关键词要一起用——`fixed` 加 `setprecision(N)`：
- `fixed` 的意思是"用定点格式显示"——就是正常小数点，不是科学计数法。
- `setprecision(N)` 的意思是"保留 N 位小数"。

如果你只写 `setprecision(2)` 不写 `fixed`——那它控制的是"有效数字位数"而不是"小数位数"。比如 `setprecision(2)` 不搭配 `fixed`，可能输出 `3.1`（两位有效数字），而不是 `3.14`。所以记住：**fixed + setprecision(N)，缺一不可。**

看一下完整示例：

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double price = 12.8;
    double discount = 0.75;
    double finalPrice = price * discount;

    cout << "原价：" << price << " 元" << endl;
    cout << "折扣：" << discount * 100 << "%" << endl;
    cout << "折后价：" << fixed << setprecision(2) 
         << finalPrice << " 元" << endl;
    return 0;
}
```

输出：

```
原价：12.8 元
折扣：75%
折后价：9.60 元
```

注意 `9.60`——用了 fixed+setprecision(2)，C++ 会自动补 0 来凑齐两位小数。这在输出金额的时候特别有用。

好，关于输出控制，你们现在记住三件事：endl 换行，`<<` 串联，`fixed` + `setprecision(N)` 控制小数位数。再加上一个——`#include <iomanip>` 别忘了。下节课我们做第五章的综合练习和总复习。

## 🖥️ 板书/屏幕要点
```
换行：
  endl    → 独立使用：cout << "..." << endl;
  \n      → 嵌入字符串："第一行\n第二行"

输出串联：
  cout << "A" << 123 << "B" << endl;
  用 << 把任何东西串起来

小数位数控制：
  #include <iomanip>   ← 千万别忘！

  double pi = 3.14159265;
  cout << fixed << setprecision(2) << pi;  // 3.14
  cout << fixed << setprecision(4) << pi;  // 3.1416

  ⚠ fixed + setprecision(N) 缺一不可！
    只用 setprecision → 控制有效数字，不是小数位
```

## ✏️ 随堂练习
写程序让用户输入圆的半径 r（double），然后计算并输出圆的周长（2πr）和面积（πr²），π 取 3.14159。周长和面积各保留 2 位小数。提示：需要 `#include <iomanip>`。测试输入 5，周长应约 31.42，面积应约 78.54。

## 📌 小结（30秒总结）
cout 输出美化三招：第一，endl 或 \n 控制换行；第二，<< 串联任意内容；第三，fixed + setprecision(N) 控制小数位数——别忘了 `#include <iomanip>`。这三个技巧，让你的程序输出从"凑合能看"升级到"专业好看"。
