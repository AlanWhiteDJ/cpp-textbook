# 第4章 第3课时：char——单个字符的"单人公寓"

**时长**：15-20分钟 | **对应教材**：第4章 · 4.5节

## 🎯 本课时目标
- 学生能区分 char（单个字符）和 string（一串字符）
- 学生知道 char 用单引号，占 1 个字节
- 学生理解 char 本质上存的是 ASCII 码值

## 📝 讲稿正文

同学们，前面两节课我们学了三种数字类型——int、long long、double。从今天开始，我们进入另外两个"领地"：文字和判断。

先来想一个问题：如果你的程序要存一个考试成绩等级——比如 A、B、C、D——用 int 行吗？把 A 存成 1、B 存成 2？技术上可以，但极其别扭。你在代码里看到 `grade = 1`，谁知道这是 A 还是 B？

C++ 专门提供了一个类型来存**单个字符**——它叫 `char`，是 character 的缩写。

char 的特点很鲜明：它只占 1 个字节，只能存**一个**字符。一个字母、一个数字、一个标点符号——正好一个，不多不少。

写法上，char 用**单引号**括起来：

```cpp
char grade = 'A';         // 成绩等级
char firstLetter = 'Z';   // 首字母
char symbol = '?';        // 标点符号
char digit = '7';         // 注意：'7' 是字符，不是数字 7！
```

这里有一个极其重要的区分——**单引号和双引号的区别。** 单引号 `'A'` 是 char，只能装一个字符。双引号 `"A"` 是 string，虽然内容看起来一样，但它们是两种完全不同的类型。用错引号是新手极其常见的错误，请大家一定记住：**单个字符 → 单引号 → char。**

好，再讲一个有趣的"内幕"——char 在计算机内部，存的其实是一个整数。

每个字符在计算机里都有一个对应的数字编码，这套编码叫 **ASCII 码**。比如：
- `'A'` 的 ASCII 码是 65
- `'B'` 的 ASCII 码是 66
- `'a'` 的 ASCII 码是 97
- `'0'` 的 ASCII 码是 48（注意：字符 0 不是数字 0！）

所以 char 变量虽然显示的是字符，但在内存里存的是一个 0 到 127 之间的整数。这就是为什么 char 只占 1 个字节——它只需要存一个很小的数字。

知道了这一点，你就可以做一些有趣的操作：

```cpp
char c = 'A';
c = c + 1;          // c 变成了 'B'（65 + 1 = 66 → 'B'）
cout << c << endl;  // 输出 B
```

把字符加 1，就得到了字母表里的下一个字母。这在竞赛里偶尔会用到——比如要把小写字母转成大写字母。

说到大小写转换，这是个实用技巧。大写字母 A-Z 的 ASCII 码是 65-90，小写字母 a-z 是 97-122。大写和小写的差是 32。所以：

- 小写转大写：减 32（`'a' - 32 = 'A'`）
- 大写转小写：加 32（`'A' + 32 = 'a'`）

现阶段不需要背 ASCII 表，知道有这回事就行。以后需要的时候查一下就好了。

好了，今天我们只学了一个类型——char。但它很特别：占空间最小（1 字节），用单引号，本质上存的是数字编码。下节课我们学 char 的"大家庭版本"——string，它能存一整句话。

## 🖥️ 板书/屏幕要点
```
char — 单个字符（1 字节）

  声明：char grade = 'A';    ← 单引号！
        char symbol = '?';

  单引号 vs 双引号：
    'A' → char（单个字符）
    "A" → string（字符串）

char 的真面目：
  存的是整数（ASCII 码）
  'A' = 65, 'B' = 66, 'a' = 97, '0' = 48

  字符可以做运算：
    char c = 'A';
    c = c + 1;     // c 变成 'B'

  大小写转换：
    'a' - 32 = 'A' （小写→大写）
    'A' + 32 = 'a' （大写→小写）
```

## ✏️ 随堂练习
写程序声明一个 char 变量，初始化为 'A'，然后用 `cout` 输出这个字符。接着把它加 1 再输出，看看是什么。再试试加 2、加 3——你会发现它在按字母表顺序走。最后尝试把 char 变量直接赋给一个 int 变量并输出，看看它对应的 ASCII 码值是多少。

## 📌 小结（30秒总结）
char 是专门装单个字符的类型——占 1 字节，用单引号，本质上是存 ASCII 码。单引号是 char，双引号是 string——这一点千万不要搞混。char 能做算术运算，加 1 就得到字母表的下一个字母，这在处理字符的时候非常有用。下节课我们学 string——char 的"大家庭版"。
