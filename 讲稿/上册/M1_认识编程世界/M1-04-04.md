# 第4章 第4课时：string 与 bool——文字串和真假的开关

**时长**：15-20分钟 | **对应教材**：第4章 · 4.6-4.7节

## 🎯 本课时目标
- 学生能用 string 存名字、一句话等文字内容
- 学生知道使用 string 需要 `#include <string>`
- 学生能理解 bool 类型——只有 true 和 false 两种值

## 📝 讲稿正文

上节课我们学了 char——装单个字符。那如果你想存一个名字呢？比如"小明"，这是两个字符。如果想存一句话呢？比如"Hello, World!"，十几个字符。

这时候 char 就装不下了，你需要一个更大的容器——**string**。

string 是"字符串"——一串字符连在一起。用**双引号**括起来：

```cpp
string name = "小明";
string greeting = "Hello, World!";
string school = "阳光小学";
```

string 用起来跟 int、double 没有太大区别——声明、赋值、输出，语法一样：

```cpp
string name = "小明";
cout << "你好，" << name << "!" << endl;  // 输出：你好，小明！
```

但是！string 有一个特殊的地方——它和之前学的 int、double、char、bool 不一样，它不是 C++ 的"原生"类型，而是标准库提供的工具。所以在使用 string 之前，你必须额外引入它的头文件：

```cpp
#include <string>   // 用 string 必须加这一行！
```

看一下完整的例子：

```cpp
#include <iostream>
#include <string>        // ← 别忘了！
using namespace std;

int main() {
    string name = "小明";
    cout << "我的名字是" << name << endl;
    return 0;
}
```

如果你忘了 `#include <string>`，编译器会说 `'string' was not declared`——它不认识 string 是什么。加上这行就行了。

现阶段你只需要知道怎么声明 string、怎么输出 string 就够了。后面会有一整章专门讲 string 的高级操作——拼接、长度、查找、截取。今天先当它是"装文字的容器"用着。

---

好，string 讲完了。我们来学今天第二个类型——**bool**。

生活中有很多问题的答案只有两种——是或不是、对或错、真或假。编程里也一样：这个学生及格了吗？今天下雨了吗？用户输入的密码正确吗？

C++ 专门为这种"二选一"的场景准备了一个类型——**bool**。它只能存两个值：`true`（真）或 `false`（假）。

```cpp
bool isPassed = true;     // 及格了
bool isRaining = false;   // 没下雨
bool hasKey = true;       // 有钥匙
bool isPrime = false;     // 不是素数
```

bool 只占 1 个字节——因为只需要区分两种状态，1 个 bit 都够了，但计算机最小分配 1 个字节。

这里有一个可能让你困惑的细节：如果你用 `cout` 输出 bool 变量，`true` 会显示为 `1`，`false` 会显示为 `0`。这不是 bug，而是 C++ 的设计——`true` 在计算机内部就是 1，`false` 就是 0。

```cpp
bool b1 = true;
bool b2 = false;
cout << b1 << " " << b2 << endl;  // 输出：1 0
```

所以 bool 看起来简单，但它是编程逻辑的灵魂——以后你每写一个 `if` 判断、每一个循环条件，背后都是 bool 在帮你做决定。学好 bool，是为下一模块打基础。

---

好了，六种类型全部介绍完了。我们来做个快速回顾：
- 数字：int（整数主力）、long long（大整数）、double（小数）
- 文字：char（单个字符，单引号）、string（一串文字，双引号，需要 `#include <string>`）
- 判断：bool（true/false）

下节课我们用 `sizeof` 来亲自测量每种类型的体积，然后做一个"选择指南"——遇到什么场景该用什么类型。

## 🖥️ 板书/屏幕要点
```
string — 字符串（一串字符）
  双引号！
  string name = "小明";
  string msg = "Hello, World!";

  ⚠ 需要 #include <string>
  ⚠ 单引号≠双引号：'A'是char，"A"是string

---
bool — 布尔值（真/假）
  只有两个值：true 或 false

  bool isPassed = true;
  bool isRaining = false;

  cout 输出：true→1，false→0

---
六种类型全貌：
  数字：int / long long / double
  文字：char / string（+ #include <string>）
  判断：bool
```

## ✏️ 随堂练习
写程序声明一个 string 变量存你的名字，一个 bool 变量存"今天心情好"（true/false 自己选）。用 cout 输出一句话："我叫[名字]，今天心情好：[bool 值]"。运行看看 bool 输出的是 true/false 还是 1/0？然后检查——如果你忘了写 `#include <string>`，编译器会怎么说？

## 📌 小结（30秒总结）
string 是用来存一串文字的——双引号，需要 `#include <string>`。bool 是"是/否"开关——只有 true 和 false 两种值，输出显示为 1 和 0。加上前两节课的 int、long long、double、char，六种类型全部到齐。下节课我们用 sizeof 来量一量它们各自占多少空间，然后做一个"选择指南"。
