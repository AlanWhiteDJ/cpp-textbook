## 📝 讲稿正文

同学们，前两节课我们学了绝对值和平方根、幂运算和比较。这些都是"确定性的"工具——给同样的输入，永远输出同样的结果。

但生活中有很多事是不确定的——掷骰子掷出几点？抽卡抽到什么？敌人从哪个方向过来？这些就需要**随机数**。

C++ 的随机数工具不在 `<cmath>` 里，而是藏在 `<cstdlib>` 这个头文件中。核心函数叫 `rand()` ——random（随机）的缩写。

`rand()` 每次被调用，返回一个 0 到 `RAND_MAX` 之间的整数。`RAND_MAX` 通常等于 32767。

最简单的用法：

```cpp
#include <cstdlib>   // rand 的家
#include <iostream>
using namespace std;

int main() {
    cout << rand() << endl;   // 比如输出 41
    cout << rand() << endl;   // 比如输出 18467
    cout << rand() << endl;   // 比如输出 6334
    return 0;
}
```

看起来很不错——每次调用输出不同的数字。但是，请你把这个程序关了再重新运行一次。

你会发现——**输出的"随机数"跟上一次运行完全一样！** 每次都是 41、18467、6334……这哪叫随机啊？

这是因为 `rand()` 不是真随机，它是**伪随机**。它用一个叫"种子"（seed）的数字作为起点，然后通过一个复杂的数学公式算出后面的数字。只要种子不变，整个"随机"序列就永远是固定的。

就像你做一道菜，配方（算法）是固定的，食材（种子）相同，做出来的菜永远一样。想做出不同的菜，就得换不同的食材。

设置种子的函数叫 `srand()` ——seed random。但我们用什么做种子呢？用**当前时间**——时间每一秒都在变，所以种子也在变，随机序列就真的"随机"了。

获取当前时间的函数是 `time(0)`，它藏在 `<ctime>` 头文件里：

```cpp
#include <cstdlib>   // rand 和 srand
#include <ctime>     // time
#include <iostream>
using namespace std;

int main() {
    srand(time(0));      // 用当前时间做种子！
    cout << rand() << endl;   // 每次运行都不一样了
    cout << rand() << endl;
    return 0;
}
```

好，现在真正的随机数有了。但 `rand()` 直接输出是一个 0~32767 的大数字。实际场景里，我们通常需要特定范围的随机数——比如骰子需要 1~6，抽卡需要 1~100。

这里有一个**万能公式**：

```
rand() % (b - a + 1) + a    →  生成 a 到 b 的随机整数
```

我们拆解一下这个公式。`rand() % N` 生成 0 到 N-1 的整数。如果你想生成 1~100，那就是 `rand() % 100 + 1`——取模 100 得到 0~99，加 1 变成 1~100。

举几个例子：

```cpp
int dice = rand() % 6 + 1;         // 1~6 的骰子
int lottery = rand() % 100 + 1;    // 1~100 的抽奖
int score = rand() % 60 + 40;      // 40~99 的分数
// 验证：区间宽度 = 60, a=40, b=99
// rand() % (99-40+1) + 40 = rand() % 60 + 40
```

记住这个公式——它把"0 到好几万的巨大随机数"映射到"你想要的任意范围"。竞赛和日常编程里，这个公式经常用到。

好了，今天的核心：`rand()` 生成随机数，但不设置种子每次结果相同。加一行 `srand(time(0))` 用时间做种子，再搭配万能公式 `rand() % (b-a+1) + a`，你想要什么范围的随机数都行。下节课我们学剩下的几个数学函数，然后做第七章总复习。

## 🖥️ 板书/屏幕要点
```
随机数：

  #include <cstdlib>   ← rand()/srand() 的家
  #include <ctime>     ← time() 的家

  rand() → 0 ~ RAND_MAX (≈32767) 的整数
  ⚠ 不设种子 = 每次运行输出相同！

  设种子：
    srand(time(0));   ← 用当前时间做种子

万能公式（生成 a~b 的随机整数）：
  rand() % (b - a + 1) + a

实例：
  骰子   → rand() % 6 + 1        (1~6)
  抽奖   → rand() % 100 + 1      (1~100)
  40~99  → rand() % 60 + 40      (60个数)
  随机真假 → rand() % 2           (0或1)
```

## ✏️ 随堂练习
写程序用 `srand(time(0))` 设置种子，然后生成并输出：
1. 一个 1~6 的骰子点数
2. 一个 1~100 的随机数
3. 一个 60~100 的随机分数

关闭程序重新运行两次，确认每次结果不一样。然后尝试**不设置种子**运行，对比结果是否相同。

## 📌 小结（30秒总结）
`rand()` 生成的是伪随机数——种子一样，序列就一样。`srand(time(0))` 用当前时间做种子，让每次运行结果不同。万能公式 `rand() % (b-a+1) + a` 生成任意范围的随机数——骰子、抽奖、模拟数据，一招搞定。从此你的程序不再"死板"，而是有了不确定性的灵魂。
