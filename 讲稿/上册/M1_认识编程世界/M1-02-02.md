# 第2章 第2课时：Hello, World!——你人生中第一个 C++ 程序

**时长**：15-20分钟 | **对应教材**：第2章 · 2.2节

## 🎯 本课时目标
- 学生能独立敲出完整的 Hello World 程序
- 学生能成功编译并运行，在屏幕上看到 "Hello, World!"
- 学生能体会到"我写的代码在电脑上跑起来了"的成就感

## 📝 讲稿正文

同学们，今天是一个值得纪念的日子——你要写出人生中第一个 C++ 程序了。

三十年前，有个叫 Brian Kernighan 的程序员写了一本书，书里的第一个示例程序就是在屏幕上输出"Hello, World!"。从那以后，全世界所有编程语言的第一课，几乎都是从 Hello World 开始的。这是一种传承，也是一个仪式——你写完这个程序，你就正式成为程序员大家庭的一员了。

好，现在请打开你上节课装好的 IDE。

第一步，新建一个 C++ 源文件。不同 IDE 操作不同，但一般都是在"文件"菜单里选"新建"，然后选"C++ Source File"或"新建源文件"。文件名可以叫 `hello.cpp`。

第二步，把下面这段代码，**一个字一个字地敲进去**。不要复制粘贴——这是你的第一个程序，每一个字符都应该从你的指尖出来。

在敲之前，我先给大家看一个"错误版"——这是新手最容易写出来的东西。你们对照一下自己的代码，看看有没有这些问题：

```cpp
include <iostream>           // 少了 # 号
using namespace std          // 少了分号！
int main() {
    cout << "Hello, World!"   // 少了分号和 endl
    return 0;
}
```

好，现在我给大家展示正确版本。请大家盯着屏幕，跟我一起敲：

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

我来带着大家一行一行敲，说一句，敲一句。

第一行：`#include <iostream>` —— 井号 include 空格 左尖括号 iostream 右尖括号。注意，是井号不是中文的井，是英文的小于号和大于号。

第二行：`using namespace std;` —— using 空格 namespace 空格 std 分号。这里有一个分号，千万不要漏掉。

第三行：空一行，让代码看起来清爽一些，然后敲 `int main() {` —— int 空格 main 左括号右括号 空格 左大括号。

第四行：按一下 Tab 键缩进，然后敲 `cout << "Hello, World!" << endl;` —— 注意 cout 是小写，双引号里的内容可以随便写，endl 最后面有分号。

第五行：`return 0;` —— return 空格 0 分号。

第六行：`}` —— 右大括号，和第三行的左大括号配对。

都敲好了吗？现在，请点击工具栏上的"编译并运行"按钮——通常是 ▶ 按钮，或者按 F11 或 F9 键。

如果一切正常，你会看到一个黑窗口弹出来，上面写着一行字：

```
Hello, World!
```

如果你看到了这行字——恭喜！你刚刚完成了一件了不起的事：你写的代码，被翻译成了计算机能懂的机器语言，然后计算机执行了你的指令，在屏幕上显示出了你想要的文字。这不是魔法，这是你创造的。

如果你看到的是一片红色错误提示——别慌。检查三件事：第一，有没有少敲分号？第二，引号有没有配对？第三，有没有用中文标点？如果检查完了还是不对，举手找老师。

好了，下一节课我们把这六行代码拆开，一行一行搞清楚它们到底在干什么。今天你只需要记住一件事：**我写出第一个程序了，它跑起来了。** 这种感觉，是你往后几百章的动力来源。

## 💬 互动提问
1. "Hello World 跑起来的那一刻，你心里什么感觉？"
2. "如果把双引号里的 'Hello, World!' 换成一串中文，会发生什么？试试看。"
3. "如果故意删掉一个分号再运行，编译器的提示信息你看得懂吗？"

## 🖥️ 板书/屏幕要点
```
✅ 正确版 Hello World：

#include <iostream>        ← 引入输入输出库
using namespace std;       ← 打开标准工具箱

int main() {               ← 程序入口
    cout << "Hello, World!" << endl;   ← 输出
    return 0;              ← 正常结束
}

新手最容易犯的 4 个错：
  ① 少了 #（写成 include <iostream>）
  ② 少了分号
  ③ 用中文标点（，；""）
  ④ 大小写写错（Cout / Main / RETURN）

编译运行按钮：▶ 或 F11 / F9
```

## ✏️ 随堂练习
在 Hello World 的基础上，修改双引号中的文字，让程序输出"你好，世界！我是[你的名字]"。编译运行，确认输出正确后，把代码和运行结果截屏保存——这是你的"编程出生证"。

## 📌 小结（30秒总结）
你刚刚完成了一个传承了三十年的仪式——写出 Hello World。六行代码，你敲的每一个字符都被编译器翻译成机器语言，然后在屏幕上显示出了你写的文字。这种感觉就叫"编程"。记着它，这是你一切的动力。下一节课我们把这六行拆开，一行行弄懂。
