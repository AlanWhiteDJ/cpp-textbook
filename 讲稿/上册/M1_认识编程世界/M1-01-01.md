# 第1章 第1课时：计算机是什么——听话的机器

**时长**：15-20分钟 | **对应教材**：第1章 · 1.1节

## 🎯 本课时目标
- 学生能用自己的话说出"计算机是什么"
- 学生能说出计算机的五个组成部分，并各举一个例子
- 学生能理解"计算机不会自己思考，只会执行指令"这个核心认知

## 📝 讲稿正文

同学们好，欢迎来到信息学竞赛 C++ 编程的第一堂课。

在正式开始写代码之前，我想先问大家一个问题——你们每个人可能都用过计算机，但有没有想过：**计算机到底是什么？**

（停顿，让学生思考几秒）

好，我来帮大家理一理。假设你面前有一台笔记本电脑，你按下开机键，屏幕亮了，桌面上有各种图标。你能用鼠标点来点去，打开浏览器看视频，打开文档写东西。

现在想象一个场景——你的一个朋友从 1980 年穿越过来，他从来没见过电脑。他指着这个金属盒子问你："这玩意儿到底是什么？它为什么会听话？"

你怎么回答他？

（让学生试着回答）

好，我们来看看标准答案。**计算机本质上是一个能执行指令的机器。** 它自己不会思考，不会"想做什么就做什么"。你给它一条指令，它执行一条；给它一百条，它执行一百条。而且执行得飞快。

大家想一想，你们家里养过狗吗？或者说见过训练过的狗吗？你扔飞盘，它叼回来；你说"坐下"，它坐下。但它不会自己决定"今天心情好，我主动给你泡杯咖啡"——它做不到，它只会按训练好的指令行动。

计算机也一样。你让它算 3+5，它就乖乖算 3+5。你不让它算，它就什么都不做。所以今天我们要建立第一个重要认知——**计算机是最笨的东西，但它也是最听话的东西。**

好，那计算机是怎么做到"听话"的呢？我们用"人体"来打个比方，计算机跟我们人一样，也有五个关键的"器官"。

第一个，**输入设备**——相当于人的眼睛和耳朵，负责把外界信息送进计算机。比如键盘、鼠标、触摸屏、麦克风、摄像头，这些都是输入设备。你敲键盘就是在"喂"信息给计算机。

第二个，**输出设备**——相当于人的嘴巴和手，负责把计算结果展示出来。显示器、打印机、音箱，这些都是输出设备。屏幕上显示的东西，就是计算机"说"给你听的。

第三个和第四个我们要放在一起说——**运算器**和**控制器**。运算器相当于大脑的"计算区"，负责做算术和逻辑判断；控制器相当于大脑的"指挥区"，负责指挥其他部件协调工作。这两个合在一起，就是我们常说的 **CPU**——中央处理器，计算机的"大脑"。

第五个，**存储器**——相当于大脑的"记忆区"，负责记住数据和指令。存储器又分两种，这个特别重要，大家一定要注意听。**内存**，就像你考试时在草稿纸上演算——考试一结束，草稿纸就扔了，上面的东西就没了。内存也一样，电脑一关机，里面存的东西全消失。**外存**，就像你的笔记本——今天写了，合上，明天打开，字还在。硬盘、U 盘，都属于外存，断电也不丢数据。

好，我们快速过一遍：输入设备、输出设备、运算器、控制器、存储器——这五个部分，任何一台计算机都有，不管是台式机、笔记本，还是你口袋里的手机。

那问题来了——这些"器官"是怎么配合工作的呢？我们下节课继续。

## 💬 互动提问
1. "你觉得手机是计算机吗？为什么？"（引导学生用五部分模型分析手机）
2. "如果把计算机比作一个人，输入设备是什么？输出设备是什么？"
3. "CPU 是哪两个部分的合称？你能念出它们的全称吗？"

## 🖥️ 板书/屏幕要点
```
计算机 = 能执行指令的机器（不会自己思考）

计算机五部分：
  输入设备    →  眼睛耳朵（键盘/鼠标/触摸屏/麦克风）
  输出设备    →  嘴巴手（显示器/打印机/音箱）
  运算器    }  →  大脑
  控制器    }  CPU = 运算器 + 控制器
  存储器     →  记忆区
    内存 = 草稿纸（关机消失）
    外存 = 笔记本（永久保存）
```

## ✏️ 随堂练习
请同学们拿出一张纸，不看书，凭记忆画出计算机的五个组成部分，并在每个部分旁边写出一个具体的例子。写完后和同桌互相检查。

## 📌 小结（30秒总结）
计算机不是聪明的东西，它是听话的东西——你下指令，它执行，一条不多、一条不少。它由五个部分组成：输入、输出、运算器、控制器、存储器。CPU 是运算器加控制器。内存像草稿纸，外存像笔记本。记住这个模型，它是一切编程的起点。
