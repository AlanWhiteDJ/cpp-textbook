## 📝 讲稿正文

同学们，今天是第七章的收官课。我们先补几个"边角料"——取整函数家族的三兄弟，然后把整个第七章的知识串成一张大图。

数学里经常遇到要对小数取整的情况。C++ 提供了三个不同的取整函数：

**`floor(x)` ——向下取整。** 不管小数部分是多少，直接往小了取。比如 `floor(3.9) = 3`，`floor(3.1) = 3`。

**`ceil(x)` ——向上取整。** 不管小数部分是多少，直接往大了取。比如 `ceil(3.1) = 4`，`ceil(3.9) = 4`。

**`round(x)` ——四舍五入。** 这个最符合我们日常习惯——大于等于 .5 就进位，小于 .5 就舍去。比如 `round(3.4) = 3`，`round(3.5) = 4`，`round(3.6) = 4`。

三个函数的名字都很好记：floor = 地板（往低处走），ceil = ceiling 天花板（往高处走），round = 四舍五入。

```cpp
#include <cmath>
#include <iostream>
using namespace std;

int main() {
    cout << floor(3.7) << endl;   // 3
    cout << ceil(3.1) << endl;    // 4
    cout << round(3.5) << endl;   // 4
    cout << round(4.5) << endl;   // 5（.5 往偶数方向取，但一般当四舍五入用）
    return 0;
}
```

注意一个坑：`round(4.5)` 的情况稍微特殊——C++ 用的是"银行家舍入法"（四舍六入五成双），4.5 往最近的偶数 4 取，所以 `round(4.5)` 在某些编译器上可能输出 4。但通常当作四舍五入就行，差异很小。

好，所有主要函数都讲完了。现在我们来做第七章的总复习，列一张完整的"速查表"。

| 函数 | 含义 | 示例 | 结果 |
|------|------|------|------|
| `abs(x)` | 整数绝对值 | `abs(-7)` | 7 |
| `fabs(x)` | 浮点绝对值 | `fabs(-3.14)` | 3.14 |
| `sqrt(x)` | 平方根 | `sqrt(16)` | 4.0 |
| `pow(a,b)` | a 的 b 次方 | `pow(2,10)` | 1024 |
| `max(a,b)` | 较大值 | `max(3,9)` | 9 |
| `min(a,b)` | 较小值 | `min(3,9)` | 3 |
| `ceil(x)` | 向上取整 | `ceil(3.1)` | 4 |
| `floor(x)` | 向下取整 | `floor(3.9)` | 3 |
| `round(x)` | 四舍五入 | `round(3.5)` | 4 |
| `rand()` | 随机整数 | `rand()%10` | 0~9 |

这张表值得你抄在笔记本上，之后编程时随时查阅。

来串一发第七章的完整脉络——

第一课，我们打开了 `<cmath>` 工具箱，学到了 `abs`（整数绝对值）、`fabs`（浮点绝对值）和 `sqrt`（平方根）。用 `sqrt(a*a + b*b)` 算出了勾股定理的斜边长度。

第二课，我们学了 `pow`——幂运算，但要注意它是 double 类型有浮点误差，小指数直接用乘法更安全。还学了 `max` 和 `min` 做大小比较——它们在 `<algorithm>` 头文件里。

第三课，我们进入了随机数的世界——`rand()` 生成伪随机数，`srand(time(0))` 用时间做种子让每次运行结果不同，万能公式 `rand() % (b-a+1) + a` 生成任意范围随机数。

第四课（本节课），我们补上取整三兄弟——floor 向下、ceil 向上、round 四舍五入。

这张"数学工具箱速查表"就是你的知识资产。后面每一章你都会遇到需要用到其中某个函数的场景——绝对值，平方根，随机数——调出来就用。

```
第七章数学工具箱全表：

    <cmath>      → abs/fabs/sqrt/pow/ceil/floor/round

```
