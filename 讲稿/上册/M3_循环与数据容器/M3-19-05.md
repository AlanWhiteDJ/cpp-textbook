## 📝 讲稿正文

同学们，今天是第19章的最后一节课。我们要学一个"神器"级别的工具——**stringstream**，然后用它来做一个综合项目。

先看一个场景。假设你拿到了一整行数据：`"张三 12 95.5"`——里面包含姓名、年龄、分数，用空格分隔。你需要把它们分别提取出来。

你可能会想：用getline读进来，然后怎么办？自己一个个字符去找空格？太麻烦了。

这时候stringstream出场了。它就像一个"假键盘"——你把一整个字符串塞给它，然后用`>>`从里面一个词一个词地读出来，就跟从cin读一样！

需要`#include <sstream>`。

```cpp
#include <sstream>
// ...

string line = "张三 12 95.5";
stringstream ss(line);   // 把line塞进ss，ss现在就像一个输入源

string name;
int age;
double score;

ss >> name >> age >> score;  // 像cin一样逐个提取！

cout << "姓名：" << name << endl;    // 张三
cout << "年龄：" << age << endl;     // 12
cout << "分数：" << score << endl;   // 95.5
```

大家看，`ss >> name >> age >> score`这一行，跟`cin >> name >> age >> score`写法一模一样。区别只是——cin的数据来自键盘，ss的数据来自你塞进去的那个字符串。

stringstream的核心价值在于：**把字符串当作输入流，用统一的`>>`语法解析**。你不需要手动找空格、不需要substr截取、不需要stoi转换——stringstream自动按空格分隔，自动做类型转换。

更实用的例子——解析多个整数：

```cpp
string data = "10 20 30 40 50";
stringstream ss(data);
int sum = 0, x;
while (ss >> x) {    // 只要还能读到整数，就继续
    sum += x;
}
cout << "总和：" << sum << endl;  // 150
```

注意`while (ss >> x)`——这个写法很巧妙。`ss >> x`既做读取操作，又返回一个布尔值表示读取是否成功。读到末尾读不出来了就退出循环。

stringstream还有一个兄弟叫`ostringstream`——反过来，把各种类型"打印"到一个字符串里。不过现阶段掌握`stringstream`就够了，它是在竞赛中解析输入数据的利器。

## 🖥️ 板书/屏幕要点
- **stringstream**：把字符串当作输入流（需 `#include <sstream>`）
- 用法：
  ```cpp
  stringstream ss(字符串);
  ss >> 变量1 >> 变量2 >> ...;  // 像cin一样提取
  ```
- `while (ss >> x)` —— 读取直到末尾
- 自动按空格分隔 + 自动类型转换
- 本章总结：声明/输入/拼接/长度/转换/stringstream

## 📌 小结
stringstream是处理"包含多个信息的字符串"的最佳工具。它让你用统一的`>>`语法从字符串中提取数据，自动处理空格分隔和类型转换。加上前面学的声明、输入、拼接、长度、转换，你现在已经具备了string的"基础工具箱"。下一章，我们要升级——学string的查找、截取、删除、替换，让你像编辑Word文档一样修改文字。
