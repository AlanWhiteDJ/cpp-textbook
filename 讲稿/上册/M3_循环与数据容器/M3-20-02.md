# 第20章 第2课时：substr——精准裁剪的剪刀手
**时长**：15-20分钟 | **对应教材**：第20章 字符串的十八般武艺

## 🎯 本课时目标
- 掌握substr的语法——起始位置+长度
- 会用来截取前缀、后缀、中间任意一段
- 能用substr做简单的字符串提取与判断

## 📝 讲稿正文

同学们，上节课我们学了find——在一大段文字里找一小段的位置。好，假设你已经找到了位置，接下来想干嘛？想把那一小段文字**剪下来**单独用。这就是今天要学的——**substr**。

substr就像一把剪刀，从字符串中间剪下一段。格式是：`s.substr(起始位置, 长度)`。

```cpp
string s = "Hello World";

string sub1 = s.substr(0, 5);      // 从下标0开始，取5个 → "Hello"
string sub2 = s.substr(6, 5);      // 从下标6开始，取5个 → "World"
string sub3 = s.substr(6);         // 从下标6开始，取到末尾 → "World"
```

注意第三种写法——不写第二个参数（长度），就默认取到字符串末尾。这个很实用，比如"从某个位置截取到结尾"。

现在我们来做几个实战场景。

**场景一：判断前缀。**

检查一个文件是不是C++源文件（看后缀是不是".cpp"）：

```cpp
string filename = "hello.cpp";
int len = filename.size();
if (filename.substr(len - 4, 4) == ".cpp") {
    cout << "这是C++源文件" << endl;
}
```

从倒数第4个位置开始取4个字符，就是后缀".cpp"。

**场景二：提取中间部分。**

身份证号18位，第7~14位是出生日期。提取出来：

```cpp
string id = "320106200805121234";  // 18位身份证号
string birthday = id.substr(6, 8);  // 从下标6开始取8位
cout << "出生日期：" << birthday << endl;  // 20080512
```

注意——下标从0开始，所以第7位对应的下标是6。8位是年月日各占的位数：4位年+2位月+2位日。

**场景三：取前几个和后几个字符。**

```cpp
string s = "abcdefg";
string first3 = s.substr(0, 3);   // "abc"
string last3 = s.substr(s.size() - 3);  // "efg"（不写长度=取到末尾）
```

好，substr虽然好用，但有一个坑——**取的长度超过了字符串的长度**。

```cpp
string s = "Hi";
string sub = s.substr(0, 10);  // 要取10个，但s只有2个字符！
```

这个在不同编译器/平台上行为不一样，可能抛出异常，也可能返回奇怪的内容。所以使用substr前最好判断长度是否够：

```cpp
if (s.size() >= 10) {
    string sub = s.substr(0, 10);
} else {
    cout << "字符串太短" << endl;
}
```

安全第一，尤其在竞赛中，输入数据可能包含各种边界情况。

现在总结一下：substr(start, len)从start位置取len个字符，省略len就到末尾。常用于截取前缀后缀、提取特定位置的字段。使用前最好检查长度，避免越界。跟find配合使用效果更佳——find找到位置，substr截取内容。

## 🖥️ 板书/屏幕要点
- **substr语法**：`s.substr(起始位置, 长度)`
- 省略长度 = 取到末尾：`s.substr(5)` → 从位置5到结尾
- 前缀截取：`s.substr(0, n)` → 前n个字符
- 后缀截取：`s.substr(s.size() - n)` → 后n个字符
- 中间截取：`s.substr(pos, len)` → 指定位置指定长度
- ⚠️ 长度不要超过字符串剩余长度

## ✏️ 随堂练习
用户输入一个字符串，程序分别输出：①前3个字符；②后3个字符；③如果长度≥6，输出去掉首尾各2个字符后的中间部分。如果字符串长度不足，输出"太短了"。例如输入"HelloWorld"，输出"Hel"、"rld"、"lloWor"。

## 📌 小结
substr是string的"精准剪刀"——从哪开始、取多长，一刀下去干净利落。常见用法包括提取前缀后缀、截取特定字段。和find搭配使用时，先找位置再截内容，非常高效。注意长度参数不要越界。
