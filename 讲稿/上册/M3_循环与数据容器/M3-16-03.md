## 📝 讲稿正文

同学们，上节课我们学会了数组的下标规则。现在我们来思考一个问题：一个数组有100个元素，你要怎么处理每一个元素？

一个一个写？`cin >> arr[0]; cin >> arr[1]; cin >> arr[2]; ...`写到第100个？不行啊，这跟不用数组有什么区别？

这个时候，我们之前学的for循环就派上大用场了。你看——数组的下标是0、1、2、3、4……正好是连续递增的整数。而for循环的循环变量i，也是在0、1、2、3、4……连续递增。这叫什么？这叫**天作之合**。

所以我们写：

```cpp
int arr[100];
for (int i = 0; i < 100; i++) {
    cin >> arr[i];    // i从0变到99，恰好访问arr[0]到arr[99]
}
```

三行代码，搞定100个元素的输入。这就是循环+数组的威力。

大家注意观察——循环变量i同时扮演两个角色：它是循环的计数器，也是数组的下标。i从0跑到99，arr[i]就依次访问第0个到第99个元素。这个模式是数组操作的基本功，一定要练到闭着眼睛都能写出来的程度。

好，掌握了这个模式之后，我们来练几个最常见的遍历场景。

**场景一：输入数组。** 刚才已经看到了，这是最基本的：

```cpp
int n;
cin >> n;
int arr[1005];
for (int i = 0; i < n; i++) {
    cin >> arr[i];
}
```

注意，这里i是从0到n-1，用的是用户输入的n，不是数组声明的1005。因为我们实际只用了前n个位置。

**场景二：输出数组。** 把数组从头到尾打印出来：

```cpp
for (int i = 0; i < n; i++) {
    cout << arr[i] << " ";
}
cout << endl;
```

如果想要更友好的输出——"第1个元素是XX，第2个元素是XX"——我们可以利用i+1来表示第几个：

```cpp
for (int i = 0; i < n; i++) {
    cout << "第" << i + 1 << "个：" << arr[i] << endl;
}
```

这里有个小技巧：对人说话要说"第1个"，但计算机里下标是0。所以我们用i+1来换算。

**场景三：倒序遍历。** 如果我想从最后一个输出到第一个呢？

```cpp
for (int i = n - 1; i >= 0; i--) {
    cout << arr[i] << " ";
}
```

注意循环变量的变化——i从n-1开始，递减到0。`i >= 0`不能写成`i > 0`，否则会漏掉arr[0]。

**场景四：求和。** 这是统计里最常用的：

```cpp
int sum = 0;
for (int i = 0; i < n; i++) {
    sum += arr[i];
}
cout << "总和：" << sum << endl;
```

大家看，sum叫"累加器"，从0开始，每遍历一个元素就往里加一个。这就是求和的标准模板。

综合起来看——输入、输出、求和、倒序，本质上都是"for循环套一个操作"。循环的骨架不变：`for (int i = 0; i < n; i++)`，变的是花括号里面做什么。这就是"一个模式，多种用途"。

现在我们来做一个小综合。输入5个同学的成绩，计算总分和平均分：

```cpp
int scores[5];
int sum = 0;

// 输入
for (int i = 0; i < 5; i++) {
    cin >> scores[i];
}

// 求和
for (int i = 0; i < 5; i++) {
    sum += scores[i];
}

double avg = (double)sum / 5;  // 强制转double，保留小数

cout << "总分：" << sum << endl;
cout << "平均分：" << avg << endl;
```

这里有一个细节——`(double)sum / 5`。为什么要加`(double)`？因为如果sum是int，5也是int，整数除以整数会丢掉小数部分。比如431除以5等于86，而不是86.2。加了`(double)`之后，sum先转成小数，再做除法，小数就保住了。

## 🖥️ 板书/屏幕要点
- **遍历标准模板**：`for (int i = 0; i < n; i++) { 操作 arr[i]; }`
- 输入模式：`cin >> arr[i];`
- 输出模式：`cout << arr[i] << " ";`
- 求和模式：`sum += arr[i];`（sum初始化为0）
- 倒序遍历：`for (int i = n-1; i >= 0; i--)`
- 整数除法注意：`(double)sum / n` 保留小数

## 📌 小结
for循环和数组是天生的搭档——循环变量i天然地对应数组下标。标准遍历模式`for (int i = 0; i < n; i++)`是所有数组操作的基础。输入、输出、求和、倒序——模式一样，变的是循环体里的操作。把这个骨架练熟，后面学什么数组操作都会快很多。
