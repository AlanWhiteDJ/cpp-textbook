## 📝 讲稿正文

同学们，前面四节课我们学了string的编辑功能——find定位、substr截取、erase删除、insert插入、replace替换。今天我们来学最后两项技能：**字符处理函数**和**两种遍历方式**。

先说字符处理函数。这些函数处理的是**单个字符**（char），不是整个string。需要`#include <cctype>`。

来看看这张表：

| 函数 | 作用 | 例子 |
|------|------|------|
| `isdigit(c)` | c是数字吗？ | `isdigit('5')` → true |
| `isalpha(c)` | c是字母吗？ | `isalpha('A')` → true |
| `isupper(c)` | c是大写吗？ | `isupper('Z')` → true |
| `islower(c)` | c是小写吗？ | `islower('a')` → true |
| `isspace(c)` | c是空白吗？ | `isspace(' ')` → true |
| `toupper(c)` | 转成大写 | `toupper('a')` → 'A' |
| `tolower(c)` | 转成小写 | `tolower('Z')` → 'z' |

注意——这些函数不是C++特有的，是C语言标准库里的，所以头文件是`cctype`（C++风格）或`ctype.h`（C风格）。

【实战一：统计字符串中的数字个数】

```cpp
string s = "a1b2c3d4";
int cnt = 0;
for (int i = 0; i < s.size(); i++) {
    if (isdigit(s[i])) cnt++;
}
cout << "数字个数：" << cnt << endl;  // 4
```

【实战二：统计大写字母、小写字母、数字、其他字符】

```cpp
string s = "Hello123!";
int upper = 0, lower = 0, digit = 0, other = 0;
for (int i = 0; i < s.size(); i++) {
    if (isupper(s[i])) upper++;
    else if (islower(s[i])) lower++;
    else if (isdigit(s[i])) digit++;
    else other++;
}
cout << "大写：" << upper << " 小写：" << lower
     << " 数字：" << digit << " 其他：" << other << endl;
```

【实战三：大小写转换——有一个大坑！】

```cpp
string s = "hello";
for (int i = 0; i < s.size(); i++) {
    toupper(s[i]);        // ❌ 完蛋！白写了
}
cout << s << endl;         // 还是 "hello"，没变！
```

为什么？因为`toupper(s[i])`**不修改s[i]本身**，而是**返回**转换后的新字符。你必须把返回值存回去：

```cpp
for (int i = 0; i < s.size(); i++) {
    s[i] = toupper(s[i]);   // ✅ 把转换结果赋值回去
}
cout << s << endl;           // "HELLO"
```

这个坑踩的人超级多——调用了toupper，但没赋值，等于没调。记住：**toupper和tolower返回新值，必须赋值回去才能生效。**

好，字符处理函数说完了。接下来我们看**遍历字符串的两种方式**。

【方式一：下标遍历】你已经很熟了：

```cpp
string s = "Hello";
for (int i = 0; i < s.size(); i++) {
    cout << s[i] << " ";  // H e l l o
}
```

【方式二：范围for循环】C++11的新语法，更简洁：

```cpp
string s = "Hello";
for (char c : s) {       // 读作：对于s里的每个字符c
    cout << c << " ";    // H e l l o
}
```

如果要修改每个字符，用引用`&`：

```cpp
for (char &c : s) {      // 注意 & ——引用，可以修改
    c = toupper(c);
}
cout << s << endl;       // "HELLO"
```

两种方式各有优劣：下标遍历可以知道位置i，范围for更简洁优雅。按需选择。

好，现在来总结第20章。你学会了find、substr、erase、insert、replace、字符处理函数，以及两种遍历方式。这七项技能构成了string的"十八般武艺"——文字在你手里，想怎么改就怎么改。

## 🖥️ 板书/屏幕要点
- 字符函数（需`#include <cctype>`）：
  - 判断：`isdigit/isalpha/isupper/islower/isspace`
  - 转换：`toupper/tolower`
- ⚠️ `toupper(c)`返回新值，必须`s[i] = toupper(s[i])`
- **下标遍历**：`for (int i=0; i<s.size(); i++)` → 知道位置
- **范围for**：`for (char c : s)` → 简洁。修改用`char &c`

## ✏️ 随堂练习
用户输入一个字符串，统计其中大写字母、小写字母、数字、其他字符各有多少个。然后将字符串中的小写字母全部转成大写，大写全部转成小写（即大小写互换），输出转换后的字符串。例如输入"Hello123!"，输出统计结果和"hELLO123!"。

## 📌 小结
字符处理函数让你能逐个判断和转换字符类型，是文本分析的基础工具。记住toupper/tolower必须赋值回去——这是最高频的坑。遍历字符串有两种方式：下标遍历能知道位置，适合需要位置信息的场景；范围for更简洁，适合只需要每个字符本身的场景。第20章到此结束——你的string武功已经大成！
