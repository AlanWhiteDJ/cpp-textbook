# 第19章 第4课时：数字与文字互转——stoi与to_string
**时长**：15-20分钟 | **对应教材**：第19章 string入门

## 🎯 本课时目标
- 掌握`stoi`将字符串转为整数
- 掌握`to_string`将整数转为字符串
- 理解数字和文字转换的常见应用场景

## 📝 讲稿正文

同学们，前面我们学会了string的声明、输入、拼接、求长度。现在来学一个非常实用的技能——**数字和文字之间的互相转换**。

为什么要转换？想一想这个场景：你要输出一句话"你的分数是95分"。95是int类型的数字，不能直接用`+`拼到string上。你需要先把95变成字符串"95"，然后拼接。

反过来，用户输入了一个数字——但输入的是字符串"2026"。你需要把它转成int类型的2026才能做加减运算。

这就是今天要学的两个函数：`stoi`（string to int，字符串转整数）和`to_string`（整数转字符串）。

**先看to_string——数字变文字。**

```cpp
int score = 95;
string msg = "你的分数是 " + to_string(score) + " 分";
cout << msg << endl;  // 你的分数是 95 分
```

一行就搞定了。`to_string(95)`返回字符串`"95"`，然后拼接进整句话里。

更实用的例子——生成报告：

```cpp
int total = 431;
double avg = 86.2;
cout << "总分：" + to_string(total) + "，平均分：" + to_string(avg) << endl;
```

注意，`to_string`也可以转double——`to_string(3.14)`返回`"3.140000"`。

**现在看反方向——stoi，文字变数字。**

```cpp
string s = "2026";
int year = stoi(s);         // year变成整数2026
cout << year + 1 << endl;   // 2027——可以参与运算了
```

这有什么用呢？比如用户在一行里输入了两个数字"123 456"，你需要把它们读出来相加。读进来的是字符串，要用stoi转成int才能算。

```cpp
string s1 = "123";
string s2 = "456";
int a = stoi(s1);   // 123
int b = stoi(s2);   // 456
cout << a + b << endl;  // 579
```

好，stoi很方便，但有一个前提——**你传给它的字符串必须是合法的数字**。

```cpp
string s = "abc";
int x = stoi(s);  // ❌ 运行时崩溃！"abc"不是数字
```

stoi遇到"abc"这种根本不是数字的东西，会直接抛出异常导致程序崩溃。所以使用stoi时，要确保你转的字符串确实是数字格式。

那怎么判断一个字符串是不是数字呢？一种方法是遍历每个字符，用`isdigit()`检查——这个我们后面会学到。现阶段，只要确保从cin读到的、且你知道是数字的字符串才用stoi转换。

另外说一下，除了stoi（转int），还有：
- `stod(s)` —— string转double
- `stoll(s)` —— string转long long（更大范围的整数）

现阶段记住stoi和to_string就够了。

现在来做练习——用户输入一个年份（如"2008"），程序输出"明年是2009年"。需要先把输入的字符串转成int，加1，再转回string拼接输出。

## 🖥️ 板书/屏幕要点
- **int → string**：`to_string(数字)` → `"95"`
- **string → int**：`stoi("2026")` → `2026`
- 常见模式：
  - 拼接输出：`"分数：" + to_string(score)`
  - 解析输入：`int x = stoi(输入的字符串)`
- ⚠️ `stoi("abc")` → 运行时崩溃（非法格式）
- 扩展：`stod(s)`转double，`stoll(s)`转long long

## ✏️ 随堂练习
写一个"年龄计算器"：用户输入出生年份（字符串），程序用stoi转成整数，计算当前年龄（假设今年是2026年），然后用to_string转回字符串，输出"你今年XX岁"。例如输入"2008"，输出"你今年18岁"。

## 📌 小结
to_string和stoi是数字与文字之间的"翻译官"——一个把数字翻成文字，一个把文字翻成数字。它们让int和string可以无缝协作，是构建用户友好型程序的必备工具。记住stoi要求字符串格式合法，不然会崩溃。
