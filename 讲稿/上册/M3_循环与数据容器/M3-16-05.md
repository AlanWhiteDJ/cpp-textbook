## 📝 讲稿正文

同学们，前面四节课我们把一维数组的基础知识学完了——声明、下标、遍历、初始化、避坑。今天这第五节课，咱们来打一场"实战"，把学到的全部知识串起来，解决几个真实的问题。

第一个问题：**一周温度记录**。

假设我们要记录一周七天的温度，然后全部输出。这个很简单，我们直接来：

```cpp
double temps[7] = {23.5, 25.0, 22.8, 26.1, 24.3, 27.0, 25.6};
for (int i = 0; i < 7; i++) {
    cout << "星期" << i + 1 << "：" << temps[i] << "℃" << endl;
}
```

看，又是我们熟悉的模式——声明+初始化+for循环遍历。i+1把计算机的0~6转成人看的星期一到星期日。

第二个问题，稍微复杂一点——**数组反转**。

什么叫反转？就是把`{1, 2, 3, 4, 5}`变成`{5, 4, 3, 2, 1}`。第一个和最后一个换，第二个和倒数第二个换……一直换到中间。

我们动手写：

```cpp
int arr[5] = {1, 2, 3, 4, 5};
int n = 5;

for (int i = 0; i < n / 2; i++) {
    // 交换 arr[i] 和 arr[n-1-i]
    int temp = arr[i];
    arr[i] = arr[n - 1 - i];
    arr[n - 1 - i] = temp;
}

// 输出：5 4 3 2 1
for (int i = 0; i < n; i++) {
    cout << arr[i] << " ";
}
```

大家注意几个关键点。第一，循环只进行n/2次，不是n次。为什么？因为每一轮交换两个元素，n个元素只需要n/2轮。如果循环n次，会把换好的又换回去——白干。

第二，交换两个变量的值需要三步——这是经典的三步交换法：先用temp存一个，再把另一个赋给它，最后把temp赋给另一个。就像两个人交换座位，需要一个人先站到过道上。

我们来推演一下：n=5，n/2=2。

- i=0：交换arr[0]和arr[4]，1和5交换 → {5,2,3,4,1}
- i=1：交换arr[1]和arr[3]，2和4交换 → {5,4,3,2,1}
- i=2：不满足i < 2，循环结束

中间那个3不需要动，它正好在中间位置。

好，第三个问题——**分段统计成绩**。

输入n和n个0到100的成绩，统计优秀（≥90）、良好（75~89）、及格（60~74）、不及格（<60）各多少人。

这个问题用数组存成绩，然后用四个计数器分别统计：

```cpp
int n, scores[105];
cin >> n;
for (int i = 0; i < n; i++) cin >> scores[i];

int excellent = 0, good = 0, pass = 0, fail = 0;

for (int i = 0; i < n; i++) {
    if (scores[i] >= 90) excellent++;
    else if (scores[i] >= 75) good++;
    else if (scores[i] >= 60) pass++;
    else fail++;
}

cout << "优秀：" << excellent << endl;
cout << "良好：" << good << endl;
cout << "及格：" << pass << endl;
cout << "不及格：" << fail << endl;
```

大家注意判断顺序——先判断≥90，再判断≥75，再判断≥60。如果用`if (scores[i] >= 60)`先判断，那90分也会被算进"及格"里。多层if-else if的顺序很重要，要从高到低写。

好，今天的内容就到这里。现在请大家动手做随堂练习——输入8个同学的成绩，输出最高分、最低分和平均分。这道题需要你把遍历、比较、求和全部用上。

## 🖥️ 板书/屏幕要点
- **数组反转**：`for (i=0; i<n/2; i++) swap(arr[i], arr[n-1-i])`
- 三步交换法：temp → a → b → temp
- 分段统计：if-else if顺序从高到低
- 成绩统计模式：输入 → 遍历 → 分类计数 → 输出

## ✏️ 随堂练习
输入8个同学的成绩（0~100），存入数组。然后输出：最高分、最低分、平均分（保留一位小数）。提示：最高分初始值设为arr[0]，从i=1开始比较；最低分类似。

## 📌 小结
今天我们用一维数组解决了三个实际问题：温度记录、数组反转、分段统计。数组反转的"前后交换、只换一半"是经典思路，分段统计的"多个计数器+按顺序判断"也是常用模式。一维数组这一章到这里就结束了——你学会了声明、初始化、遍历、反转、统计。下一章，我们在这个基础上解锁更多的"数组魔法"——求和均值、桶计数、插入删除。
