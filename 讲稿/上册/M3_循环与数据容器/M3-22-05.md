## 📝 讲稿正文

同学们，今天是M3模块的最后一节课。我们先学一个高效的去重方法，然后来一次模块总复习。

在第17章我们学了去重——用一个新数组，双重循环检查每个元素是否已经出现过。那个方法清晰但效率不高。今天教一个更高效的方法——**排序后双指针去重**。

思路很简单：先把数组排序（相同的数会紧挨在一起），然后用两个指针，一个指向"不重复区间的末尾"，一个扫描整个数组。遇到跟"不重复区间末尾"不同的数，就把它加进去。

```cpp
int n, a[1000];
cin >> n;
for (int i = 0; i < n; i++) cin >> a[i];

// 第一步：排序（相同的数会相邻）
sort(a, a + n);

// 第二步：双指针去重
int pos = 0;  // pos指向"不重复区间"的最后一个位置
for (int i = 1; i < n; i++) {
    if (a[i] != a[pos]) {
        pos++;
        a[pos] = a[i];  // 把新元素覆盖到去重区
    }
}

// 输出去重结果（0到pos共pos+1个元素）
for (int i = 0; i <= pos; i++)
    cout << a[i] << " ";
```

来推演一下。假设排好序的数组是`{1, 1, 2, 3, 3, 4}`：

- pos=0，i=1：a[1]=1，等于a[0]=1，跳过。
- i=2：a[2]=2，不等于a[0]=1，pos变成1，a[1]=2。数组变成`{1, 2, 2, 3, 3, 4}`。
- i=3：a[3]=3，不等于a[1]=2，pos变成2，a[2]=3。数组变成`{1, 2, 3, 3, 3, 4}`。
- i=4：a[4]=3，等于a[2]=3，跳过。
- i=5：a[5]=4，不等于a[2]=3，pos变成3，a[3]=4。

最终不重复区间是0~3：`{1, 2, 3, 4}`。

这个方法的优势是不需要新数组，直接在原数组上操作，空间效率更高。`sort`函数我们后续会专门学，今天先用。

现在，让我们来一次**M3模块总复习**。这个模块是本书最长的模块——7章35课时。你学到了什么？

**第16章：一维数组。** 数组是一排同类型的储物柜，下标从0开始。声明用`int arr[N]`，遍历用`for(i=0; i<n; i++)`。记住n个元素下标范围0到n-1。越界是定时炸弹。

**第17章：数组的魔法。** 求和用累加器，最值用擂台法（初始值设a[0]！），桶计数用下标做分类。插入从后往前移，删除从前往后移，反转首尾交换n/2次，去重用双重循环。

**第18章：二维数组。** `int a[3][4]`是3行4列。遍历用双重循环——外层行内层列。转置`b[j][i]=a[i][j]`。主对角线i==j，副对角线i+j==n-1。传参必须指定列数。

**第19章：string入门。** string是一串字符。声明`string s = "Hello"`。cin>>读词，getline读行，混用加`cin.ignore()`。用`+`拼接，`size()`求长度。stoi和to_string做数字文字互转。stringstream解析字符串。

**第20章：string十八般武艺。** find找位置，substr截取，erase删除，insert插入，replace替换。字符函数isdigit/isalpha/toupper/tolower。两种遍历方式。

**第21章：ASCII世界。** bit→Byte→KB→MB→GB。ASCII三大规律：48、65、97。char本质是整数，`ch-'0'`转真数字，±32做大小写转换。二进制除2取余倒序排列。

**第22章：综合实战。** 容器选择看数据形状。矩阵旋转用坐标公式。螺旋填数用方向数组。字符串统计用inWord标记法。成绩排名用平行数组+捆绑交换。数据去重用排序+双指针。

这些知识不是孤立的——矩阵旋转用到了二维数组的遍历，成绩排名用到了数组的排序和string，螺旋填数用到了数组和循环。**综合题就是把学会的小工具按逻辑串起来。**
