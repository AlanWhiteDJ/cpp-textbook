## 📝 讲稿正文

同学们，前面三章我们把数字玩出了花来——int存整数、double存小数、数组存一堆数、二维数组存表格。但大家有没有想过一个问题——你的名字是什么类型？

"张三"——这可不是int。int只能存数字。"Hello World"——也不是char。char只能存一个字母。

我们需要一种新的数据类型，能存**一串字符**——字母、数字、符号、甚至中文。在C++里，这个类型叫**string**。

string的中文意思是"字符串"。你可以把它理解成"一串字符组成的序列"。比如：

```cpp
string name = "Alice";          // 一串英文字母
string greeting = "你好！";      // 中文也可以
string mixed = "Player2_Win!";  // 字母、数字、符号混在一起
```

你看，跟int对比一下：

- `int score = 95;` —— 只能存一个整数
- `char grade = 'A';` —— 只能存一个字符（注意char用单引号）
- `string name = "Alice";` —— 可以存一串字符（string用双引号）

char是"一颗糖"，string是"一包糖"。char是独生子，string是可以有很多孩子的大家庭。

好，声明string很简单，跟int差不多。但要注意——**用string需要`#include <string>`**。有些编译器不写也能用，但规范起见必须写。

string的初始化有多种花样，我们来看看：

```cpp
string s1;                       // 空字符串""——什么都没装
string s2 = "Hello";             // 常规初始化——跟int一样
string s3("World");              // 用括号也行——效果一样
string s4 = s2;                  // 复制另一个string——把s2的内容拷贝给s4
string s5(5, 'A');              // "AAAAA"——5个字符'A'
```

前面几种都比较好理解，最后一种`string(5, 'A')`是什么意思？就是"把字符A重复5遍"，结果等于"AAAAA"。

这个语法有什么用呢？一个很实用的场景——打印分隔线：

```cpp
cout << string(30, '-') << endl;  // 输出30个短横线
cout << string(20, '=') << endl;  // 输出20个等号
```

不用写循环，一行代码就能生成一条华丽的分隔线。这在做"名片生成器"、"菜单界面"的时候特别好用。

现在我们来比较一下string和其他类型：

| 类型 | 存什么 | 例子 | 大小 |
|------|--------|------|------|
| int | 整数 | 95 | 4字节 |
| double | 小数 | 3.14 | 8字节 |
| char | 单个字符 | 'A' | 1字节 |
| string | 一串字符 | "Hello" | 不定，按需 |

string的大小不是固定的——"Hi"占的空间比"Hello World"少。它就像一个"弹性口袋"，装多少就多大。

另外提醒大家，string里的中文字也算一个字符。`string s = "你好世界"`，`s.size()`会返回4（取决于编码实现），不是8。这个我们后面会详细讲。

## 🖥️ 板书/屏幕要点
- **string = 一串字符**（char = 一个字符）
- `char` 用单引号 `'A'`，`string` 用双引号 `"Hello"`
- 声明：`string 变量名;`（需 `#include <string>`）
- 初始化方式：
  - `string s = "Hello";`
  - `string s("Hello");`
  - `string s(5, 'A');` → "AAAAA"
  - `string(30, '-')` → 30个短横线（分隔线神器）
- string是"弹性口袋"——大小按需变化

## 📌 小结
string是C++专门用来处理文字的类型。它和char的关系就像"一包糖"和"一颗糖"。声明string需要include头文件，初始化有多种灵活的方式，其中`string(n, char)`用来生成重复字符特别实用。从今天开始，你的程序不仅能算数字，还能处理和展示文字了。
