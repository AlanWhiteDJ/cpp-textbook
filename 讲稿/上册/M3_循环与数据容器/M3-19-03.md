## 📝 讲稿正文

同学们，前两节课我们把string存起来、读进来了。现在我们来玩文字的"基本操作"——**拼接**和**量长度**。

先看拼接。在C++里，string可以用`+`直接把两段文字粘在一起，特别直观：

```cpp
string first = "Hello";
string second = "World";
string result = first + " " + second;  // "Hello World"
cout << result << endl;
```

这就像拼积木——"Hello"加一个空格加"World"，变成"Hello World"。比Java那种要用什么StringBuilder的方式简单多了。

除了`+`，还有`+=`——在原有文字后面追加：

```cpp
string s = "Hello";
s += " World";     // s现在变成"Hello World"
s += "!";          // 变成"Hello World!"
```

`+=`就是"在后面接上"的意思。比如你要构建一个欢迎语：

```cpp
string name = "小明";
string msg = "你好，";
msg += name;
msg += "！欢迎回来！";
cout << msg << endl;  // 你好，小明！欢迎回来！
```

好，拼接看起来很简单对吧？但有一个坑——**两边都是字面量的时候不能直接`+`**。

什么叫字面量？就是用双引号直接写的字符串，比如`"Hello"`、`"World"`。

```cpp
string s = "Hello" + "World";  // ❌ 编译错误！
```

为什么？因为`+`运算符要求至少一边是string对象。两边都是C风格字符串（字面量），编译器不知道怎么加。

怎么解决？

```cpp
string s = string("Hello") + "World";  // ✅ 先把一边变成string
// 或者：
string s = "Hello";
s += "World";                            // ✅ 也没问题
```

规则很简单：**`+`两边至少要有一个string变量或显式转换的string对象**。

好，拼接搞定。接下来看**长度**。

想知道一个字符串有几个字符？用`.size()`或`.length()`——两种写法完全等价。

```cpp
string s = "Hello";
cout << s.size() << endl;     // 5
cout << s.length() << endl;   // 5（跟size()一模一样）
```

长度在很多场景下都用得到：

```cpp
if (s.size() == 0)  cout << "空字符串" << endl;
if (s.size() > 10)  cout << "名字好长！" << endl;
if (name.size() >= 3 && name.size() <= 16) cout << "用户名合法" << endl;
```

比如检查用户名长度是否在3到16个字符之间——这在注册页面里很常见。

注意，`size()`返回的不是int，而是一个叫`size_t`的无符号整数类型。对于现阶段的学习，你把它当int用基本没问题。但如果编译器给出warning，可以用`(int)s.size()`强制转换。

另外，中文字也算一个字符：

```cpp
string s = "你好世界";
cout << s.size() << endl;  // 输出取决于编码，通常UTF-8下是12
```

这个跟编码有关，不同环境下可能不一样。对于竞赛来说，标准输入的中文一般用英文字符处理即可，重点掌握英文字符串的长度判断。

## 🖥️ 板书/屏幕要点
- **拼接**：`s1 + s2`、`s1 + "文字"`、`string("A") + "B"`
- **追加**：`s += "文字";`（在末尾接上）
- ⚠️ 不能 `"A" + "B"` ——两边都是字面量
- ✅ `string("A") + "B"` ——至少一边是string对象
- **长度**：`s.size()` 或 `s.length()`（等价）
- 空字符串判断：`s.size() == 0`
- 长度验证：`if (s.size() >= 3 && s.size() <= 16)`

## ✏️ 随堂练习
写一个程序：询问用户的姓（string）和名（string），用`+`拼接成全名（中间加空格），用`size()`输出全名的总字符数（不含空格），判断全名长度是否在2~20之间，输出判断结果。例如输入"张 小美"→ 全名"张小美"（3字符），输出"3 合法"。

## 📌 小结
string的拼接就像搭积木——用`+`和`+=`把文字粘在一起。但要记住至少一边是string对象，两边都是字面量不行。`size()`和`length()`获取字符串长度，常用于长度验证。这两个操作是string日常使用的基础，后面学的查找、截取、替换都要依赖它们。
