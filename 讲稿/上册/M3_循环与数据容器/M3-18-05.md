## 📝 讲稿正文

同学们，今天是我们第18章的最后一节课。前面我们学了二维数组的声明、遍历、转置、对角线、函数传参——现在该来打一场"实战"了。

第一个实战项目——**杨辉三角**。这是中国古代数学的瑰宝，在西方叫"帕斯卡三角"。它的规则很简单：每行第一个和最后一个元素是1，其他元素等于它"左上角"加上"正上方"两个数之和。

用二维数组来实现的话，`a[i][j] = a[i-1][j-1] + a[i-1][j]`。注意，j从1开始到i-1，因为每行两端是1不需要计算。

```cpp
int a[10][10] = {0};  // 全部初始化为0

// 生成杨辉三角前10行
for (int i = 0; i < 10; i++) {
    a[i][0] = 1;          // 每行第一个是1
    a[i][i] = 1;          // 每行最后一个也是1
    for (int j = 1; j < i; j++) {
        a[i][j] = a[i-1][j-1] + a[i-1][j];
    }
}

// 输出
for (int i = 0; i < 10; i++) {
    for (int j = 0; j <= i; j++) {  // 只需要输出到第i列
        cout << a[i][j] << " ";
    }
    cout << endl;
}
```

我们来仔细分析一下。外层循环i是行号，从0到9。每行先设第一个和最后一个为1。但注意——当i=0时，a[0][0]被设了两次1，没关系。当i=1时，a[1][0]=1, a[1][1]=1，内层循环j从1到0——循环条件是j < 1，所以一次都不执行，正确。

当i=2时，a[2][0]=1, a[2][2]=1，然后j=1执行一次：a[2][1] = a[1][0] + a[1][1] = 1 + 1 = 2。输出第二行是"1 2 1"。完美！

大家注意输出循环——`j <= i`而不是`j < 10`。因为第i行只有i+1个有效元素，后面的都是0，不需要输出。

好，杨辉三角搞定了。现在来看第二个实战——**矩阵加法**。

两个矩阵相加，就是对应位置的元素相加：`C[i][j] = A[i][j] + B[i][j]`。前提是两个矩阵大小相同。

```cpp
int A[3][3] = { {1,2,3}, {4,5,6}, {7,8,9} };
int B[3][3] = { {9,8,7}, {6,5,4}, {3,2,1} };
int C[3][3];

for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        C[i][j] = A[i][j] + B[i][j];
    }
}

// 输出C
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        cout << C[i][j] << " ";
    }
    cout << endl;
}
// 输出：10 10 10 / 10 10 10 / 10 10 10
```

A和B互为"互补矩阵"——每个对应位置的和都是10。这个结果验证了我们的代码正确。

现在我们来做本章的总复习。二维数组这一章，你掌握了什么？

- 声明和初始化——`int a[3][4] = {0}`一键清零
- 双重循环遍历——外层行、内层列
- 行列求和——外层在哪求哪就固定哪个维度
- 转置——`b[j][i] = a[i][j]`
- 对角线——主对角线i==j，副对角线i+j==n-1
- 边界遍历——四个条件取或
- 函数传参——必须指定列数
- 杨辉三角——递推公式`a[i][j] = a[i-1][j-1] + a[i-1][j]`

这些技能组合起来，已经能解决很多"表格型"问题了。现在来做随堂练习——输入一个矩阵，求每行每列的和。

## 🖥️ 板书/屏幕要点
- **杨辉三角**：
  - 规则：两端为1，中间=左上+正上
  - 代码：`a[i][j] = a[i-1][j-1] + a[i-1][j]`
  - 输出：`j <= i`（只输出有效部分）
- **矩阵加法**：`C[i][j] = A[i][j] + B[i][j]`
- 18章技能总览：声明/遍历/转置/对角线/边界/传参/杨辉

## 📌 小结
杨辉三角是经典的递推型二维数组问题——每个元素依赖上一行的相邻两个元素。矩阵加法是基本的矩阵运算——对应位置相加。通过这两个实战，我们巩固了二维数组的核心操作模式。第18章到此结束——从一维到二维，你的数据处理能力上了一个台阶。下一章，我们将进入一个全新的领域——文字处理，string正式登场！
