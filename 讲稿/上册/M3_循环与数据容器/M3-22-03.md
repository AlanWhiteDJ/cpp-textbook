## 📝 讲稿正文

同学们，上节课我们学了矩阵旋转——用坐标公式直接转换。今天我们来学一个更有趣的二维数组经典题——**螺旋填数**。

题目：给定n，在n×n的矩阵中，从左上角开始，顺时针螺旋填入1到n²。

以n=4为例，结果应该长这样：

```
 1  2  3  4
12 13 14  5
11 16 15  6
10  9  8  7
```

这个怎么填？你不能用简单的双重循环——因为行列不是按顺序走的。你需要**模拟"走迷宫"的过程**。

思路是这样的：从(0,0)开始，先向右走，碰到边界或已经填过的格子就右转（顺时针），再继续走……直到填完n²个数。

这里需要用到今天的新武器——**方向数组**。

```cpp
int dx[] = {0, 1, 0, -1};  // 右、下、左、上（x行变化）
int dy[] = {1, 0, -1, 0};  // 右、下、左、上（y列变化）
```

dx和dy是配对的。dir=0代表向右：dx=0（行不变），dy=1（列+1）。dir=1代表向下：dx=1（行+1），dy=0（列不变）。dir=2向左，dir=3向上。

方向数组的好处是——你不需要写四个方向的if-else，只要用一个dir变量加这两行数组就能统一处理。

好，来看完整代码：

```cpp
int n;
cin >> n;
int a[20][20] = {0};

int dx[] = {0, 1, 0, -1};  // 行变化
int dy[] = {1, 0, -1, 0};  // 列变化
int dir = 0;                // 当前方向：0右 1下 2左 3上
int x = 0, y = 0;           // 当前位置

for (int k = 1; k <= n * n; k++) {
    a[x][y] = k;            // 填当前格子

    int nx = x + dx[dir];   // 按当前方向走一步
    int ny = y + dy[dir];

    // 如果下一步出界或已经填过，就转弯
    if (nx < 0 || nx >= n || ny < 0 || ny >= n || a[nx][ny] != 0) {
        dir = (dir + 1) % 4;    // 顺时针转90度
        nx = x + dx[dir];       // 重新算下一步
        ny = y + dy[dir];
    }
    x = nx;
    y = ny;
}

// 输出矩阵（对齐打印）
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        printf("%3d", a[i][j]);  // 占3格对齐
    }
    cout << endl;
}
```

我们来推演前几步。n=4：

- k=1：a[0][0]=1。nx=0+0=0, ny=0+1=1。没出界，a[0][1]=0没填过。x=0,y=1。
- k=2：a[0][1]=2。nx=0, ny=2。没出界。x=0,y=2。
- k=3：a[0][2]=3。nx=0, ny=3。x=0,y=3。
- k=4：a[0][3]=4。nx=0, ny=4——出界了！(ny>=4)。转弯！dir变成1（向下）。nx=0+1=1, ny=3+0=3。x=1,y=3。
- k=5：a[1][3]=5。nx=2, ny=3。x=2,y=3。
- ……以此类推

这个算法的精髓在于**"先走一步看看，不行就转弯"**。就像你在黑暗中摸索——先伸脚试探，碰到墙就转向。

方向数组`(dir+1)%4`实现顺时针转向——0变1，1变2，2变3，3变0，循环。

另外注意，输出时用了`printf("%3d", a[i][j])`——让每个数字占3格宽度，不管是一位数还是两位数都能对齐。这是一个输出格式的小技巧。

现在来做练习——把n改成5，观察输出。再试试逆时针螺旋填数（只需改方向数组的顺序和转弯方向）。

## 🖥️ 板书/屏幕要点
- **螺旋填数核心**：模拟走迷宫 + 方向数组
- **方向数组**：`dx[]={0,1,0,-1}, dy[]={1,0,-1,0}`（右下左上）
- **转弯条件**：出界(`nx<0||nx>=n||ny<0||ny>=n`) 或 已填(`a[nx][ny]!=0`)
- **转弯公式**：`dir = (dir + 1) % 4`（顺时针90度）
- 输出对齐：`printf("%3d", a[i][j])`
- 逆时针：改方向数组顺序、改转弯方向即可

## 📌 小结
螺旋填数是"模拟+数组"的经典结合。方向数组让四个方向的切换变得优雅——一个dir变量加上dx/dy数组，替代了冗长的if-else。核心思想是"先试探下一步，不行就转弯"。这个"试探+转向"的模式在走迷宫、棋盘遍历等模拟题中都会用到。
