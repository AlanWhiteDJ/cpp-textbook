## 📝 讲稿正文

同学们，上节课我们学会了char本质是整数，能用算术做大小写转换和数字提取。但每次都写`ch-32`或`ch-'0'`，代码可读性不太好——别人一看"减32是啥意思？"

C++标准库给我们提供了一套**字符处理函数**，把底层算术包装成了语义清晰的名字。这些函数在`<cctype>`头文件里。

我们来看两个维度——**判断函数**和**转换函数**。

**【判断函数】** —— 回答"这个字符是什么类型的？"

| 函数 | 问题 | 底层逻辑 |
|------|------|----------|
| `isdigit(c)` | 是数字吗？ | 48 ≤ c ≤ 57 |
| `isalpha(c)` | 是字母吗？ | (65 ≤ c ≤ 90) 或 (97 ≤ c ≤ 122) |
| `isupper(c)` | 是大写吗？ | 65 ≤ c ≤ 90 |
| `islower(c)` | 是小写吗？ | 97 ≤ c ≤ 122 |
| `isspace(c)` | 是空白吗？ | 空格、换行、制表符等 |

每个函数返回一个int——非零表示"是"，0表示"否"。所以在if里可以直接用。

来看代码：

```cpp
#include <cctype>
#include <iostream>
using namespace std;

int main() {
    char c = 'b';
    cout << isalpha(c) << endl;   // 非零——'b'是字母
    cout << isdigit(c) << endl;   // 0——'b'不是数字
    cout << islower(c) << endl;   // 非零——'b'是小写

    if (isalpha(c)) {
        cout << c << "是字母" << endl;
    }
    return 0;
}
```

现在你知道这些函数底层在干嘛了吗？`isdigit(c)`本质上就是`return c >= '0' && c <= '9'`。`isupper(c)`就是`return c >= 'A' && c <= 'Z'`。它们只是把ASCII范围判断包装成了好读的函数名。

**【转换函数】** —— 大小写互转。

| 函数 | 作用 | 底层逻辑 |
|------|------|----------|
| `toupper(c)` | 转大写 | 如果小写(97~122)，减32；否则不变 |
| `tolower(c)` | 转小写 | 如果大写(65~90)，加32；否则不变 |

好，这里有一个**必须再强调一遍的坑**：

```cpp
string s = "hello";
for (int i = 0; i < s.size(); i++) {
    toupper(s[i]);        // ❌ 没用！没赋值回去！
}
cout << s << endl;         // 还是"hello"
```

`toupper(s[i])`做了转换，返回了新字符，但你没有把返回值存回s[i]！这就像你算了一道题但没写答案。

正确写法：

```cpp
for (int i = 0; i < s.size(); i++) {
    s[i] = toupper(s[i]);   // ✅ 把返回值赋回去
}
cout << s << endl;           // "HELLO"
```

那为什么在cout里直接用就没问题？因为`cout << toupper('a')`会输出`A`——cout直接用了toupper的返回值，不需要存回变量。但如果你要修改字符串，就必须赋值。

还有一个实用的组合——**判断+转换**。比如"把字符串里的小写字母转大写，其他不变"：

```cpp
string s = "Hello World 123!";
for (char &c : s) {
    if (islower(c)) {
        c = toupper(c);
    }
}
cout << s << endl;  // "HELLO WORLD 123!"
```

用范围for循环加引用，代码简洁又清晰。

现在来做练习——统计用户输入的一行文字中有多少个字母、数字、空格，并把字母全部转成大写输出。

## 🖥️ 板书/屏幕要点
- 头文件：`#include <cctype>`
- **判断函数**：isdigit/isalpha/isupper/islower/isspace
- **转换函数**：toupper/tolower
- ⚠️ toupper/tolower**返回新值**，修改原字符必须`s[i] = toupper(s[i])`
- 底层逻辑：ASCII范围判断 + ±32运算
- 范围for修改：`for (char &c : s) c = toupper(c);`

## ✏️ 随堂练习
用户输入一个字符串，完成：①统计其中大写字母、小写字母、数字、空格的数量；②将字符串中的所有大写字母转小写、小写字母转大写（即大小写互换），数字和其他字符不变；③输出统计结果和转换后的字符串。

## 📌 小结
字符处理函数是ASCII知识的标准化封装——判断函数帮你识别字符类型，转换函数帮你做大小写互转。它们底层的逻辑就是ASCII范围判断和±32运算。记住toupper/tolower必须赋值回去，这是使用频率最高的陷阱。这些函数让代码更可读——`isalpha(c)`比`c>='A'&&c<='Z'||c>='a'&&c<='z'`优雅得多。
