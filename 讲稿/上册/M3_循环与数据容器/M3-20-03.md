# 第20章 第3课时：erase与insert——删除与插入的文字手术
**时长**：15-20分钟 | **对应教材**：第20章 字符串的十八般武艺

## 🎯 本课时目标
- 掌握erase的三种用法：删除一段、删到末尾、全部删除
- 掌握insert在指定位置插入文字
- 能组合erase和insert完成文字修改

## 📝 讲稿正文

同学们，前面两节课我们学了find（找到位置）和substr（剪下一段）。今天我们来学两个"真刀真枪"的修改操作——**erase（删除）**和**insert（插入）**。这两个操作直接改变原字符串，是真正的"文字手术"。

先看erase。它就像橡皮擦，擦掉字符串中的某一段。

```cpp
string s = "Hello Beautiful World";

s.erase(5, 10);                    // 从下标5开始，删10个字符
cout << s << endl;                 // "Hello World"

s.erase(5);                        // 从下标5开始，删到末尾
cout << s << endl;                 // "Hello"

s.erase();                         // 不写参数，全部删除
cout << s << endl;                 // ""
```

erase有三种形式，我们来逐个分析。

**形式一：`s.erase(起始位置, 删除长度)`。** 从指定位置开始，删掉指定个数的字符。比如`"Hello Beautiful World".erase(5, 10)`——从位置5（第一个空格后）开始，删掉10个字符" Beautiful"，剩下"Hello World"。

**形式二：`s.erase(起始位置)`。** 只写一个参数，从指定位置删到字符串末尾。`"Hello World".erase(5)`——从位置5删到末尾，剩下"Hello"。这种形式在"截断字符串"时很实用——保留前n个字符，后面的全删。

**形式三：`s.erase()`。** 不写参数，清空整个字符串。相当于`s = ""`。

erase一个很实用的场景——**去掉字符串开头的空格**：

```cpp
string s = "   Hello";
while (s.size() > 0 && s[0] == ' ') {
    s.erase(0, 1);  // 每次删掉第一个字符（空格）
}
cout << "[" << s << "]" << endl;  // [Hello]
```

循环检查第一个字符是不是空格，是就删掉。每次只删一个，直到第一个字符不是空格为止。

好，erase搞定了。接下来看**insert**——在指定位置插入一段文字。

```cpp
string s = "Hello World";

s.insert(5, " Beautiful");         // 在下标5处插入
cout << s << endl;                 // "Hello Beautiful World"

s.insert(0, "Say: ");              // 在开头插入
cout << s << endl;                 // "Say: Hello Beautiful World"
```

insert的格式是`s.insert(在哪插, 插什么)`。注意——它在指定位置**之前**插入。比如`s.insert(5, " Beautiful")`，在位置5（第一个空格）之前插入" Beautiful"，原来的内容整体后移。

在开头插入就写`s.insert(0, ...)`，在末尾插入就写`s.insert(s.size(), ...)`——不过末尾追加用`+=`更方便。

insert和数组的插入操作在思想上是一样的——把后面的内容往后挪，腾出空间放新内容。只不过string帮你做好了这一切，你只需要说"在哪插、插什么"，不用手动移动。

erase和insert可以组合使用——先删掉一段，然后在同一个位置插一段新的。这其实就是我们下节课要学的**replace**。

## 🖥️ 板书/屏幕要点
- **erase三种形式**：
  - `s.erase(pos, len)` → 删pos起的len个字符
  - `s.erase(pos)` → 从pos删到末尾
  - `s.erase()` → 清空全部
- **insert**：`s.insert(pos, str)` → 在pos处插入str
- 去开头空格：循环`erase(0, 1)`
- 去末尾空格：循环`erase(s.size()-1, 1)`
- erase + insert = 先删后插（replace的底层逻辑）

## 📌 小结
erase是橡皮擦——指定位置和长度，擦掉一段内容。insert是胶水——指定位置和要插入的文字，塞进去。它们配合使用就能实现"先删后插"的替换效果。string的编辑功能越来越完整了——find定位、substr截取、erase删除、insert插入，下节课我们用replace把这几个操作合成一招。
