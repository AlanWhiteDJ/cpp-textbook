## 📝 讲稿正文

同学们，上节课我们知道了计算机只认0和1。那它怎么存字母呢？答案很简单——给每个字符分配一个数字编号。这套编号规范，就叫**ASCII码**。

ASCII的全称是American Standard Code for Information Interchange——美国信息交换标准代码。名字很长，但核心就一句话：**每个英文字符对应一个0到127之间的数字**。

我们来看这张表里最重要的三组数字：

| 字符 | ASCII值 | 字符 | ASCII值 | 字符 | ASCII值 |
|------|---------|------|---------|------|---------|
| '0' | 48 | 'A' | 65 | 'a' | 97 |
| '1' | 49 | 'B' | 66 | 'b' | 98 |
| '2' | 50 | 'C' | 67 | 'c' | 99 |
| ... | ... | ... | ... | ... | ... |
| '9' | 57 | 'Z' | 90 | 'z' | 122 |

仔细观察这张表，你会发现**三大规律**：

**规律一：数字字符连续。** '0'到'9'的ASCII码是48到57，连续递增。所以`'5' - '0'`等于`53 - 48 = 5`——字符数字减去'0'就得到了真正的数值。

**规律二：大写字母连续。** 'A'到'Z'的ASCII码是65到90。所以`'C' - 'A'`等于`67 - 65 = 2`——'C'是字母表里的第2个（从0开始数）。

**规律三：小写字母连续。** 'a'到'z'的ASCII码是97到122。同样连续。

更重要的规律——**小写字母比对应的大写字母大32**。

```cpp
'a' - 'A' = 97 - 65 = 32
'b' - 'B' = 98 - 66 = 32
'z' - 'Z' = 122 - 90 = 32
```

这就是大小写转换的底层原理！`toupper('a')`做的事情就是`'a' - 32 = 97 - 32 = 65 = 'A'`。

好，我们来看一段验证代码：

```cpp
for (char c = 'A'; c <= 'Z'; c++) {
    cout << c << "=" << (int)c << "  ";
}
cout << endl;

for (char c = 'a'; c <= 'z'; c++) {
    cout << c << "=" << (int)c << "  ";
}
cout << endl;

for (char c = '0'; c <= '9'; c++) {
    cout << c << "=" << (int)c << "  ";
}
```

运行这段代码，你会亲眼看到：'A'到'Z'是65到90，'a'到'z'是97到122，'0'到'9'是48到57。三大规律一目了然。

这三个数字请背下来：**48、65、97**。它们分别是'0'、'A'、'a'的起点。记住这三个数字，你就等于记住了整个ASCII表的核心。

那ASCII码一共多少个？0到127，共128个。其中：
- 0~31：控制字符（如换行、回车、制表符——不可打印）
- 32~126：可打印字符（空格、数字、大小写字母、标点符号）
- 127：DEL（删除字符）

后来为了支持更多语言，ASCII扩展到了0~255（扩展ASCII），多出来的128个位置用来存法语、德语等欧洲语言的字符。

至于中文——中文有几万个字符，128个位置远远不够。所以中文用GBK、UTF-8等更"大"的编码方式，通常一个汉字占2~4个Byte。

现在来做个小练习。写一段程序，输出大写字母A到Z，每个字母后面跟它的ASCII值。看看是不是65到90。

## 🖥️ 板书/屏幕要点
- **ASCII码** = 字符的数字编号（0~127）
- **三大规律**：
  - '0'~'9' → 48~57
  - 'A'~'Z' → 65~90
  - 'a'~'z' → 97~122
- **核心记忆点**：48、65、97
- 'a' - 'A' = 32（大小写转换的数学基础）
- `(int)字符` → 查看它的ASCII值
- `(char)数字` → 查看数字对应的字符

## ✏️ 随堂练习
写程序分别输出大写字母A~Z、小写字母a~z、数字0~9各字符及其ASCII值。验证三大规律。然后手动计算：'M'的ASCII值是多少？'m'的ASCII值是多少？（不查表，用规律推算）

## 📌 小结
ASCII码就是字符和数字之间的翻译表。三大规律是核心——数字从48开始，大写从65开始，小写从97开始，大小写差32。记住这三个起点数字（48、65、97），你就掌握了ASCII码的精髓。这也是为什么C++里char能当int用——因为字符本质上就是一个整数。
