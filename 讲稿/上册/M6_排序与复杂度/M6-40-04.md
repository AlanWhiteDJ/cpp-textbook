## 📝 讲稿正文

竞赛里有一类题出现频率特别高——**排行榜**。给你一堆人的姓名、各科成绩，让你按某些规则排名。这类题的核心就是：**结构体 + 多关键字 cmp**。

来看一道典型例题。有 n 个学生，每人有姓名、语文成绩、数学成绩。排名规则是：先按总分从高到低排；总分相同按语文成绩从高到低；还相同按姓名从小到大（字典序）。

我们来一步步拆解。首先定义结构体：

```cpp
struct Student {
    string name;
    int chinese, math;
    int total() const { return chinese + math; }
};
```

注意我在结构体里直接写了一个 `total()` 函数来算总分——这样 cmp 里调用 `a.total()` 就行了，不用每次手动加。

然后是 cmp 函数：

```cpp
bool cmp(const Student &a, const Student &b) {
    if (a.total() != b.total())
        return a.total() > b.total();     // 总分高的在前 → 降序用 '>'
    if (a.chinese != b.chinese)
        return a.chinese > b.chinese;     // 语文高的在前 → 降序用 '>'
    return a.name < b.name;               // 姓名字典序 → 升序用 '<'
}
```

大家注意看比较方向：总分和语文是**降序**（高的排前面），所以用 `>`；姓名是**升序**（字典序小的排前面），所以用 `<`。这个 `<` 和 `>` 的方向非常关键，搞反了排名就全错了。

主程序就很简单了：

```cpp
int main() {
    Student s[100];
    int n;
    cin >> n;
    for (int i = 0; i < n; i++)
        cin >> s[i].name >> s[i].chinese >> s[i].math;

    sort(s, s + n, cmp);

    for (int i = 0; i < n; i++)
        cout << s[i].name << " " << s[i].total() << endl;
    return 0;
}
```

这道题就是结构体排序的经典模板。以后碰到任何排行榜问题，套路都一样：**定义结构体 → 写 cmp → `sort(s, s+n, cmp)`**。

再来一个变式——**年龄排序**。有 n 个人，每人有出生年月日和姓名。按年龄从大到小排（即出生早的在前，也就是出生年份小的在前）。年龄完全相同按姓名升序。

```cpp
struct Person {
    string name;
    int year, month, day;
};

bool cmp(const Person &a, const Person &b) {
    if (a.year != b.year) return a.year < b.year;       // 出生早(年小)在前
    if (a.month != b.month) return a.month < b.month;
    if (a.day != b.day) return a.day < b.day;
    return a.name < b.name;
}
```

注意这里出生年份越小 → 年纪越大 → 排在前面。三个日期字段逐级比较，最后比较姓名。

好，给大家总结一下排行榜题的通用模板。cmp 的结构就是：

```cpp
if (第一关键字不相等) return 第一关键字的比较;
if (第二关键字不相等) return 第二关键字的比较;
...
return 最后关键字的比较;
```

升序用 `<`，降序用 `>`。每个关键字只写一行，一行搞定一个层级，简单、清晰、不容易出错。

## 🖥️ 板书/屏幕要点
- **排行榜 = 结构体 + cmp + sort**
- **cmp 多关键字模板**：
  ```cpp
  if (a.k1 != b.k1) return 升/降序 ? (a.k1 < b.k1) : (a.k1 > b.k1);
  if (a.k2 != b.k2) return ...;
  return ...;
  ```
- **方向口诀**：升序用 `<`，降序用 `>`
- **结构体内函数**：`int total() const { return ...; }` 可以在 cmp 里直接用

## 📌 小结
排行榜题是竞赛排序的"主食"。结构体定义清楚，cmp 按层级写，每个关键字一行 if-return，就能稳稳拿分。下节课我们学 `stable_sort` 和排序题的常见模式总结。
