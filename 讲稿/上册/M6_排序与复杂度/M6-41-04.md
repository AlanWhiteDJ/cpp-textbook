## 📝 讲稿正文

上节课我们理解了 partition 的原理。今天把完整的快速排序代码写出来，并解决最坏 O(n²) 的问题。

先看完整的递归框架：

```cpp
void quickSort(int l, int r) {
    if (l >= r) return;                // 终止条件
    int p = partition(l, r);           // 分区，p 是基准的最终位置
    quickSort(l, p - 1);               // 递归排左边
    quickSort(p + 1, r);               // 递归排右边
}
```

很简单，对吧？分区完后基准在位置 p，左边 `[l, p-1]` 都 ≤ 基准，右边 `[p+1, r]` 都 ≥ 基准。两边各自递归就行。

现在关键问题来了——**怎么选基准**？如果每次选最右边元素，对已排序数组 `[1,2,3,4,5]` 会退化到 O(n²)。我们必须避免这个问题。

最常用的解法是**随机化**——在 `[l, r]` 范围内随机选一个位置，把那个元素和最右边的元素交换，然后再以最右边为基准。这样一来，每次 p 的位置是随机的，期望上是 O(n log n)。

```cpp
#include <cstdlib>   // rand()
#include <ctime>     // time()

int partition(int l, int r) {
    // 随机选基准，与最右元素交换
    int pivotIdx = l + rand() % (r - l + 1);
    swap(a[pivotIdx], a[r]);

    int pivot = a[r];
    int i = l - 1;
    for (int j = l; j < r; j++) {
        if (a[j] <= pivot) {
            i++;
            swap(a[i], a[j]);
        }
    }
    swap(a[i + 1], a[r]);
    return i + 1;
}
```

主程序中需要先初始化随机种子：

```cpp
int main() {
    srand(time(0));   // 用当前时间作为随机种子
    // ... 读入数据 ...
    quickSort(0, n - 1);
    // ...
}
```

`rand() % (r - l + 1)` 产生一个 0 到 `r-l` 的随机数，加上 l 就映射到了 `[l, r]` 范围。然后把选中的值和 a[r] 交换——后面的 partition 逻辑完全不用改。

除了随机化，还有一种叫**三数取中法**：取 `a[l]`、`a[mid]`、`a[r]` 的中位数作为基准。这个比纯随机更稳定，但代码多一点。竞赛中随机化已经足够好了。

好，现在看 partition 里的另一个细节。我在 if 条件里用了 `a[j] <= pivot`，写的是 `<=` 而不是 `<`。为什么？如果用 `<`，等于 pivot 的元素全部被分在右边不动，在某些情况下（比如所有元素相等），分区会极度不均匀——每次都是 0 和 n−1。用 `<=` 把等于基准的元素也交换到左边一些，左右更均衡。

至于快排的空间复杂度，大家要注意——虽然它没有像归并那样用 O(n) 的临时数组，但**递归本身占用的栈空间**也是要算的。递归深度在平均情况下是 O(log n)，所以空间复杂度是 O(log n)。最坏情况下（如退化时）递归深度是 O(n)，这也是随机化的另一个好处。

最后说一句——在正常的竞赛中，**用 `sort()` 就行**，不需要手写快排。但理解快排的 partition 思想在解决"第 k 小"、"荷兰国旗"等问题时非常有用。而且有些题目就是让你手写快排的某个变种，比如快速选择算法。

## 🖥️ 板书/屏幕要点
- **quickSort 框架**：终止条件 → partition → 递归左右
- **随机化**：`pivotIdx = l + rand() % (r-l+1)`，`swap(a[pivotIdx], a[r])`
- **随机种子**：`srand(time(0))` 放在 main 开头
- **`<=` vs `<`**：`<=` 让分区更均衡
- **空间复杂度**：O(log n)（递归栈），随机化保证不退化

## 📌 小结
快排的完整实现包含两个函数：partition 分区和 quickSort 递归，加上随机化选基准来避免最坏情况。记住随机种子要设、终止条件要写。下节课我们把归并和快排放在一起全面对比。
