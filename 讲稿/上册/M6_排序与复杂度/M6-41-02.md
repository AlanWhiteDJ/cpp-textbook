## 📝 讲稿正文

上节课我们理解了归并排序的"分治"思想。今天来写代码，把它变成可以运行的程序。

归并排序需要两个函数。第一个是核心——**`merge()` 合并函数**：把两个已经各自有序的子数组合并成一个有序数组。第二个是主函数——**`mergeSort()`**：负责递归地划分和调用 merge。

我们先看 merge 的实现。假设数组 a 已经有两个有序的部分——`a[l..mid]` 和 `a[mid+1..r]`。我们需要把它们合并成一个从 l 到 r 整体有序的数组。

这里有一个关键技术叫**双指针归并**。我们用 i 指向左半当前元素，j 指向右半当前元素，k 指向临时数组的当前位置。每次比较 a[i] 和 a[j]，谁小就把谁放进 tmp[k]，然后对应的指针往后移。当一个半走完了，把另一半剩下的元素直接拷贝过去。

```cpp
const int MAXN = 100005;
int a[MAXN], tmp[MAXN];          // tmp 是合并用的临时数组

void merge(int l, int mid, int r) {
    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j])       // 用 <= 保证稳定性
            tmp[k++] = a[i++];
        else
            tmp[k++] = a[j++];
    }
    while (i <= mid) tmp[k++] = a[i++];   // 左边有剩余，直接抄
    while (j <= r)   tmp[k++] = a[j++];   // 右边有剩余，直接抄
    for (int p = l; p <= r; p++)          // 拷回原数组！
        a[p] = tmp[p];
}
```

注意，最后的 `for (int p = l; p <= r; p++) a[p] = tmp[p];` 千万不能忘！很多同学归并排了半天发现数组没变——就是忘了拷回去。因为上层递归用的是原数组 a，你必须把合并结果同步回 a。

mergeSort 递归函数就相对简单了：

```cpp
void mergeSort(int l, int r) {
    if (l >= r) return;                   // 只剩 1 个或 0 个,天然有序
    int mid = l + (r - l) / 2;            // 防止 (l+r) 溢出
    mergeSort(l, mid);                    // 递归排序左半
    mergeSort(mid + 1, r);                // 递归排序右半
    merge(l, mid, r);                     // 合并！
}
```

这里有两个注意点。第一，`if (l >= r) return;` 是递归终止条件——区间里只剩 1 个或 0 个元素，不需要排序。忘了写这个就会无限递归，爆栈崩溃。

第二，计算 mid 时用的是 `l + (r - l) / 2` 而不是 `(l + r) / 2`。虽然数学上一样，但 `l + r` 在 l 和 r 都很大时可能**溢出 int 范围**！用 `l + (r - l) / 2` 就不会有这个问题，是一个安全的好习惯。

完整调用方式：

```cpp
int main() {
    int n;
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    mergeSort(0, n - 1);
    for (int i = 0; i < n; i++) cout << a[i] << " ";
    return 0;
}
```

好，现在复杂度分析：每次递归把问题规模缩小一半，递归深度是 log₂n。在每一层，所有的 merge 合起来刚好遍历整个数组一遍，开销是 O(n)。所以总复杂度 = O(n) × O(log n) = O(n log n)，**无论什么数据都一样**。空间复杂度是 O(n)，因为需要开一个和原数组等大的 tmp 临时数组。

另外，`a[i] <= a[j]` 的那个 `<=` 也很重要——它保证了当两个元素相等时，左边的先进入结果，保持了稳定性。如果写成 `<`，稳定性就没了。归并排序是**稳定**的。

## 🖥️ 板书/屏幕要点
- **两个函数**：`merge(l, mid, r)` 合并，`mergeSort(l, r)` 递归划分
- **双指针归并**：i 指左半，j 指右半，k 指 tmp，谁小放谁
- **三个 while**：归并 + 收尾左 + 收尾右
- **容易忘的**：最后把 tmp 拷回 a！终止条件 `l >= r`！
- **复杂度**：O(n log n)，空间 O(n)，✅ 稳定

## 📌 小结
归并排序的代码有两个关键：merge 的双指针归并（三个 while），以及 mergeSort 的递归分治。记住别忘了拷回和终止条件。下节课我们学另一种分治排序——快速排序，它的思路更有趣。
