## 📝 讲稿正文

上节课我们理解了基数排序的原理——逐位排序，从低位到高位。今天来把它变成可以运行的代码。

基数排序由两部分组成：一个"按某一位排序"的计数排序函数，和一个外层循环按位依次处理。

先看按位排序函数：

```cpp
// 对 a[] 中 n 个元素，按第 exp 位（1=个位, 10=十位, 100=百位...）做稳定计数排序
void countSortByDigit(int a[], int n, int exp) {
    int output[n];                       // 输出数组
    int cnt[10] = {0};                   // 每位只有 0~9，10 个桶
    
    // 1. 统计当前位数字的出现次数
    for (int i = 0; i < n; i++)
        cnt[(a[i] / exp) % 10]++;        // 提取第 exp 位的数字
    
    // 2. 前缀和
    for (int i = 1; i < 10; i++)
        cnt[i] += cnt[i - 1];
    
    // 3. 从后往前放置（保证稳定）
    for (int i = n - 1; i >= 0; i--) {
        int digit = (a[i] / exp) % 10;
        output[cnt[digit] - 1] = a[i];
        cnt[digit]--;
    }
    
    // 4. 拷回原数组
    for (int i = 0; i < n; i++)
        a[i] = output[i];
}
```

提取某一位数字的公式：**`(a[i] / exp) % 10`**。当 exp=1 时，就是 `(a[i]/1)%10 = a[i]%10`，个位。exp=10 时，`(a[i]/10)%10`，十位。exp=100 时，百位。非常好用。

注意 cnt 数组大小只有 10——不管数据多大，每一位只有 0~9 十种可能。这就是节省空间的地方。

现在外层循环就简单了——从个位开始，每次 exp 乘以 10，直到最大值的所有位都处理完：

```cpp
void radixSort(int a[], int n) {
    // 找最大值，确定有多少位
    int maxVal = a[0];
    for (int i = 1; i < n; i++)
        if (a[i] > maxVal) maxVal = a[i];
    
    // 从个位到最高位，逐位排序
    for (int exp = 1; maxVal / exp > 0; exp *= 10)
        countSortByDigit(a, n, exp);
}
```

`maxVal / exp > 0` 这个条件很巧妙。比如 maxVal=802，当 exp=1 时 802/1=802 > 0，继续；exp=10 时 802/10=80 > 0，继续；exp=100 时 802/100=8 > 0，继续；exp=1000 时 802/1000=0，停止。正好处理了个位、十位、百位三轮。

完整测试：

```cpp
int main() {
    int a[] = {170, 45, 75, 90, 802, 24, 2, 66};
    int n = 8;
    
    radixSort(a, n);
    
    for (int i = 0; i < n; i++)
        cout << a[i] << " ";
    // 输出：2 24 45 66 75 90 170 802
    return 0;
}
```

复杂度分析：外层循环执行 d 轮（d 是位数），每轮 O(n)（遍历 n 个元素统计+放置）+ O(10)（前缀和，可忽略）。总复杂度 O(d × n)。对 int 类型，d ≤ 10，近似 O(n)。

空间复杂度：每轮需要一个 output[n] 和 cnt[10]，总体 O(n + 10) = O(n)。

需要注意的局限：以上实现只能处理**非负整数**。如果要处理负数，需要先整体偏移（找最小值，整体减去最小值，排完再加回来），或者用更复杂的处理（符号位分开）。另外，这里用的是十进制基数（exp ×= 10），如果想更快可以用 2 的幂次（比如 256 进制，每轮 256 个桶），但代码会复杂些。

## 🖥️ 板书/屏幕要点
- **两个函数**：`countSortByDigit()` 按位排序 + `radixSort()` 外层循环
- **提取数字**：`(a[i] / exp) % 10`
- **cnt 大小**：固定 10（每位 0~9）
- **循环轮数**：`for (exp=1; maxVal/exp>0; exp*=10)`
- **复杂度**：O(d × n)，d=位数，空间 O(n)
- **局限**：仅非负整数，负数需偏移

## 📌 小结
基数排序的代码核心是按位计数排序——提取数字 → 统计 → 前缀和 → 从后往前放置 → 拷回。外层 exp 每轮乘 10。十轮以内搞定任何 int 范围。下节课我们把计数、基数、桶排序放在一起对比，做非比较排序的总结。
