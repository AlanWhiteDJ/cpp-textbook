## 📝 讲稿正文

今天我们来解决一道经典题——**最大子段和**。给定数组，找一个连续子数组，使它的和最大。

比如数组 {-2, 1, -3, 4, -1, 2, 1, -5, 4}，最大子段是 {4, -1, 2, 1}，和为 6。

暴力做法是三重循环枚举所有子数组，O(n³)。优化一点用前缀和，O(n²)。但用分治能做到 O(n log n)，而且思维非常漂亮。

分治怎么做？把数组从中间切开，最大子段和可能出现在三个位置：

**① 完全在左半部分**——比如左半的最大子段。
**② 完全在右半部分**——比如右半的最大子段。
**③ 跨越中点**——子段从左边某处开始，跨越 mid，到右边某处结束。

前两种情况递归求解。第三种情况需要在"合"这一步专门计算。

跨越中点的子段怎么求？它必须包含 mid 和 mid+1。从 mid 向左延伸，累加过程中记录最大值（左半最大后缀和）；从 mid+1 向右延伸，同样记录最大值（右半最大前缀和）。两者相加就是跨越中点的最大子段和。

```cpp
int maxSubarraySum(int a[], int l, int r) {
    if (l == r) return a[l];                  // 边界
    
    int mid = l + (r - l) / 2;
    int leftMax = maxSubarraySum(a, l, mid);       // ① 左半
    int rightMax = maxSubarraySum(a, mid + 1, r);  // ② 右半
    
    // ③ 跨越中点
    int leftSum = 0, maxLeftSum = -1e9;
    for (int i = mid; i >= l; i--) {
        leftSum += a[i];
        maxLeftSum = max(maxLeftSum, leftSum);
    }
    int rightSum = 0, maxRightSum = -1e9;
    for (int i = mid + 1; i <= r; i++) {
        rightSum += a[i];
        maxRightSum = max(maxRightSum, rightSum);
    }
    int crossMax = maxLeftSum + maxRightSum;
    
    return max({leftMax, rightMax, crossMax});     // 三选一
}
```

我们跟踪一下跨越中点的计算。数组 {2, -4, 5, -2, 3, -1, 8, -3}，n=8，mid=3（值 -2）。

从 mid=3 向左：累加 -2（max=-2），累加 5（max=5），累加 -4（max=5），累加 2（max=5）。左半最大后缀和 = 5（子段 {5} 或者 {2,-4,5} 取决于从哪开始，但最大后缀是 5）。

从 mid+1=4（值 3）向右：累加 3（max=3），累加 -1（max=3），累加 8（max=10），累加 -3（max=10）。右半最大前缀和 = 10（子段 {3,-1,8}）。

crossMax = 5 + 10 = 15（子段 {5,-2,3,-1,8}）。左半最大 2，右半最大取哪段呢？递归算出来就行。最终答案取三者的最大值。

这个分治的复杂度：T(n) = 2T(n/2) + O(n) → O(n log n)。

但其实最大子段和有 O(n) 的解法——Kadane 算法。那为什么要学分治版？因为分治版的"跨越中点"思想极其重要。最近点对、棋盘覆盖等经典分治问题，核心都在于"跨越中线的部分怎么合并"。分治版最大子段和是这块的入门练习。

以后你遇到更复杂的问题——二维最大子矩阵、树上的最长路径——"切开、递归两边、处理跨中线"这个思路会反复出现。

## 🖥️ 板书/屏幕要点
- 最大子段和：三选一——左半 ①、右半 ②、跨越中点 ③
- 跨越中点 = 左半最大后缀和 + 右半最大前缀和
- 从 mid 向两侧延伸，取累加过程的最大值
- 复杂度：T(n) = 2T(n/2) + O(n) → O(n log n)

## 📌 小结
"跨越中点"是分治中"合"这一步的经典技巧——把问题拆成"纯左"、"纯右"、"跨中"三类，分别处理。最大子段和的分治解法是这个思想的最佳入门。下节课我们学最近点对思想和主定理。
