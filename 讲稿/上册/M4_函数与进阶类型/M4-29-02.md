# 第29章 第2课时：ifstream——从文件读取数据

**时长**：15-20分钟 | **对应教材**：第29章 29.2节

## 🎯 本课时目标
- 掌握 `ifstream` 的完整使用流程：打开→检查→读取→关闭
- 能使用 `fin >>` 从文件读取各种类型的数据
- 养成打开文件后检查是否成功的习惯

## 📝 讲稿正文

上节课的 freopen 很方便——但有个硬伤：一旦重定向，所有输入输出都走文件，屏幕上看不到任何东西。想同时在屏幕上调试、又读写文件？freopen 做不到。

这时候需要更灵活的方案——**文件流（File Stream）**。

C++ 提供了三个文件流类：
- `ifstream`——input file stream，从文件**读**
- `ofstream`——output file stream，向文件**写**
- `fstream`——file stream，既能读又能写

今天先学读——`ifstream`。

它的用法和 `cin` 几乎一样，只是数据来源从键盘换成了文件。来看完整流程（板书）：

```cpp
#include <fstream>    // 文件流需要这个头文件
#include <iostream>
using namespace std;

int main() {
    ifstream fin("data.txt");   // ① 打开文件

    if (!fin.is_open()) {       // ② 检查是否打开成功 ← 非常重要！
        cout << "文件打开失败！" << endl;
        return 1;
    }

    int x;
    while (fin >> x) {          // ③ 像 cin 一样读，读到文件尾自动结束
        cout << "读到：" << x << endl;
    }

    fin.close();                 // ④ 关闭文件（好习惯）
    return 0;
}
```

拆开四步走：

**第一步：打开文件。** `ifstream fin("data.txt");`——声明一个 ifstream 对象 `fin`，同时打开 `data.txt`。`fin` 就像 `cin` 的"替身"——`cin` 从键盘读，`fin` 从文件读。

**第二步：检查是否打开成功。** 这是铁律！文件可能不存在、路径写错、没有权限……如果不检查，后面所有 `fin >> x` 都静默失败——x 的值是未定义的垃圾，程序不报错，你都不知道出了问题。

```cpp
ifstream fin("不存在的文件.txt");
int x;
fin >> x;    // ❌ 静默失败——x 的值未定义！
```

所以每次打开文件后，必须检查：

```cpp
if (!fin.is_open()) {  // 或 if (!fin)
    cout << "错误：无法打开文件！" << endl;
    return 1;
}
```

**第三步：读取数据。** 用法和 `cin` 完全一样！`fin >> 变量` 可以读 int、double、string、char 等任何类型。空格和换行自动跳过。

连续读取（板书）：

```cpp
string name;
int age;
double score;
fin >> name >> age >> score;  // 一行读三个字段
```

循环读取直到文件末尾（板书）：

```cpp
while (fin >> x) {  // 读到文件尾时，fin >> x 返回 false，循环结束
    // 处理 x
}
```

**第四步：关闭文件。** `fin.close();`——释放资源。虽然 C++ 在 `fin` 离开作用域时会自动关闭，但显式 `.close()` 是好习惯——让代码意图更明确。

**完整的读取流程总结：**

```
ifstream fin("文件名")  →  打开
if (!fin.is_open())      →  检查！检查！检查！
fin >> 变量              →  读取（和cin语法一样）
fin.close()              →  关闭
```

好了，现在你已经会从文件读数据了。下节课学怎么往文件里**写**数据——让程序有"存档"功能。

## 🖥️ 板书/屏幕要点
```
ifstream 四步流程：
  ① 打开：ifstream fin("文件名");
  ② 检查：if (!fin.is_open()) { 错误处理; }
  ③ 读取：fin >> 变量; （和 cin 语法一样）
  ④ 关闭：fin.close();

头文件：#include <fstream>

关键方法：
  fin.is_open()  → 是否打开成功
  fin.eof()      → 是否到文件末尾

⚠️ 铁律：打开文件后必须检查！
```

## 📌 小结
ifstream 用于从文件读取数据，用法和 cin 几乎一样（`fin >> 变量`）。四步流程：打开 → 检查 → 读取 → 关闭。检查 `fin.is_open()` 是铁律——不检查可能导致静默失败、读到垃圾数据。`while (fin >> x)` 是逐行/逐词读取的标准写法，读到文件末尾自动结束。需 `#include <fstream>`。
