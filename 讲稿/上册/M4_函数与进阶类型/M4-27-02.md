# 第27章 第2课时：结构体数组——批量管理同类型数据

## 🎯 本课时目标
- 掌握结构体数组的创建、初始化和遍历

## 📝 讲稿正文

上节课我们创建了一个 `Student` 结构体，可以存一个学生的信息。但一个班有 45 个学生——难道要声明 45 个 `Student` 变量？

当然不用。我们有数组！**结构体数组** = 把结构体这种类型当做数组的元素类型。

语法和普通数组一模一样（板书）：

```cpp
Student class1[45];  // 元素类型是 Student，数组长度 45

// 给第 0 个学生赋值
class1[0] = {"小明", 13, 96.5};
class1[1] = {"小红", 14, 92.0};
class1[2] = {"小刚", 13, 88.5};
```

访问也很自然——先取数组元素（得到 `Student`），再用 `.` 取成员：

```cpp
cout << class1[0].name << endl;    // "小明"
cout << class1[1].score << endl;   // 92.0
class1[2].score = 95.0;            // 修改分数
```

**对比一下"平行数组"写法**——这是很多人学 struct 之前会用的：

```cpp
// ❌ 平行数组 —— 同一人的信息分在三处
string names[45];
int ages[45];
double scores[45];
```

平行数组的问题：`names[i]`、`ages[i]`、`scores[i]` 是同一个人吗？要你自己保证——一旦索引对错了，名字和分数就错位了。而且排序时三个数组要同步移动——灾难。

而结构体数组：一个人就是一个 `Student`，整整齐齐。排序也简单——交换两个 `Student`，所有信息一起跟着走。

**结构体数组的常见操作**：

遍历全班，输出每个学生的信息（板书）：

```cpp
for (int i = 0; i < 45; i++) {
    cout << class1[i].name << "："
         << class1[i].score << " 分" << endl;
}
```

计算全班平均分（板书）：

```cpp
double sum = 0;
for (int i = 0; i < 45; i++) {
    sum += class1[i].score;
}
double avg = sum / 45;
cout << "平均分：" << avg << endl;
```

找最高分的学生（板书）：

```cpp
int bestIdx = 0;  // 先假设第 0 个是最高分
for (int i = 1; i < 45; i++) {
    if (class1[i].score > class1[bestIdx].score) {
        bestIdx = i;  // 发现更高的，更新记录
    }
}
cout << "最高分：" << class1[bestIdx].name
     << " " << class1[bestIdx].score << endl;
```

这里我们用了"擂台法"——先让第一个人站上去，后来的和他比，更高的就替换。结果 `bestIdx` 指向最高分的那个学生。

查找某个学生（板书）：

```cpp
string target = "小明";
bool found = false;
for (int i = 0; i < 45; i++) {
    if (class1[i].name == target) {
        cout << "找到了！" << class1[i].name
             << " " << class1[i].age << "岁"
             << " " << class1[i].score << "分" << endl;
        found = true;
        break;
    }
}
if (!found) cout << "没找到" << endl;
```

所有的操作——遍历、求和、找最大、查找——都是你之前学过的数组操作，只不过现在数组的元素不是 int，而是一个 Student。把 Student 当成一个整体来处理，比处理三个平行数组舒服太多了。

**结构体数组的初始化**：除了逐个赋值，也可以声明时整体初始化：

```cpp
Student class1[3] = {
    {"小明", 13, 96.5},
    {"小红", 14, 92.0},
    {"小刚", 13, 88.5}
};
```

三个花括号套在大花括号里——外层是数组，内层是每个学生的三个字段。

结构体数组是学生信息管理、成绩统计、排名系统的核心数据结构。下节课我们学结构体指针和 `->` 运算符，让操作更灵活。

## 🖥️ 板书/屏幕要点
```
结构体数组：
  Student class1[45];
  class1[i].成员     ← 访问第i个学生的某个成员

常见操作：
  遍历求和：sum += class1[i].score;
  找最大：  if (class1[i].score > best) ...
  查找：    if (class1[i].name == target) ...

初始化：
  Student arr[3] = {
      {"小明", 13, 96.5},
      {"小红", 14, 92.0},
      {"小刚", 13, 88.5}
  };
```

## 📌 小结
结构体数组 = 元素类型是结构体的数组。访问用 `数组名[i].成员`。相比平行数组（多个分离数组），结构体数组将同一对象的信息绑定在一起——排序时整个对象一起移动，不会对错位。常见操作（遍历、求和、找最大、查找）都是标准数组操作，只是元素换成了结构体。
