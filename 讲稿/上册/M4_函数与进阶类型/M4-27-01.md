# 第27章 第1课时：定义结构体——创造你自己的数据类型

**时长**：15-20分钟 | **对应教材**：第27章 27.1-27.2节

## 🎯 本课时目标
- 理解 struct 的作用——把多个相关数据打包成一个新类型
- 掌握结构体定义语法（struct + 成员 + 末尾分号）
- 学会创建结构体变量并用 `.` 访问成员

## 📝 讲稿正文

同学们，想象你是一个班主任，要给全班 45 个学生建档案。每个学生有：名字、年龄、分数。

用以前的知识，你得这样（板书）：

```cpp
string name1, name2, name3, ..., name45;  // 45 个名字
int age1, age2, age3, ..., age45;         // 45 个年龄
double score1, score2, ..., score45;      // 45 个分数
```

135 个变量！而且 `name3`、`age3`、`score3` 明明是同一个人的信息，却分在三处——一不留神就把 `name3` 和 `age5` 配了对。

你肯定在想：**能不能像档案袋一样，把一个人的所有信息打包在一起？**

C++ 说：能。用 **struct（结构体）**。

struct 让你把多个不同类型的数据打包，创造一种**全新的数据类型**。就像 int 存整数、double 存小数——现在你可以创造一个 `Student` 类型，专门存学生信息。

来看怎么定义（板书）：

```cpp
struct Student {
    string name;    // 成员1：姓名
    int age;        // 成员2：年龄
    double score;   // 成员3：分数
};  // ← 注意这个分号！绝对不能省！
```

拆开看：

| 部分 | 含义 |
|------|------|
| `struct` | 关键字：告诉编译器"我要定义结构体" |
| `Student` | 结构体名（你起的，之后就成了一种新类型） |
| `{ ... }` | 里面列出所有成员（也叫字段） |
| `;` | 重要！结构体定义结尾必须加分号 |

现在 `Student` 就是一种类型了——和 `int`、`double` 一样，你可以声明变量：

```cpp
Student s1;    // s1 是 Student 类型的变量
Student s2;    // s2 是另一个
```

**怎么给 s1 赋值？用 `.`（点运算符）**（板书）：

```cpp
Student s1;
s1.name = "小明";
s1.age = 13;
s1.score = 96.5;

cout << s1.name << " 今年 " << s1.age
     << " 岁，考了 " << s1.score << " 分" << endl;
```

`.` 读作"的"——`s1.name` 就是"s1 的 name"。简单直观。

**也可以在声明时直接初始化**（板书）：

```cpp
Student s2 = {"小红", 14, 92.0};  // 按成员顺序：name, age, score
```

大括号里按成员定义的顺序赋值，每个值之间用逗号隔开。

来，咱们对比一下有 struct 和没 struct 的区别。

没 struct：30 个学生 → 90 个独立变量 → 灾难。

有 struct：一个 `Student` 类型 → 30 个 `Student` 变量 → 清爽：

```cpp
Student class1[30];  // 一个数组，每个元素是个 Student！

class1[0] = {"小明", 13, 96.5};
class1[1] = {"小红", 14, 92.0};
// 每个学生都在自己的"档案袋"里，不会搞混
```

**常见错误提醒：**

最容易犯的错——**结构体末尾忘了分号**。`struct Student { ... }` 后面没有分号，编译器报一堆莫名其妙的错——因为它以为你还在写定义。

第二个错——**成员定义时初始化**。`struct S { int x = 0; };` 这在 C++11 之后可以，但竞赛中谨慎使用。最保险的做法是用构造函数初始化（后面会讲）。

第三个错——结构体定义完了却不用。struct 只是"蓝图"，要声明变量才算是"盖房子"。`struct Student { ... };` 只是定义了类型，还没创建任何学生。你得写 `Student s1;` 才算生成了一个学生。

好了，理解了 struct 之后，你会发现编程的思路都变了——以前是"零散的数据"，现在可以"打包的数据"。这对后面学排序、学链表、学各种数据结构来说，是必备基础。

## 🖥️ 板书/屏幕要点
```
定义结构体：
  struct 名字 {
      类型 成员1;
      类型 成员2;
      ...
  };                    ← 分号不能忘！

创建变量：
  名字 变量;
  变量.成员 = 值;       ← . 读作"的"

初始化：
  名字 变量 = {值1, 值2, ...};  按成员顺序

例：struct Student { string name; int age; double score; };
    Student s = {"小明", 13, 96.5};
    cout << s.name;
```

```cpp

```

## 📌 小结
struct 把多个相关数据打包成一个新类型（就像档案袋）。定义用 `struct 名字 { 成员列表 };`，别忘了分号。创建变量后，用 `.` 访问成员（`s.name`）。可以在声明时用 `{ }` 按顺序初始化。struct 让数据从"零散"变"打包"，是学习数据结构的第一步。
