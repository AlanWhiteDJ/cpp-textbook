## 📝 讲稿正文

前面几节我们把结构体的"静态结构"搞清楚了——怎么定义、怎么用数组、怎么用指针。今天学"动态操作"——把结构体传给函数，以及自动初始化的构造函数。

**一、结构体作为函数参数——三种方式**

方式一：值传递。复制整个结构体，函数里改不影响外面：

```cpp
void printStudent(Student s) {            // 值传递：复制一份
    cout << s.name << " " << s.score << endl;
}
```

方式二：传指针。只传地址，不复制数据，高效：

```cpp
void addScore(Student* p, double bonus) { // 传指针
    p->score += bonus;                    // 用 -> 修改原数据
}

// 调用
Student s = {"小明", 13, 90};
addScore(&s, 5.5);     // 传地址
cout << s.score;       // 95.5
```

方式三：传引用。结合了效率和简洁语法：

```cpp
void addScore(Student &s, double bonus) { // 传引用
    s.score += bonus;                     // 用 . 修改原数据（语法更自然）
}
```

引用传递和指针传递效果一样（都能改原数据），但引用用 `.` 而不是 `->`，语法上更像操作普通变量。

**二、构造函数——创建对象时自动初始化**

有时候你希望创建 `Student` 变量时自动填入默认值。比如不传参数时，名字默认"未命名"，分数默认 0。

这就需要**构造函数**（板书）：

```cpp
struct Student {
    string name;
    int age;
    double score;

    // 构造函数：名字和结构体名相同，没有返回类型！
    Student() {
        name = "未命名";
        age = 0;
        score = 0.0;
    }

    // 带参数的构造函数
    Student(string n, int a, double s) {
        name = n;
        age = a;
        score = s;
    }
};

// 使用：
Student s1;                          // 调用无参构造 → 默认值
Student s2("小明", 13, 96.5);        // 调用带参构造
```

关键点：
- 构造函数名 = 结构体名
- **没有返回类型**（连 void 都不写）
- 创建对象时自动调用

有了构造函数，再也不会出现"忘了赋初始值"的问题。而且带参构造函数让创建对象更简洁——一行代码搞定。

**三、综合实战——学生成绩排序**

把前面的知识串起来，完成一个完整任务：输入 N 个学生，按总分从高到低排序输出。

我们来一步步分析（板书）：

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student {
    string name;
    int total;   // 总分

    Student() { name = ""; total = 0; }
    Student(string n, int t) : name(n), total(t) {}
};

// 交换两个学生（引用传递）
void swap(Student &a, Student &b) {
    Student temp = a;
    a = b;
    b = temp;
}

// 冒泡排序（按总分降序）
void sortByScore(Student arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j].total < arr[j + 1].total) {  // 降序
                swap(arr[j], arr[j + 1]);
            }
        }
    }
}

int main() {
    int n; cin >> n;
    Student students[100];

    // 输入
    for (int i = 0; i < n; i++) {
        int a, b, c;
        cin >> students[i].name >> a >> b >> c;
        students[i].total = a + b + c;
    }

    // 排序
    sortByScore(students, n);

    // 输出
    for (int i = 0; i < n; i++) {
        cout << students[i].name << " " << students[i].total << endl;
    }
    return 0;
}
```

看，整个过程行云流水：结构体把姓名和总分绑在一起，交换时整个 `Student` 一起换，排序函数清晰易懂。这就是结构体的威力——**让数据"成团"移动，不会散架**。

## 🖥️ 板书/屏幕要点
```
结构体作函数参数：
  值传递：void f(Student s)      → 改副本，不影响原值
  指针传递：void f(Student* p)   → p->成员 访问，能改原值
  引用传递：void f(Student &s)   → s.成员 访问，能改原值

构造函数：
  struct S {
      S() { ... }               // 无参构造，没有返回类型
      S(int x) { ... }          // 带参构造
  };

综合：结构体数组 + swap + 冒泡 = 成绩排序
```

## 📌 小结
结构体能以值、指针、引用三种方式传入函数。指针和引用都能修改原数据，指针用 `->`，引用用 `.`。构造函数（与结构体同名、无返回类型）在创建对象时自动初始化。结构体数组排序时，swap 交换整个结构体——所有成员一起移动，数据不会"散架"。这是 struct 相比平行数组的核心优势。
