# 第23章 第1课时：认识函数——把代码装进"遥控器按钮"

**时长**：15-20分钟 | **对应教材**：第23章 23.1节

## 🎯 本课时目标
- 理解函数的核心概念——有名字的代码块，接收输入、返回输出

- 学会调用函数并在表达式中使用返回值

## 📝 讲稿正文

同学们好！今天我们要学一个让代码瞬间变清爽的超级武器——**函数**。

先问大家一个问题：你们家遥控器上有多少个按钮？（等学生回应）对，电源键、音量键、数字键……你按一下"音量+"，声音就变大。但你不需要知道电路板里发生了什么——按就完事了。

编程里的函数，就是这样的"遥控器按钮"。

回忆一下你之前写过的代码。求两个数的最大值，你写过多少遍？判断素数，写过多少遍？每用一次就把 `if (a > b)` 那段逻辑重新敲一遍。手酸不说，万一有一处写错了，你得满世界找出来改。

如果把"求最大值"写成一个函数，叫 `max`——以后需要比大小的时候，一行 `max(3, 5)` 就搞定。写一次，用无数次。这就是函数的魅力。

**那么问题来了：C++ 里怎么定义一个函数？**

一个函数由四个部分组成。我们来看"求两个数最大值"的例子（板书）：

```cpp
int max(int a, int b) {        // 返回类型int，名字max，参数a和b
    if (a > b) {
        return a;              // 返回较大的那个
    } else {
        return b;
    }
}
```

咱们拆开看这四个部分：

**第一，返回类型**——`int`。意思是这个函数算完之后，会"吐"出一个 int 类型的值。就像自动贩卖机：你投币（传参数），它出货（返回值）。

**第二，函数名**——`max`。你起的名字，以后就用它来"按按钮"。命名规则和变量一样：字母开头，见名知义。

**第三，参数列表**——`(int a, int b)`。括号里写这个函数需要什么"原材料"。max 需要两个整数，所以声明了两个 int 参数。参数之间用逗号隔开。

**第四，函数体**——大括号 `{ ... }` 里的具体代码。这就是"按钮按下后要执行的操作"。

最后用 `return` 把计算结果交出去。这里的 `return a` 或 `return b` 就是把较大的数"吐"给调用者。

**那怎么用这个函数呢？** ——调用它：

```cpp
int main() {
    int result = max(10, 20);   // 调用max，10和20就是实参
    cout << result << endl;     // 输出 20
    cout << max(7, 3) << endl;  // 也可以直接在表达式里用 → 7
    return 0;
}
```

注意看：调用时直接写 `max(10, 20)`，参数之间用逗号分开。**千万不要写成 `max(int 10, int 20)`**——类型声明只在定义时写，调用时只传值！

还有一点：`max(7, 3)` 可以直接塞进 `cout` 里，因为它的返回值就是 7，等价于 `cout << 7 << endl;`。

咱们来对比一下有函数和没函数的区别。假设程序里三个地方需要求最大值：

没函数：每个地方都写一遍 `if (a > b) ... else ...`，三遍，代码又长又容易出错。

有函数：定义一次 `max`，三个地方各一行 `max(x, y)`。干干净净。

这就是**封装**的思想——把一段逻辑打包，取个名字，以后随用随调。就像你把冬天的衣服装进压缩袋——不占地方，要用时拿出来就行。

好了，咱们现在来做个小练习，看看大家掌握了没有。

## 🖥️ 板书/屏幕要点
```
函数定义四要素：
  返回类型  函数名  (参数列表) {
      函数体
      return 返回值;
  }

示例：int max(int a, int b) { ... }

调用：函数名(实参1, 实参2, ...)
      int result = max(10, 20);
      调用时不要写类型！
```

## ✏️ 随堂练习
1. 写一个函数 `int square(int n)`，返回 n 的平方。在 `main()` 里调用它输出 1 到 5 的平方。
2. 改正下列代码中的错误：
```cpp
max(int a, int b) {
    if (a > b) return a;
}
```
3. 写一个函数 `double average(double a, double b)`，返回两数的平均值。测试 `average(7.5, 8.5)`。

## 📌 小结
今天认识了函数的本质——把一段重复使用的代码打包成"遥控器按钮"。定义了四个要素：返回类型、函数名、参数列表、函数体。调用时用 `函数名(实参)`，返回值可以直接参与运算。记住：定义时写类型，调用时不写类型。下节课我们来看不返回值的函数——`void`。
