# 第23章 第2课时：无返回值的函数——void 与 return 的使用

**时长**：15-20分钟 | **对应教材**：第23章 23.2节

## 🎯 本课时目标
- 掌握 `void` 函数的定义和使用场景
- 理解 `return;` 在 void 函数中的作用——提前退出
- 能区分"有返回值"和"无返回值"两种函数

## 📝 讲稿正文

上节课我们写了 `max` 函数，它有返回值——调用之后能拿到一个 int，可以赋给变量、可以参与运算。

但并非所有函数都需要"吐"东西回来。有些函数就是专门**做一件事**的——比如打印一条分隔线、显示一段菜单——做完就完了，不需要返回任何结果。

这时候我们用 `void`。

`void` 是英文"空白、虚无"的意思。`void` 函数就是"不返回任何值的函数"。来看例子（板书）：

```cpp
void printLine(int n) {        // void —— 不返回值
    for (int i = 0; i < n; i++) {
        cout << "-";
    }
    cout << endl;
    // 没有 return 语句
}

int main() {
    printLine(20);             // 输出 20 个短横线
    cout << "学生名单" << endl;
    printLine(20);
    return 0;
}
```

运行效果：20 个横线，标题"学生名单"，再来 20 个横线。整齐美观。

注意看：`printLine` 函数体末尾没有 `return`。因为它是 `void`——"我不需要交作业，我只是来干活的"。

但是！`void` 函数里**可以**写 `return;`——不带值的那种。它表示"提前下班"。

比如说，你写了个打印菜单的函数，但用户还没登录？那就直接 return，后面的菜单不打印了：

```cpp
void showMenu(bool isLoggedIn) {
    if (!isLoggedIn) {
        cout << "请先登录！" << endl;
        return;                // 提前结束，后面不执行
    }
    cout << "1. 开始游戏" << endl;
    cout << "2. 设置" << endl;
    cout << "3. 退出" << endl;
}
```

这里 `return;` 不带值，作用就是"打住，后面的代码别跑了"。这在错误处理、边界判断中非常常用。

好，咱们来总结一下两种函数的区别：

| | 有返回值函数 | void 函数 |
|---|---|---|
| 返回类型 | `int`、`double`、`bool` 等 | `void` |
| 能不能 return 值 | `return 值;` | 不行！ |
| 能不能 return 不带值 | 不行（除非返回类型 void） | `return;` 可以 |
| 调用后能赋值吗 | `int x = func();` | 不能！`void x = f();` 报错 |

再强调一个高频错误——**void 函数里写 `return 值;`**。比如：

```cpp
void max(int a, int b) {
    return a > b ? a : b;  // ❌ 编译错误！void 函数不能 return 值
}
```

编译器会直接报错：void 函数不能返回一个值。正确做法是把返回类型改成 `int`。

反过来，`int` 类型的函数每条路径都**必须 return 值**。比如下面这个就有问题：

```cpp
int max(int a, int b) {
    if (a > b) return a;  // ✅ 这条路有return
    // ❌ 但当 b >= a 时，走到这里没有return！
}
```

编译器会警告，运行时可能返回一个垃圾值。正确写法必须在所有路径上都有 return。

好了，咱们来做一个生活中常用的例子——写一个"倒计时"函数：

```cpp
void countdown(int n) {
    for (int i = n; i >= 1; i--) {
        cout << i << "..." << endl;
    }
    cout << "发射！🚀" << endl;
}
```

调用 `countdown(5)` 就会从 5 倒数到 1，然后发射。这个函数只负责"表演"，不需要返回值。

理解了 void，其实你就理解了 C++ 函数的完整面貌。有返回值的像"计算器"——给你原料，你算个结果还给我。void 函数像"执行器"——给你指令，你干完活就完事。两种各有用处，缺一不可。

## 🖥️ 板书/屏幕要点
```
void 函数：不返回任何值
  void 函数名(参数) { ... }

提前退出：return;（不带值）

错误示范：
  void f() { return 5; }     ❌ void 不能 return 值
  int f() { if(x) return 1; } ❌ 不是所有路径都有 return
```

## 📌 小结
`void` 是不返回值函数的返回类型。它用于"只做事、不交结果"的场景，如打印、显示菜单。void 函数里可以写 `return;` 提前退出，但不能 `return 值;`。有返回值的函数必须每条路径都有 return，否则编译警告、运行时出问题。
